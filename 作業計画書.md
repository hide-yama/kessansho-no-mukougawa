## Phase 3：執筆 ⬜

### 執筆の進め方

**基本方針**: 第1話から順番に執筆（物語のトーンを確立）

**1話あたりの作業フロー**:
```
1. 話ごとの設計書作成
   ├─ シーン割り
   ├─ 使用する企業データの確認
   ├─ セリフの方向性
   └─ 学習ポイントの明確化
        ↓
2. 初稿執筆（10,000〜12,000字 + コラム800〜1,200字）
        ↓
3. レビュー・修正
   ├─ 読者視点でのフィードバック
   ├─ 整合性チェック
   └─ 執筆ガイドラインとの照合
```
