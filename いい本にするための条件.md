# いい本にするための条件

## 本書が「売れる本」になるための絶対条件

---

## 1. 大前提：この本は「教科書」ではなく「エンタメ」である

### 読者は「勉強したい」のではない

読者が本を買う本当の理由：
- **不安を解消したい**：「決算書が読めないとマズい気がする」
- **武器が欲しい**：「数字に強くなれば、キャリアが開ける」
- **賢くなった気分を味わいたい**：「これ読んだらちょっとデキる人になれそう」

教科書的な解説を求めているわけではない。
**「読んでいる時間そのものが楽しい」体験**を求めている。

### 本書のジャンル定義

```
ビジネス × ストーリー × 実用
```

- ビジネス：リアルな企業分析スキルが身につく
- ストーリー：キャラクターと物語で読み進められる
- 実用：明日から使える知識がある

---

## 2. 成功の絶対条件

### 条件1：最初の10ページで「心を奪う」

**本屋で立ち読みされる時間は平均30秒。**

冒頭で以下のいずれかを感じさせなければ、棚に戻される：
- 「この続きが気になる」（好奇心）
- 「これは自分のことだ」（共感）
- 「知らないとヤバいかも」（恐怖）
- 「読んだら得しそう」（欲望）

### 条件2：主人公に「感情移入」させる

**ミサキは読者の分身でなければならない。**

読者がミサキを好きになる要素：
- 弱さがある（完璧じゃない）
- 成長する（変化する）
- 勇気がある（逃げない）
- 正直である（本音を言う）

ミサキが成長するたびに、読者も「自分も成長した」と感じる。

### 条件3：各話に「驚き」がある

**「へぇ〜！」と声が出る瞬間が最低2回/話**

驚きの種類：
- 意外な事実（「メルカリの粗利率は69%もある」）
- 逆説的な真実（「売上が大きいほど危険なこともある」）
- 身近な接続（「あなたが払った手数料がこの利益率を作っている」）

### 条件4：「保存したくなる」コンテンツがある

**SNSでシェアされる/スクショされる箇所を意図的に作る**

保存される要素：
- 一枚で完結する図解
- 覚えやすいフレームワーク
- 印象的な名言・格言

### 条件5：読後に「行動」が変わる

**読者が本を閉じた後、実際に決算書を開く**

そのために必要なこと：
- 「どこを見ればいいか」が明確
- 「何から始めればいいか」がわかる
- 「自分にもできそう」と思える難易度設計

---

## 3. 本書の「敵」を設定する

### ストーリーには「対立構造」が必要

**読者とミサキが共に戦う「敵」を設定する。**

#### 見える敵：ミサキ自身の苦手意識
- 数字を見ると頭が真っ白になる
- 自分には向いていないという思い込み
- 過去の失敗体験（上司に怒られた等）

#### 見えない敵：会社の危機？
- ミサキの会社の決算書に不穏な数字がある？
- 上司から「うちの会社、大丈夫か調べてくれ」と依頼される？
- この危機を発見し、解決策を提案するまでが物語の軸

#### 成長の対価
- 数字が読めるようになることで、知りたくなかった真実も見えてしまう
- 「無知の幸福」を失う代わりに、「見える力」を得る

## 4. 構造設計

### 全12話の構造

```
【第I部：覚醒編】第1〜4話
  ミサキが数字への恐怖を克服し、基礎スキルを身につける

【第II部：探求編】第5〜8話
  より深い分析手法を学び、視野が広がる

【第III部：実践編】第9〜12話
  学んだスキルを統合し、実際の課題を解決する
```

### 各話の起承転結

```
起：謎/問題の提示（冒頭フック）
承：探求と学習（対話による解説）
転：意外な発見/新たな謎（驚きのポイント）
結：統合と次への引き（クリフハンガー）
```

---

## 5. 読者体験のデザイン

### 読書中に読者が感じるべき感情の流れ

```
第1話：不安 → 安心 → 「面白いかも」
第2話：好奇心 → 発見 → 「自分でもできそう」
第3話：恐怖 → 理解 → 「知ってよかった」
第4話：混乱 → 整理 → 「つながった！」
第5話：挑戦 → 達成感 → 「視野が広がった」
第6話：発見 → 興奮 → 「株価が読める！」
第7話：共感 → 応援 → 「赤字企業も面白い」
第8話：尊敬 → 納得 → 「安定の価値」
第9話：競争心 → 分析欲 → 「比較って楽しい」
第10話：俯瞰 → 洞察 → 「業界が見える」
第11話：統合 → 自信 → 「全部つながった」
第12話：決意 → 行動 → 「明日から使う！」
```

## 6. 差別化ポイント

### 既存の財務本との違い

| 既存の本 | 本書 |
|---------|-----|
| 網羅的に説明 | ストーリーで厳選 |
| 架空の企業 | 実在の有名企業 |
| 読者は傍観者 | 読者はミサキと共に学ぶ |
| ゴールは理解 | ゴールは行動 |
| 読んで終わり | 読後に決算書を開きたくなる |

### 本書だけの武器

1. **実在企業の生データ**：トヨタ、ユニクロ、メルカリ、ニトリ
2. **師弟関係のドラマ**：安藤先生とミサキの人間的な交流
3. **成長譚**：読者自身の成長を物語として体験
4. **保存可能な図解**：SNSシェア、壁に貼れるフレームワーク
5. **コラム**：現実世界の裏話、歴史、トリビア

## 8. 最終ゴール

### この本を読み終えた読者の姿

```
Before：「決算書？ 難しそう。見たくない」
After ：「決算書？ 面白い。毎四半期チェックしてる」
```

### 読者の変化

1. 決算書を開くことへの抵抗がなくなる
2. 数字の向こうに「ストーリー」が見えるようになる
3. 自分の会社、取引先、投資先を分析できるようになる
4. 「数字が読める人」としての自信を持つ
5. 経済ニュースの見方が変わる

## 結論

**この本は「財務分析の教科書」ではない。**

**「数字が苦手だった若者が、師匠との出会いを通じて世界の見え方を変えていく冒険譚」である。**

読者がこの冒険に参加したくなる仕掛けを、全話に張り巡らせる。
