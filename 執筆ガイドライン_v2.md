## 第1章：本書のDNA

### ワンライン・ピッチ
> 「数字が怖かった若手社員が、穏やかで核心を突くアナリストとの出会いを通じて"決算書の向こう側"を読み解く力を手に入れる13週間の成長譚」

### 本書のジャンル
**ビジネス・エンターテイメント**

これは教科書ではない。読者が「続きが気になって眠れない」と思うビジネス小説であり、読み終えたときに「明日から決算書を読んでみよう」と思える実用書でもある。

### コアメッセージ
**「数字は嘘をつかない。でも、見方を知らないと何も語ってくれない」**

## 第3章：物語の設計

### メインプロット

```
【発端】
ミサキは経営企画部に異動したが、数字が苦手で毎日が地獄。
上司から「3ヶ月後に、うちの会社の分析レポートを経営会議で発表しろ」と命じられる。

【出会い】
途方に暮れていたミサキは、社外アドバイザーの安藤先生と出会う。
「私に1週間に1回、2時間だけくれ。3ヶ月で君を変えてみせる」

【成長】
毎週の対話を通じて、ミサキは少しずつ数字が読めるようになる。
トヨタ、ユニクロ、メルカリ、ニトリ——実在企業を題材に学ぶ。

【危機】
第10話で、ミサキは自社の決算書に「不穏な兆候」を発見する。
「この数字、おかしくないですか？」

【解決】
ミサキは学んだスキルを総動員して分析し、経営会議で発表。
問題点と改善策を提案し、経営陣を驚かせる。

【変化】
かつて数字から逃げていたミサキは、今や「数字で物語を語れる人」になった。
```

### サブプロット

1. **安藤先生の過去**：なぜ彼は若手育成に情熱を燃やすのか？
   → リーマンショック時の失敗と贖罪

2. **ミサキの自社分析**：彼女の会社には本当に問題があるのか？
   → 第11話で明らかになる危機、第13話で解決

3. **成長の代償**：数字が読めるようになると、見たくないものも見える
   → 知識は武器であり、時に呪いでもある

## 第6章：コラムの設計

### コラムの目的

1. **本編で語れなかった深掘りを提供**
2. **読者に「得した感」を与える**
3. **息抜きとして機能する**
4. **SNSでシェアされやすいコンテンツを作る**

### コラムの種類（6タイプ）

#### Type A：歴史・エピソード型
その話で扱った概念や企業の「裏話」「歴史」「逸話」を紹介。
例：「ユニクロ、アメリカ撤退の真実」「ROEブームはどこから来たか」

#### Type B：人物伝型
偉大な投資家、経営者、アナリストのエピソード。
例：「ウォーレン・バフェットが決算書で最初に見る項目」

#### Type C：失敗事例型
数字を読み違えて失敗した事例。教訓として。
例：「エンロン事件——粉飾決算はなぜ見抜けなかったか」

#### Type D：実践Tips型
読者がすぐ使えるテクニック。
例：「決算書を10分で読むための3ステップ」「無料で使える財務データベース5選」

#### Type E：比較・ランキング型
興味を引くランキングや意外な比較。
例：「営業利益率ランキング：意外な1位はあの会社」

#### Type F：Q&A型
読者が抱きそうな疑問に答える形式。
例：「よくある質問：なぜ赤字でも株価が上がるのか？」

### コラムのルール

1. **本編と独立して読めること**（本編を読んでなくても楽しめる）
2. **1つのコラムで1つのメッセージ**（詰め込みすぎない）
3. **具体的なエピソードから入る**（抽象論から始めない）
4. **最後に「だから何」を明確に**

## 第8章：文体ルール

### 対話文

```
【ミサキ】
- 丁寧語（「〜です」「〜ですか」）
- 疑問形が多い
- 心の声を（　）で描写

例：
「あの、ちょっといいですか」
「正直に言っていいですか？　まだよくわかってないです」
（また数字だ。頭が真っ白になりそう——）
```

```
【安藤先生】
- 穏やかな丁寧語（「〜だね」「〜だよ」）
- 問いかけが多い（相手に考えさせる）
- 肯定から入り、核心を突く

例：
「君はどう思う？」（穏やかに、急かさず）
「いい質問だね。実はね——」
「そう、そこなんだよ」（静かに、しかし確信を持って）
「私も昔、同じところで躓いたよ」
```

### 地の文

- 三人称限定視点（ミサキ視点）
- 簡潔に。情景描写は最小限
- ミサキの内面を描写してOK
- 読者への問いかけは控えめに

### 禁止表現

| 禁止 | 代替 |
|-----|-----|
| とても/非常に/すごく | 具体的な数字や比較で表現 |
| 〜と言われている | 誰が言っているか明記 |
| 重要です | なぜ重要かを示す |
| 基本的に | 削除するか具体化 |
| ある意味 | 削除 |

## 第11章：この本の魂

### 読者への約束

この本を読み終えた読者は、決算書を開くことが怖くなくなる。

数字の羅列を見て、その向こうにいる人間——経営者の決断、社員の努力、顧客の選択——を想像できるようになる。

会議で「この数字はどう読む？」と聞かれたとき、自信を持って答えられるようになる。

### 著者への問いかけ

各話を書き終えたとき、自問してほしい：

1. **この話を読んで、読者の世界の見え方は変わるか？**
2. **ミサキの成長を通じて、読者は自分の成長を感じられるか？**
3. **読者は次話を読みたくてたまらなくなるか？**
