## 4. なぜ粗利率が69%もあるのか：プラットフォームビジネスの構造

### 4.1 プラットフォームビジネスの売上原価とは

メルカリの売上原価には以下が含まれます：

| 項目 | 内容 |
|------|------|
| 決済手数料 | クレジットカード会社等への支払い |
| システム関連費用 | サーバー・クラウド費用 |
| カスタマーサポート費用 | 問い合わせ対応人件費 |
| 配送関連費用 | らくらくメルカリ便等の原価 |

### 4.2 製造業との決定的な違い

| 項目 | メルカリ（プラットフォーム） | トヨタ（製造業） |
|------|---------------------------|----------------|
| 粗利率 | **約69%** | 約18〜20% |
| 在庫リスク | なし（仲介のみ） | あり（数兆円規模） |
| 製造原価 | なし | 鉄鋼・部品・人件費等 |
| 収益モデル | 取引手数料10% | 製品販売 |

**プラットフォームビジネスの粗利率が高い理由**：

1. **在庫を持たない**
   - メルカリは売り手と買い手をマッチングするだけ
   - 商品の仕入れ・製造コストがゼロ
   
2. **限界費用が極めて低い**
   - 追加の取引が発生してもコストはほぼ一定
   - ユーザーが増えるほど利益率が改善

3. **ネットワーク効果**
   - ユーザーが増えるほど価値が上がる
   - 後発参入が困難（競争優位の確立）

## 5. なぜ粗利率69%なのに営業利益率は10%程度なのか

### 5.1 販管費の構造

メルカリの販管費は売上収益の約60%を占めます。

| 費目 | 内容 | 備考 |
|------|------|------|
| 広告宣伝費 | テレビCM、デジタル広告 | 最大の費目 |
| 人件費 | エンジニア・マーケティング | 高スキル人材が多い |
| ポイント費用 | 還元キャンペーン | 顧客獲得コスト |
| 不正対策費用 | 詐欺・なりすまし対策 | 近年増加 |

### 5.2 広告宣伝費の推移

| 期 | 広告宣伝費 | 対売上比率 |
|----|-----------|-----------|
| 2017年6月期 | 142億円 | **64.3%** |
| 2018年6月期 | 169億円 | 47.3% |
| 2019年6月期 | 193億円 | 37.4% |
| 2020年6月期 | 343億円 | 45.0% |
| 2021年6月期 | 315億円 | 28.8% |
| 2022年6月期 | 377億円 | 25.6% |
| 2023年6月期 | 330億円 | 19.2% |
| 2024年6月期 | 442億円 | 23.6% |

**出典**: IRBank、有価証券報告書

### 5.3 「粗利率が高いのに利益率が低い」構造の解説

```
売上収益      1,874億円   100%
 - 売上原価      580億円   ▲31%
────────────────────────────
売上総利益    1,294億円    69%  ← 高い粗利率
 - 販管費      1,119億円   ▲60%
   うち広告宣伝費 442億円
   うち人件費   約300億円
────────────────────────────
営業利益       175億円    9.3% ← 低い営業利益率
```

**結論：成長投資（広告宣伝費・人件費）に粗利益の大半を投下しているため**

### 6.2 黒字化の転換点：何が変わったか

**2023年6月期に本格的に黒字転換した理由**：

1. **広告宣伝費の抑制**
   - 2022年6月期の377億円 → 2023年6月期は330億円に削減
   - 「投資から収穫」のフェーズへ移行

2. **国内フリマ事業の成熟**
   - MAU 2,260万人で国内最大のフリマアプリに
   - 新規ユーザー獲得から既存ユーザー活性化へシフト

3. **Fintech事業の黒字化**
   - メルカードの発行枚数急増
   - 定額払いの金利収入が安定収益に

4. **US事業の赤字縮小**
   - 人員削減・マーケティング費用の見直し
   - 2025年6月期には通期黒字化達成

5. **不正対策の効果**
   - AI活用による不正利用者の検知
   - 不正関連費用が半減

## 8. Fintech事業の詳細

### 8.1 メルペイの仕組み

メルペイは以下の決済手段を提供：

| サービス | 仕組み |
|---------|--------|
| 残高払い | メルカリ売上金をチャージして決済 |
| あと払い | 翌月払い（手数料無料〜880円） |
| 定額払い | リボ払い相当（年率15%） |
| iD決済 | Apple Pay/モバイルiD連携 |

### 8.2 メルカードの収益モデル

2022年11月にサービス開始したクレジットカード「メルカード」は、以下の収益構造を持ちます：

| 収益源 | 内容 |
|--------|------|
| 加盟店手数料 | JCB加盟店からの手数料収入 |
| 定額払い金利 | **年率15%**の金利収入（主力） |
| 決済手数料節約 | 自社決済比率向上でコスト削減 |

**メルカードの特徴**：
- 発行枚数: **340万枚**（2024年6月時点）
- 発行ペース: 国内トップレベル（年間約220万枚）
- 債権残高: **1,872億円**
- 回収率: **99.2%**（貸倒リスク低い）

### 8.3 フィンテック事業の成長戦略

1. **「メルカリ」との相乗効果最大化**
   - メルカード保有者のメルカリ内ARPU（1人あたり売上）が50%向上
   - 売上金でカード支払い → 現金流出なし

2. **定額払いの拡大**
   - 年率15%の金利収入はストック型ビジネス
   - 債権残高の積み上げで安定収益

3. **自社決済比率の向上**
   - 外部カード会社への手数料削減
   - 2024年時点で自社決済比率6pt向上

1. **TAM（獲得可能市場規模）の大きさ**
   - 日本のCtoC市場: 約2兆円
   - フィンテック市場: 数十兆円規模
   - 米国市場: さらに巨大

   - 国内フリマ事業単体では**営業利益率40%**

   - 2,000万人以上のMAU
   - 後発が追いつけない「勝者総取り」の構造

   - 売上成長率が高ければ高PSRが許容される

## 10. 結論：プラットフォームビジネスの財務特性

### 10.1 「なぜ粗利率が高いのに利益率は低いのか」への回答

**答え：成長投資（広告宣伝費・人件費）に粗利益を投下しているから**

| フェーズ | 戦略 | 結果 |
|---------|------|------|
| 成長期（2013〜2022年） | 粗利益を広告に全投入 | 市場シェア獲得・赤字 |
| 収穫期（2023年〜） | 広告費を抑制 | 黒字転換・利益成長 |

### 10.2 「なぜ赤字でも評価されたのか」への回答

**答え：将来の利益ポテンシャルが見えていたから**

1. **国内事業は既に高収益**（営業利益率40%）
2. **ネットワーク効果で競争優位**（後発が追いつけない）
3. **成長投資を止めればいつでも黒字化可能**
4. **TAMが大きい**（市場がまだ成長途上）

### 10.3 メルカリの教訓

| ポイント | 内容 |
|---------|------|
| プラットフォームの強さ | 在庫リスクなし、限界費用ほぼゼロ |
| 成長と利益のトレードオフ | 広告費を削れば利益は出るが成長は止まる |
| 複合的な収益化 | フリマ→決済→カードと収益源を多角化 |
| 長期投資の覚悟 | 米国事業は10年以上赤字でも継続 |
