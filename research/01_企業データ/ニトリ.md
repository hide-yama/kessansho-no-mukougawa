| 2025年3月期 | 9,290億円 | 1,204億円 | **13.0%** |

### 営業利益率低下の要因

**① 為替（円安）の影響【最大要因】**
- 商品の約90%を海外（主にアジア）で生産・輸入
- 1円の円安で年間約20億円の減益要因
- 2023年3月期は為替影響で経常利益が381億円押し下げられた
- 数年単位の為替予約を取りやめたため、円安がダイレクトに仕入価格に反映

**② 物流費・人件費の上昇**
- 海上輸送コンテナの需給逼迫による運賃高騰
- 国内物流の2024年問題（ドライバー不足）
- 賃金改定による人件費増加

**③ 島忠事業の利益率の低さ**
- 2021年に島忠を連結子会社化
- 島忠事業の経常利益率は5%程度（ニトリ事業は約20%）
- 全社の利益率を押し下げる構造

**出典**: 日本経済新聞「ニトリ24年ぶり最終減益」、妄想する決算

## 5. SPAモデルの詳細

### ニトリのSPA（製造物流IT小売業）の特徴

ニトリは自社を「製造物流IT小売業」と定義し、一般的なSPA（製造小売業）に**物流機能とIT**を加えた独自モデルを構築している。

```
原材料調達 → 製造 → 物流 → 販売 → 配送
    ↑         ↑      ↑      ↑      ↑
  すべて自社グループでコントロール
```

### 自社工場・物流網の強み

**製造拠点**:
- インドネシア（1994年設立）：タンス、食器棚など
- ベトナム（2004年設立）：キッチンボード、マットレスなど
- 協力工場：世界18地域、約740社

**物流網**:
- 国内配送センター（DC）：仙台DC、名古屋DC、幸手DC等
- 海外物流センター：中国・恵州、上海プロセスセンター
- 配送網の人口カバー率：99%
- 貿易業務（輸入申請、許可、契約）をすべて自社で実施

**IT基盤**:
- 2022年「ニトリデジタルベース」設立
- ECプラットフォーム整備
- 海外店舗の販売データ・顧客ニーズをリアルタイム収集

### ユニクロのSPAとの比較

| 項目 | ニトリ | ユニクロ |
|------|--------|----------|
| **自社業態** | 製造物流IT小売業 | SPA（製造小売業） |
| **自社工場** | **あり**（ベトナム、インドネシア） | なし（100%外部委託） |
| **生産拠点** | 自社工場＋協力工場740社 | 中国、バングラデシュ等400拠点以上 |
| **物流** | **自社完結**（ホームロジスティクス） | 外部委託中心 |
| **製造管理** | 自社工場をマザー工場として協力工場を指導・管理 | 匠チームによる技術指導 |
| **商品配送** | 自社配送（大型家具の設置まで） | 外部委託 |

**共通点**:
- 企画から販売まで一貫管理で中間マージン排除
- 低価格・高品質を両立
- 顧客との直接接点からトレンドを把握

**相違点**:
- ニトリは物流・配送まで自前主義（より垂直統合が深い）
- ユニクロは生産を完全外部委託（身軽で機動的）
- ニトリは大型家具の「配送・設置」が差別化要素

**出典**: ITmedia「日本発垂直統合型自前主義は通用するか」、Business Insider Japan、ニトリ公式サイト

| 目標 | 数値 |

| うち海外 | 2,000店舗 |
| 売上高 | 3兆円 |
| うち海外 | 1兆円〜1.5兆円 |

### 米国撤退の教訓

- 2013年「Aki-Home」ブランドで米国進出
- 2023年に全店舗閉鎖、完全撤退
- **撤退理由**:
  - トランプ政権時代の対中関税25%
  - 中国からの輸入に依存するビジネスモデルが成立困難
  - 現地競合との価格競争

### ユニクロのような海外成功を再現できるか？

**課題と障壁**:

| 課題 | 詳細 |
|------|------|
| 商品特性 | 家具は衣料より輸送コストが高く、ローカライズが難しい |
| 住宅事情 | 国ごとに住宅の広さ・スタイルが異なる |
| 競合環境 | IKEAという巨大なグローバル競合が存在 |
| ブランド認知 | 欧米での知名度がほぼゼロ |
| 物流コスト | 大型家具の国際輸送・ラストマイル配送のコスト |

**アジア市場での可能性**:
- 住宅事情・生活スタイルが日本に近い
- サプライチェーンの中核がアジアにある
- 台湾は進出7年目で単年度黒字化達成
- インド：2032年までに289店舗構想

**出典**: WWD Japan「ニトリが米国撤退」、Digima「ニトリの海外進出と失敗から学ぶ再起戦略」

### ビジネスモデルの共通点

| 共通点 | 詳細 |
|--------|------|
| SPA型 | 企画→製造→販売を一貫管理 |
| 低価格・高品質 | 中間マージン排除でコスト優位 |
| プライベートブランド | 商品の大半が自社開発 |
| アジア生産 | 製造拠点はアジア中心 |
| 国内寡占 | それぞれの分野で国内トップ |

### なぜユニクロは海外で成功し、ニトリはまだ道半ばなのか？

| 要因 | ユニクロ | ニトリ |
|------|----------|--------|
| **商品特性** | 衣料品は輸送しやすい、サイズ展開で対応可能 | 家具は大型・重量物、住宅事情に依存 |
| **ローカライズ** | サイズ・色展開で対応可能 | 国ごとの住宅事情に合わせた商品開発が必要 |
| **グローバル競合** | ZARA、H&Mと戦える差別化（機能性素材） | IKEAという巨人が存在 |
| **旗艦店戦略** | 主要都市の一等地に大型旗艦店 | アジア中心、SC内出店が主 |
| **海外進出時期** | 2001年〜（約24年の蓄積） | 2007年〜（約18年） |
| **失敗からの学習** | 英国撤退→中国成功のピボット | 米国撤退→アジア集中へピボット中 |
| **ブランド浸透** | 「LifeWear」コンセプトが世界で浸透 | 「お、ねだん以上。」は日本限定の認知 |

### 海外展開の時間軸

```
【ユニクロ】
2001年 英国進出（失敗→撤退）
2002年 中国進出（成功の起点）
2006年 NY旗艦店
2022年 欧州・北米黒字化
2024年 海外売上が国内を逆転

【ニトリ】
2007年 台湾進出（初の海外）
2013年 米国進出
2014年 中国進出
2023年 米国撤退
2024年 インド進出
2032年 海外2,000店舗目標
```

**結論**: ニトリは「アジア版IKEA」を目指す戦略にピボットしつつある。ユニクロが欧州黒字化まで約20年かかったことを考えると、ニトリの海外事業も長期的視点での評価が必要。

**出典**: ファーストリテイリング決算資料、NetIB-News「ユニクロ、無印良品、ニトリ 海外で伸ばす事業戦略」

## 8. 結論：「斜めの比較」から見えるもの

### SPAモデルの普遍性と限界

ニトリとユニクロは、ともにSPAモデルを採用した「異業種の双子」である。しかし、その海外展開の成否には大きな差がある。

| 観点 | 示唆 |
|------|------|
| **SPAの強み** | 両社とも国内市場では圧倒的な競争優位を構築 |
| **商品特性の壁** | 衣料品と家具では、輸送コスト・ローカライズ難易度が根本的に異なる |
| **時間軸** | グローバル展開には20年以上の時間がかかる |
| **失敗の価値** | 両社とも失敗（ユニクロ：英国、ニトリ：米国）から学び戦略を修正 |

### ニトリの今後の注目ポイント

1. **円安環境への適応**: 為替ヘッジ戦略の再構築
2. **アジア出店加速**: 年間100店→200店体制の構築
3. **島忠シナジー**: 利益率改善（5%→12%目標）
4. **インド市場**: 289店舗構想の進捗

### ビジネス書での活用ポイント

> **「SPAモデルは万能ではない」**
> 
> 同じSPAでも、商品特性（輸送コスト、ローカライズ難易度）によって海外展開の難易度は大きく異なる。ユニクロの成功をそのまま家具業界に適用できない理由がここにある。

> **「グローバル展開は20年計画」**
> 
> ユニクロが欧州・北米で黒字化するまで約20年。ニトリも2007年の台湾進出から17年が経過し、ようやくアジア展開が軌道に乗り始めた段階。短期的な成果を求めすぎると判断を誤る。
