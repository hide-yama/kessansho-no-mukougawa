## 1. 本書のユニークな価値（類書との差別化）

### 1-1. 市場における位置づけ

```
                  [エンタメ性:高]
                        |
   「会計の世界史」        |     ★本書★
                        |
[入門] ─────────────────┼───────────────── [実践]
                        |
   「決算書がスラスラ      |     「財務3表一体理解法」
     わかる」系           |
                        |
                  [エンタメ性:低]
```

### 1-2. 類書との差別化ポイント

| 差別化軸 | 類書 | 本書 |
|----------|------|------|
| **事例** | 架空企業 or 有名企業の断片的紹介 | **実在企業の財務を徹底解剖**（トヨタ、ユニクロ、アクセンチュア、トランスコスモス等） |
| **読者との距離** | 読者と無関係な事例 | **読者自身のキャリアに近い事例**（中村さんペルソナ） |
| **学習ゴール** | 「知識」を得る | **「自分ごと」として体感し、実践できる** |
| **不正・リスク** | 事件として紹介（エンロン等） | **読者が見抜く力を身につける**（半沢直樹展開） |
| **物語構造** | 解説の合間にストーリー | **物語の中で自然に学ぶ**（ミサキの成長＝読者の成長） |

### 1-3. 本書だけの強み

**「ビジネスモデルの違いが利益率を決める」を体感できる**

| 企業 | 営業利益率 | ビジネスモデル |
|------|-----------|---------------|
| アクセンチュア | **14.7%** | 知識を売る（コンサル型） |
| トランスコスモス | **3.9%** | 労働時間を売る（BPO型） |

同じ「IT関連」でも、**何を売るか**で利益率が4倍違う。この差を「中村さん」の実体験として語れることが、本書の最大の差別化要因。

## 2. メインペルソナ「中村さん」

### 2-1. 基本プロフィール

| 項目 | 内容 |
|------|------|
| **名前** | 中村さん（仮名） |
| **年齢・性別** | 42歳・男性 |
| **職種** | Webディレクター（フロントエンドエンジニア、デザイナー経験あり） |
| **現状** | 独立したて。トランスコスモス経由でSMBC案件を業務委託で受注 |
| **将来計画** | 自分の事業を立ち上げ、軌道に乗ったら業務委託を卒業 |

### 2-2. キャリアの変遷（3社の経験）

```
株式会社シグナル（約100名、PR×デジタル）
    │
    │ 2023年：アクセンチュアに買収される
    │ 2024年：吸収合併で法人消滅
    ↓
アクセンチュア（営業利益率15%の世界）
    │
    │ 独立を決意
    ↓
独立 → トランスコスモス経由でSMBC案件
    （営業利益率4%の世界）
    │
    ↓
【今ここ】自分の事業を立ち上げたい
```

### 2-3. 中村さんが体験した「数字の物語」

| 体験 | 財務的な意味 | 本書で学べること |
|------|-------------|-----------------|
| 自分がいた会社が買収された | 会社の価値＝約17〜19億円 | M&Aと企業価値評価 |
| アクセンチュアの高給を体験 | 高利益率→高報酬が可能 | ビジネスモデルと収益構造 |
| トランスコスモスの薄利多売を目撃 | BPOの構造的な利益率制約 | 業界による「正常値」の違い |
| SMBCの仕事を担当 | 銀行の特殊な財務構造 | 銀行財務の読み方 |

### 2-4. 中村さんの課題（本書で解決したいこと）

**表面的なニーズ：**
- 財務諸表を読めるようになりたい
- 取引先・外注先の信用度を数字で判断したい
- 自分の事業の数字を経営視点で見たい

**深層のニーズ（本当の動機）：**
- 「騙されたくない」「損したくない」
- 数字に強い事業主として見られたい
- 自分がいた会社の買収を、後から理解したい
- 次は「数字がわかる側」に立ちたい

### 2-5. 中村さんの強み

- アクセンチュア出身でビジネス感覚がある
- 大企業（SMBC）の仕事を回せるプロジェクト管理能力
- 技術がわかるので「数字」への抵抗感は比較的低い
- 論理的思考ができる

## 3. コアメッセージ

### 3-1. メインメッセージ

> **「数字は嘘をつかない。でも、見方を知らないと何も語ってくれない」**

### 3-2. サブメッセージ

| # | メッセージ | 対応する学び |
|---|-----------|-------------|
| 1 | 利益率の違いは、ビジネスモデルの違い | アクセンチュア vs トランスコスモス |
| 2 | 数字の向こうには、人の物語がある | シグナル買収、従業員の運命 |
| 3 | 「何かおかしい」と気づく感度が、あなたを守る | 半沢直樹展開、不正検知 |
| 4 | 財務諸表は、会社の「健康診断書」 | B/S、P/L、CFの基本 |
| 5 | 数字を読めると、世界の見え方が変わる | 読者の変身 |

### 3-3. 読者への約束

本書を読み終えたとき、あなたは：

- 取引先の決算書を見て「この会社、大丈夫かな」と判断できる
- ニュースで「営業利益率15%」と聞いて、その意味がわかる
- 自分の事業の数字を「経営者の目」で見られる
- 「何かおかしい」という違和感を数字で説明できる

## 4. 半沢直樹展開の位置づけ

### 4-1. 採用理由

単なるエンタメ要素ではなく、**「不正を見抜く力」という実用スキル**として位置づける。

**読者が得られるもの：**
- 業績の良し悪しを判断する力（従来の財務分析）
- **「何かおかしい」と気づく感度**（不正検知の視点）

### 4-2. 物語での展開（案）

| 話数 | 展開 | 学べる不正検知スキル |
|------|------|---------------------|
| 第11話 | 不穏な数字を発見 | 売掛金とCFの乖離、売上債権回転期間の異常 |
| 第12話 | 証拠を固める | 関連当事者取引の調査、有報注記の読み方 |
| 第13話 | 経営会議で告発 | 統合的な財務分析、プレゼンテーション |

### 4-3. 敵役：黒田専務（案）

| 項目 | 内容 |
|------|------|
| 名前 | 黒田専務（60歳） |
| 役職 | 東洋テック専務取締役、海外事業本部長 |
| 表の顔 | 「海外事業を立て直した功労者」として社内評価が高い |
| 裏の顔 | 数字を操作して自分の手柄を作っていた |

### 4-4. 不正の手口（財務分析で発見可能なもの）

| 手口 | 財務上の兆候 | 学べるスキル |
|------|-------------|-------------|
| 架空売上の計上 | 売掛金の異常な膨張 | 売上債権回転期間の分析 |
| 在庫の水増し | 棚卸資産の急増 | 在庫回転期間の分析 |
| 費用の先送り | 利益とCFの乖離 | CF計算書の読み方 |
| 関連会社への付け替え | 関連当事者取引の急増 | 有報の注記の読み方 |

### 4-5. 読者体験の設計

- ミサキが学ぶ財務分析スキルが、終盤で専務の不正発見につながる
- 読者も「この数字の違和感、私も気づけた！」という体験ができる
- 「数字を読める」ことが「自分を守る力」になると実感

## 5. 本書で得られる「変身」

### 5-1. Before → After（読者のセリフ）

**Before：「数字苦手」を隠すセリフ**

| シーン | 言いたくないセリフ |
|--------|------------------|
| 取引先との会食 | 「いやー、私、数字弱いんで（苦笑）」 |
| 後輩からの質問 | 「決算書？...まあ、売上見とけばいいんじゃない？」 |
| 投資の話題 | 「株とかよくわかんないから、貯金派で」 |
| 独立準備中 | 「まあ、売上さえ立てば何とかなるでしょ」 |
| 不祥事ニュースを見て | 「えー、信じてたのに...全然わからなかった」 |

**After：「数字が読める人」のセリフ**

| シーン | 言ってみたいセリフ |
|--------|------------------|
| 取引先との会食 | **「御社、営業CFがずっとプラスですよね。うちも見習いたいです」** |
| 後輩からの質問 | **「売上だけ見てもダメ。利益率とキャッシュ、両方見なきゃ」** |
| 投資の話題 | **「あの株、PER高いけどPSRで見ると割安かもね」** |
| 独立準備中 | **「まず損益分岐点を出して、月いくら必要か計算しないと」** |
| 不祥事ニュースを見て | **「やっぱりな。売掛金の回転期間、3年前からおかしかった」** |

**究極のAfterセリフ（本書のゴール）**

> **「この数字、おかしくないですか？」**
>
> 会議で、取引先で、投資判断で——この一言が言えたら、あなたは「数字が読める人」になっている。

5. **「おかしい」と思ったら調べる**（リスク回避）

## 6. まとめ：本書の独自性

### 一言で言うと

> **「自分ごと」として財務諸表を学び、数字で身を守る力を身につける物語**

### 3つの柱

1. **実在企業の徹底分析**：トヨタ、ユニクロ、アクセンチュア、トランスコスモス等
2. **読者に近いペルソナ**：独立したての個人事業主「中村さん」
3. **実用的なクライマックス**：不正を見抜く「半沢直樹展開」

### 競合に対する優位性

| vs 類書 | 本書の優位性 |
|---------|-------------|
| vs 教科書型 | 物語で「自分ごと」化できる |
| vs ストーリー型 | 実在企業の本物の数字を使う |
| vs 投資本 | 投資だけでなく「事業主としての視点」を提供 |
| vs 不正本 | 読者自身が見抜く力を身につける設計 |
