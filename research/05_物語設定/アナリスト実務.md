# 証券アナリストの実務
## 元アナリストが語る真実——ビジネス書のための設定資料

*「株価を予想する仕事」と思われがちだが、本質は違う。我々の仕事は「企業の真実を見抜くこと」だ。*

---

# 第1章：証券アナリストとは何者か

## 1.1 仕事の概要

証券アナリストとは、企業や業界の経済状況を調査・分析し、投資判断のための情報を提供する専門家である。株価が企業の実力を正しく反映しているかを分析し、「買い」「中立」「売り」の投資判断を下す。

**アナリストとエコノミストの違い：**

| 職種 | 分析対象 | 主な成果物 |
|------|----------|-----------|
| **アナリスト** | 個別企業・業界 | 銘柄分析レポート、目標株価 |
| **エコノミスト** | マクロ経済全体 | 経済見通し、金利・為替予測 |

アナリストの仕事は「銘柄分析」と呼ばれ、特定の企業や業界に焦点を絞って分析する。

---

## 1.2 セルサイドとバイサイドの違い

証券アナリストには大きく2つの立場がある。この違いを理解することは、キャラクター設定において極めて重要だ。

### セルサイド・アナリスト

**所属：** 証券会社（野村證券、大和証券、ゴールドマン・サックスなど）

**役割：** 顧客（機関投資家）に対する情報提供サービスとしてレポートを作成。「証券会社の顔役」として、メディアへの露出も多い。

**特徴：**
- アナリストレポートを外部に公開
- 日経ヴェリタスなどの「人気アナリストランキング」で順位付けされる
- カバーする企業数は多いところで400〜500社、外資系では150社前後
- 上場企業約3,500社のうち、アナリストがカバーしているのは**わずか10%**
- 年収は800万〜3,000万円超（外資系上位）

**セルサイドの宿命：**
> 「一度調査を公式に開始すれば、継続的かつ連続性を持って意見を発信していく責務を負う。レポートは一回書いて『後は知ったことではない』というものでは決してない」
> —— 日興アイ・アール「アナリストのつぶやき」

### バイサイド・アナリスト

**所属：** 運用会社（アセットマネジメント会社）、保険会社、銀行、年金基金など

**役割：** 自社の資産運用のためにレポートを作成。調査結果は社内のファンドマネージャーに還元される。

**特徴：**
- 調査結果は基本的に社内機密
- 対外的な発信はほとんどない
- 運用パフォーマンスに直結するため責任は重い
- セルサイドより「広く浅く」から「狭く深く」へ
- 年収は800万〜1,400万円（日系）、2,000万〜6,000万円（外資系・ヘッジファンド）

### 両者の決定的な違い——「何に賭けるか」

> 「セルサイドアナリストは予想の精度を問われ、バイサイドアナリストは市場との認識ギャップを重視する」
> —— 河北博光（元ファンドマネージャー）

**セルサイド：** 「当たる確率」が重要。多くの人が「そうなる」と思う方向を予測する。

**バイサイド：** 「リターン」が重要。多くの人が「そうならない」と思っていることが実現したときの大きなリターンを狙う。

この違いが、同じ企業を見ても異なる投資判断を下す理由となる。

## 1.3 必要なスキル・資格

### 必須資格

| 資格名 | 概要 | 難易度 |
|--------|------|--------|
| **CMA（日本証券アナリスト協会認定アナリスト）** | 日本で最も権威ある資格。1次・2次試験＋3年の実務経験が必要 | 合格率：1次約50%、2次約45% |
| **CFA（米国証券アナリスト）** | グローバルで認知度が高い。Level 1〜3の3段階 | 合格率：各レベル約40〜50% |
| **証券外務員（1種・2種）** | 証券会社での業務に必須の基礎資格 | 比較的容易 |

**CMA取得の流れ：**
1. 第1次レベル講座受講（6月〜翌1月）→ 受講料：54,000〜60,000円
2. 第1次試験受験（年2回：春・秋）→ 受験料：15,000円
3. 第2次レベル講座受講 → 受講料：57,000円
4. 第2次試験受験（年1回：6月）→ 受験料：15,000円
5. 3年以上の実務経験
6. CMA登録

**勉強時間の目安：** 1次試験150時間、2次試験100時間、合計約250時間

### 必要なスキル

| スキル | 重要度 | 詳細 |
|--------|--------|------|
| **財務分析力** | ◎ | 財務諸表を読み解き、企業の真の姿を見抜く |
| **数理的思考** | ◎ | バリュエーション、統計分析、モデル構築 |
| **コミュニケーション力** | ◎ | 経営者への取材、投資家へのプレゼン |
| **文章力** | ○ | レポート作成、複雑な分析を平易に伝える |
| **英語力** | ○ | 海外情報の収集、グローバル投資家との対話 |
| **体力・精神力** | ○ | 決算シーズンの激務、早朝出社 |

## 1.4 キャリアパス

### 典型的なキャリアステップ

```
【新卒〜3年目】ジュニアアナリスト
　↓
【4〜7年目】シニアアナリスト（セクター担当）
　↓
【8〜12年目】チーフアナリスト / リサーチヘッド
　↓
【それ以降】ファンドマネージャー / 経営層 / 独立
```

### キャリアの分岐点

**セルサイド → バイサイド：**
MiFID II規制（欧州の金融規制）の影響で、セルサイドからバイサイドへの転職が増加。セルサイドでの経験は、バイサイドで即戦力として評価される。

**独立の道：**
経験を積んだ後、フリーランスとして独立し、企業や投資家向けに分析・アドバイス業務を提供する道もある。

**他業界への転身：**
- FAS（財務アドバイザリー）：バリュエーションの専門性を活かす
- 事業会社の財務・IR部門
- コンサルティングファーム

# 第2章：1日のスケジュール

## 2.1 通常期の典型的な1日

### 【セルサイド・アナリストの場合】

```
05:30  起床、海外市場の動向チェック（米国市場終値、欧州の動き）
06:30  出社、朝のルーティン開始
07:00  朝会（モーニングミーティング）
       - 前日の海外市場まとめ
       - 本日の注目イベント共有
       - アナリスト間の情報交換
08:00  顧客向けモーニングコール（機関投資家への情報提供）
09:00  東証寄付き、マーケット監視
09:30  企業取材（電話または訪問）
12:00  昼食（社内または取材先で簡単に）
13:00  レポート執筆 / データ分析 / 業績モデル更新
15:00  東証大引け、本日の市場まとめ
16:00  投資家との面談 / カンファレンスコール
18:00  翌日の準備、追加リサーチ
19:00  退社（通常期）
       ※決算シーズンは22:00〜24:00も珍しくない
```

### 朝のルーティン——勝負は寄付き前に決まる

> 「トレーダーの一日の特徴は、通常の会社員よりも早い時間から仕事を開始すること。取引市場が朝9時に開始されるため、その前に市場の動向を確認し、情報を整理する必要がある」
> —— キャリアガーデン「トレーダーの1日」

アナリストも同様だ。**日本の深夜でも海外市場は動いている**。早朝に起きて、米国市場の終値、欧州の経済指標、為替の動きを確認することは必須のルーティンである。

### 日中の活動——取材と分析の繰り返し

午前中は主に情報収集。企業への電話取材、IR担当者との面談、業界関係者へのヒアリングを行う。

午後はレポート執筆とデータ分析。四半期ごとに業績モデルを更新し、投資判断を見直す。

### 夕方〜夜——投資家との対話

市場が閉まった後は、機関投資家との面談やカンファレンスコールが入る。**世界中の投資家との面談をこなし、深夜まで働くことも多い**。

---

## 2.2 決算発表シーズンの1日

決算発表シーズン（3月決算企業の場合は4〜5月、10〜11月）は、アナリストにとって最も過酷な時期だ。

### 【決算シーズンの1日】

```
05:00  起床、前日深夜発表の決算チェック
06:00  出社、決算速報レポート作成開始
07:00  朝会（決算発表企業の分析共有）
08:00  顧客向け緊急レポート配信
09:00  マーケット監視、株価反応の確認
10:00  決算説明会（午前の部）出席
12:00  昼食を取りながらレポート執筆
13:00  決算説明会（午後の部）出席
15:00  市場終了、本日の決算まとめ
16:00  追加取材（経営陣への1on1取材）
18:00  決算分析レポート執筆
21:00  本日発表分のレポート完成
22:00  翌日の決算発表企業の事前分析
24:00  退社（または会社で仮眠）
```

### 決算シーズンの現実

> 「頻繁な企業取材を通して四半期ごとに業績モデルを更新してレポートを書き、同時に、世界中の投資家との面談もこなし、深夜まで働くことも多い職業です」
> —— 日興アイ・アール「アナリストのつぶやき」

この時期、アナリストは**月に8本以上のレポート**を書くこともある。1本のレポートを書くために、決算短信の分析、業績モデルの更新、経営陣への取材、投資判断の見直しという一連の作業をこなさなければならない。

---

## 2.3 繁忙期と閑散期

### 繁忙期

| 時期 | 理由 |
|------|------|
| **4〜5月** | 3月決算企業の本決算発表 |
| **7〜8月** | 第1四半期決算発表 |
| **10〜11月** | 中間決算発表 |
| **1〜2月** | 第3四半期決算発表 |

### 閑散期

| 時期 | 過ごし方 |
|------|----------|
| **6月、9月、12月** | 中長期のテーマリサーチ、業界レポート作成、海外視察 |
| **年末年始** | 翌年の相場見通し作成、自己研鑽 |

閑散期は、普段手が回らない**中長期テーマの調査**や**新規カバレッジ企業の選定**に時間を使う。

# 第3章：情報収集の方法

## 3.1 使う情報源

### 一次情報（最も重要）

| 情報源 | 入手先 | 重要度 |
|--------|--------|--------|
| **決算短信** | 東証適時開示（TDnet） | ◎ |
| **有価証券報告書** | EDINET（金融庁） | ◎ |
| **決算説明会資料** | 各社IRサイト | ◎ |
| **中期経営計画** | 各社IRサイト | ◎ |
| **株主総会招集通知** | 各社IRサイト | ○ |
| **コーポレートガバナンス報告書** | 東証 | ○ |

> 「上場企業の有価証券報告書は無料で入手可能。一次情報なので信頼度は最高レベル」

### 二次情報

| 情報源 | 詳細 | 用途 |
|--------|------|------|
| **ブルームバーグターミナル** | 世界中の金融データ、リアルタイム株価、アナリストレポート | 必須ツール |
| **日経電子版** | 日本の経済ニュース、企業動向 | 毎日チェック |
| **会社四季報** | 東洋経済の企業情報誌、記者コメント | 銘柄発掘 |
| **業界専門誌** | 各業界の専門メディア | 深堀り調査 |

**ブルームバーグターミナルの威力：**
> 「ブルームバーグニュースでは、市場に影響を与えるニュースをリアルタイムで詳細に伝えます。また、ブローカーやその他の情報ソースが提供する優れたリサーチや一流アナリストによる独自コンテンツを入手することも可能です」

### その他の情報源

| 情報源 | 詳細 |
|--------|------|
| **業界団体資料** | 日本自動車工業会、電子情報技術産業協会など |
| **官公庁統計** | 経済産業省、財務省、日銀など |
| **消費者調査** | ニールセン、インテージなど |
| **特許情報** | J-PlatPat（特許庁） |
| **SNS・口コミ** | Twitter、転職サイトの口コミ |

## 3.2 情報の優先順位

### 情報収集の鉄則

**1. 一次情報を最優先**
決算短信、有価証券報告書など、企業が公式に発表した情報を最も重視する。

**2. 複数ソースで確認**
一つの情報源だけで判断しない。特に「噂」レベルの情報は、複数のルートで裏取りする。

**3. 時系列で追う**
単発の情報より、時系列の変化を追う。「過去3年の在庫推移」「5年間の研究開発費の推移」など。

### 見落としがちな情報源

| 情報源 | なぜ見落とすか | 発見できること |
|--------|----------------|----------------|
| **株主総会質疑応答** | 参加が面倒 | 経営陣の本音、株主の懸念 |
| **採用情報** | 投資と無関係に見える | 事業拡大の方向性、組織の課題 |
| **転職サイトの口コミ** | 信頼性が不明 | 社内の雰囲気、実際の労働環境 |
| **特許出願情報** | 技術的で読みにくい | 将来の製品開発の方向性 |
| **地方新聞** | 全国紙に載らない | 地方の工場閉鎖、地域への影響 |

---

## 3.3 情報を得るためのネットワーク構築

### アナリストのネットワーク

| 相手 | 得られる情報 |
|------|-------------|
| **IR担当者** | 公開情報の解釈、未公開だが開示可能な情報 |
| **経営者** | 経営方針、業界の見通し、競合の動向 |
| **業界の専門家** | 技術動向、業界構造の変化 |
| **同業他社のアナリスト** | 情報交換、異なる視点 |
| **取引先企業** | サプライチェーンの状況、需要動向 |
| **コンサルタント** | 業界の構造的な問題、ベストプラクティス |

### ネットワーク構築の現実

> 「IR部隊への電話番号やメールアドレスは、しっかりとわかりやすい場所に明記されていますか？いくら積極的な金融マンでも、よく知りもしない会社に、取締役宛てにいきなり電話をかける人はそう多くない」
> —— 日興アイ・アール「アナリストのつぶやき」

最初のアプローチは勇気がいる。しかし、**IRは投資家の問い合わせに対応することが仕事**であり、遠慮する必要はない。

# 第4章：企業訪問・決算説明会

## 4.1 準備すべきこと

### 事前リサーチの項目

| 項目 | 調べる内容 |
|------|-----------|
| **財務データ** | 過去5年の業績推移、セグメント別売上、利益率の変化 |
| **株価動向** | 過去1年の株価推移、バリュエーション（PER、PBR） |
| **競合比較** | 同業他社との比較、市場シェアの変化 |
| **経営陣** | 社長の経歴、経営方針、過去の発言 |
| **最新ニュース** | 直近3ヶ月のプレスリリース、報道 |
| **他のアナリストレポート** | 自社以外のアナリストの見方 |

> 「企業情報については、取材当日に調べてきたことを質問しても構いません。事前に興味を持ってよく調べてきたことが伝わると、相手から好印象を持ってもらえることが多いです」

---

## 4.2 質問の仕方——良い例、悪い例

### 良い質問の例

**オープンエンドで深掘りできる質問：**
- 「今期の中国市場の状況について、詳しく教えていただけますか？」
- 「御社の強みは〇〇だと理解していますが、競合との差別化要因をどうお考えですか？」
- 「中計の3年目ですが、進捗をどう評価されていますか？」

**数字の確認を促す質問：**
- 「営業利益の改善要因について、価格要因と数量要因の内訳を教えていただけますか？」
- 「設備投資の増額分は、どのセグメントに配分されるのでしょうか？」

### 悪い質問の例

**答えにくい質問：**
- 「株価はいつ上がりますか？」（→ IRは株価予想をしない）
- 「御社の株は買いですか？」（→ 投資判断は投資家が行う）
- 「ライバルの〇〇社についてどう思いますか？」（→ 他社批判は避ける）

**準備不足が露呈する質問：**
- 「御社は何をしている会社ですか？」（→ 事前に調べておくべき）
- 「前期の売上はいくらでしたか？」（→ 公開情報を質問するのは失礼）

---

## 4.3 聞くべきこと、聞いてはいけないこと

### 聞くべきこと

| カテゴリ | 質問例 |
|----------|--------|
| **業績の見通し** | 「下期の受注見通しはいかがですか？」 |
| **競争環境** | 「価格競争の状況に変化はありますか？」 |
| **投資計画** | 「M&Aの検討状況を教えてください」 |
| **リスク要因** | 「現在最も懸念されている経営課題は何ですか？」 |
| **株主還元** | 「配当方針に変更の予定はありますか？」 |

### 聞いてはいけないこと

| 禁止事項 | 理由 |
|----------|------|
| **未公開の業績情報** | インサイダー情報に該当する可能性 |
| **他社の機密情報** | 取引先や競合の機密は答えられない |
| **人事に関する詳細** | プライバシーに関わる |
| **訴訟の詳細** | 係争中の案件は開示制限がある |

---

## 4.4 経営者の発言から何を読み取るか

### 「言葉の裏」を読む技術

**楽観的すぎる発言は要注意：**
> 「予想以上に順調です」→ 具体的な数字を聞く
> 「計画通りに進んでいます」→ 計画自体が保守的でないか確認

**曖昧な表現に注目：**
- 「前向きに検討中」→ まだ決まっていない
- 「さまざまな選択肢を検討」→ 迷っている
- 「厳しい環境が続く」→ 具体的にどう厳しいのか深掘り

### プロの取材テクニック

> **取材者：**「営業利益はどのくらい伸びているのですか？」
> **IR担当：**「予想以上に順調です」
> **取材者：**「すごいですね。前年比で10％以上の伸びですか？」
> **IR：**「まあ、そうですね」
> **取材者：**「もしかすると20％以上ですか？」
> **IR：**「そうかもしれませんね」
> **取材者：**「もしかして50％以上ですか？」
> **IR：**「そこまでではないですけど、結構いい伸びです」
> **取材者：**「20%から30%の間ですか？」
> **IR：**「まあ、そのレンジの上に近いところかな」
>
> → 営業利益が25〜30％近く伸びていることが分かった
>
> —— ダイヤモンド・ザイ「プロ仕込みの電話取材のコツ」

**「10%ルール」と「30%ルール」：**
企業は、当初計画と比べて売上で上下10％以上、利益で上下30％以上、業績が異なる場合、修正を開示しなければならない。この閾値を意識した質問が有効。

# 第5章：レポートの書き方

## 5.1 アナリストレポートの構成

### 標準的な構成

```
【1ページ目】
- 表題（アナリストが伝えたいことを一言で）
- 投資判断（買い / 中立 / 売り）
- 目標株価
- 注目ポイントの要約

【2ページ目以降】
- 業績分析
- 業界動向
- バリュエーション
- リスク要因
- 業績予想表
```

### レポートの種類

| 種類 | 内容 | 頻度 |
|------|------|------|
| **イニシエーション・レポート** | 新規カバレッジ開始時の詳細分析 | 不定期 |
| **決算レビュー** | 決算発表後の分析 | 四半期ごと |
| **速報版** | 重要イベント発生時のコメント | 随時 |
| **業界レポート** | セクター全体の分析 | 月1〜2本 |

---

## 5.2 重視されるポイント

### 日本証券アナリスト協会の基準

> 「アナリスト・レポートには、綿密な調査・分析に基づく合理的かつ十分な根拠を持つこと、事実と意見を明確に区別すること、が求められる」
> —— 日本証券アナリスト協会「アナリスト・レポート作成の際の留意事項」

### 読まれるレポートの特徴

| 特徴 | 詳細 |
|------|------|
| **明確な投資判断** | 「買い」「売り」を曖昧にせず、根拠とともに示す |
| **独自の視点** | 他のアナリストとは異なる切り口、新たな発見 |
| **数字の裏付け** | 感覚ではなく、データで主張を支える |
| **タイムリーさ** | 情報の鮮度、速報性 |
| **読みやすさ** | 図表の活用、要点の明確化 |

---

## 5.3 よくある間違い

| 間違い | 問題点 |
|--------|--------|
| **事実と意見の混同** | 「売上は好調である」（→ 事実なのか意見なのか不明） |
| **根拠のない目標株価** | 計算過程が不透明 |
| **過度な断定** | 「必ず〇〇になる」という表現 |
| **誇大な表現** | 「革命的」「空前の」など煽り文句 |
| **リスク要因の軽視** | ポジティブ情報ばかりでリスクを記載しない |

# 第6章：リーマンショック時の経験

## 6.1 当時の状況——世界が崩壊した日

2008年9月15日、米国の大手投資銀行リーマン・ブラザーズが負債総額6,000億ドル超で破綻。「100年に一度」と言われる金融危機が始まった。

> 「リーマン破綻から2週間で経済の血液と言えるマネーが凍結し、世界中の中央銀行による大量資金供給という輸血だけでは立ち直れない心肺停止の一歩手前のような状況となり、間一髪で救われていたことに気付いていなかった人は、ある意味幸せだった」
> —— アセットマネジメントOne「リーマンショックの深層」

### 株価の暴落

| 指標 | 高値→安値 | 下落率 |
|------|-----------|--------|
| 日経平均 | 18,300円→7,055円 | **△61%** |
| NYダウ | 14,000ドル→6,500ドル | **△54%** |
| ドル円 | 110円→87円 | **△21%（円高）** |

---

## 6.2 アナリストは何を見落としたか

### 見落とし①：システミックリスクの過小評価

> 「サブプライムローン問題は認識していた。しかし、それが世界中の金融機関に波及し、システム全体を崩壊させるとは予想できなかった」

**教訓：** 個別企業の分析だけでなく、システム全体のリスクを見る視点が必要。

### 見落とし②：「Too Big to Fail」の過信

> 「リーマンは大きすぎて潰れないと思っていた。政府が救済すると信じていた」

**教訓：** 「ありえない」と思うことが起こりうる。最悪のシナリオを常に想定する。

### 見落とし③：流動性リスク

> 「平時は問題なく取引できる金融商品が、パニック時には買い手がつかなくなる」

**教訓：** 市場の流動性は一瞬で消える。

---

## 6.3 そこから学んだこと

### 教訓①：レバレッジの危険性

負債を積み上げて利益を拡大する経営モデルは、平時は高収益だが、危機時には脆い。

### 教訓②：相関リスクの怖さ

「分散投資しているから安心」は幻想。危機時には、あらゆる資産が同時に下落する。

### 教訓③：長期投資の重要性

> 「私自身、2008年の入社後すぐに始めたバランスファンドでの投信積立は、一時的に評価がマイナスになりました。2009年には1割ほど目減りしていましたが、マイナスを抱えながら積立を続けました。その結果、1年半ほどで資産がプラスに転じました」
> —— リーマンショック時に証券会社に入社した元社員

**相場は循環する。** 上がり続けることがない代わりに、ずっと下がり続けることもない。

# 第7章：若手アナリストへのアドバイス

## 7.1 最初の1年でやるべきこと

### 基礎固め

| やるべきこと | 詳細 |
|--------------|------|
| **財務諸表を読み込む** | カバー企業の有価証券報告書を最低3年分、隅々まで読む |
| **業績モデルを作る** | Excelで自分なりの業績予想モデルを構築する |
| **企業訪問に同行する** | 先輩アナリストの取材に同行し、質問の仕方を学ぶ |
| **資格取得** | CMA1次試験の合格を目指す |
| **英語力強化** | 海外の情報源にアクセスできるようにする |

### 担当セクターの知識

> 「実務経験は、証券アナリストとしてスキルを磨く上で欠かせない要素です。キャリアを積む中でレポート作成や企業訪問などの業務に携わり、セクターごとの専門知識を深めることで、次第に責任ある役割を担うことができるようになります」

---

## 7.2 避けるべき間違い

### 若手アナリストがやりがちなミス

| ミス | なぜ問題か | 対策 |
|------|-----------|------|
| **会社の言い分を鵜呑み** | 経営陣は常に楽観的なことを言う | 競合や取引先からも情報を取る |
| **直近の業績だけ見る** | 一時的な要因を見落とす | 最低5年の時系列で分析 |
| **株価予想に固執** | 予想が外れると信用を失う | 判断の根拠を明確にする |
| **コミュニケーション不足** | 社内の情報共有が疎か | 先輩・同僚と積極的に議論 |
| **健康管理の軽視** | 体調を崩すと仕事ができない | 睡眠と運動を確保 |

---

## 7.3 成長するための習慣

### 日々の習慣

| 習慣 | 効果 |
|------|------|
| **毎朝のニュースチェック** | 日経、Bloomberg、業界ニュースを30分 |
| **週1回のレポート執筆** | 文章力は書くことでしか向上しない |
| **月1回の新規企業訪問** | カバレッジ外の企業も見ることで視野が広がる |
| **四半期ごとの振り返り** | 自分の予想と実績の乖離を分析 |

### 長期的な成長のために

> 「証券アナリストとしてキャリアを積むには、継続的な学習とネットワークの構築が欠かせません。市場は常に変化しているため、最新の動向に対する知識を更新し続ける必要があります」

**専門性の深化：** 担当セクターでは「誰よりも詳しい」と言えるレベルを目指す。

**ネットワークの構築：** 投資家、企業、同業他社との関係を大切にする。

**独自の視点の確立：** 他のアナリストとは異なる「自分ならでは」の分析軸を持つ。

# 終章：アナリストの矜持

## 師匠が若手に伝えるべき言葉

> 「株価を当てることがアナリストの仕事だと思っているなら、君はまだ何もわかっていない」

> 「我々の仕事は、企業の真実を見抜くことだ。経営者が語らない本音、財務諸表の行間、業界の構造変化——それを読み解き、投資家に伝える」

> 「『買い』と書いた銘柄が下がったとき、君は何を感じる？悔しさか、恥ずかしさか。もし『なぜ間違えたのか』を考えないなら、君は成長しない」

> 「リーマンショックのとき、私は何も予測できなかった。あの経験が、私を変えた。『わからないことがある』と認める謙虚さこそが、アナリストに必要な資質だ」

> 「数字は嘘をつかない。しかし、嘘をつくために数字を使う人間はいる。だから、数字の裏を読め」
