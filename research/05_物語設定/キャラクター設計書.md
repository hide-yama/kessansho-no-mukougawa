## 1. 田中ミサキ（主人公）

### 1-1. 基本プロフィール

| 項目 | 内容 |
|------|------|
| **名前** | 田中ミサキ（たなか みさき） |
| **年齢** | 24歳 |
| **所属** | 東洋テック株式会社 経営企画部（異動して半年） |
| **前所属** | 同社 経理部（2年間） |
| **出身** | 地方の国立大学文学部（日本文学専攻） |
| **住まい** | 東京都内、一人暮らし |

### 1-2. 外見・印象

- 小柄で童顔。年齢より若く見られがち
- 真剣に話を聞くとき、少し首をかしげる癖がある
- 表情が豊か。考えていることが顔に出やすい
- 服装は清潔感のあるオフィスカジュアル

### 1-3. 性格

| 特徴 | 具体的な描写 |
|------|-------------|
| **負けず嫌い** | 表には出さないが、内心では「絶対にできるようになる」と思っている |
| **正直** | 嘘がつけない。わからないことを「わかった」と言えない |
| **本質を突く** | 専門用語に惑わされず、「つまりどういうことですか？」と聞ける |
| **自己評価が低い** | 「私なんかが経営企画部にいていいのか」と思っている |
| **成長意欲** | 苦手なことから逃げたくない。克服したいという意志がある |

### 1-4. バックストーリー

大学時代、経済学の講義で教授に「君の計算、全部間違ってるよ」と笑われたトラウマがある。以来、数字を見ると頭が真っ白になる。

就活では「人の話をよく聞く」「本質を見抜く質問ができる」と評価されて内定。なぜか経理部に配属されて絶望したが、先輩に助けられながら2年間生き延びた。

経営企画部への異動は、元上司・佐々木課長の推薦。「田中は数字より人を見ろ。だが経営企画なら、両方必要だ。苦手を克服しろ」

### 1-5. ミッション

**3ヶ月後の経営会議で、自社（東洋テック）の分析レポートを発表する**

これがミサキに課せられた試練であり、物語の軸となる。

### 1-6. 成長アーク

```
第1話：恐怖 → かすかな希望
  「数字が怖い」→「この人に教われば、私も変われるかも」

第5話：理解 → 小さな自信
  「なるほど、そういうことか」→「少しずつわかってきた」

第9話：応用 → 成長の実感
  「自分で分析できるようになった」→「もっと知りたい」

第11話：発見 → 衝撃
  「この数字、おかしくない？」→「これは見過ごせない」

第13話：行動 → 変身完了
  「私が言わなければ」→「数字が読める人になった」
```

### 1-7. 口癖・特徴的なセリフ

- 「正直に言っていいですか？」
- 「ちょっと待ってください」
- 「それって、つまり——」
- 「わかったふりだけはしたくないんです」
- （心の声）「また数字だ。でも、逃げたくない」

### 1-8. 読者との接点

ミサキは「数字が苦手な読者」の分身。彼女の疑問は読者の疑問であり、彼女の成長は読者の成長を体現する。

**読者がミサキに共感するポイント**：
- 数字への苦手意識
- 「わからない」と言えない恥ずかしさ
- でも、このままじゃダメだという焦り
- 成長したいという意志

## 2. 安藤誠一（メンター）

### 2-1. 基本プロフィール

| 項目 | 内容 |
|------|------|
| **名前** | 安藤誠一（あんどう せいいち） |
| **年齢** | 58歳 |
| **経歴** | 野村證券（25年）→ 独立系投資顧問（8年）→ フリーのアドバイザー |
| **肩書** | 元シニアアナリスト、現アドバイザー |
| **東洋テックとの関係** | 社外取締役候補として関わり始める |
| **家族** | 妻と二人暮らし。娘（30歳）は独立 |

### 2-2. 外見・印象

- 中肉中背。白髪交じりの短髪
- 穏やかな目元。話を聞くとき、相手の目をじっと見る
- 服装は落ち着いたジャケットスタイル
- 声は低めで落ち着いている。急かさない話し方

### 2-3. 性格

| 特徴 | 具体的な描写 |
|------|-------------|
| **穏やか** | 声を荒げない。どんな質問にも落ち着いて答える |
| **核心を突く** | 穏やかな口調だが、問いかけは本質をえぐる |
| **気づかせる** | 答えを教えるのではなく、相手が自分で気づくように導く |
| **謙虚** | 自分の失敗談を隠さない。むしろ積極的に話す |
| **忍耐強い** | 相手のペースに合わせる。焦らせない |
| **時々遠い目をする** | 過去の失敗を思い出すとき、ふと表情が曇る |

### 2-4. バックストーリー

証券会社時代、優秀なアナリストとして活躍。数字を読む力には自信があった。

**転機：リーマンショック（2008年）**

担当銘柄の分析で「問題ない」と判断。その銘柄で大損した個人投資家から電話を受けた。

> 「安藤さん、あなたを信じていたのに……」

泣きながら話す声を聞いて、安藤は気づいた。

> 「私は数字だけを見て、数字の向こうにある人間を見ていなかった」

その日から、安藤は変わった。数字の「読み方」ではなく「活かし方」を教えることに情熱を注ぐようになった。

### 2-5. 教え方のスタイル

**「教える」のではなく「気づかせる」**

| 従来の教え方 | 安藤先生のスタイル |
|-------------|-------------------|
| 答えを教える | 問いかけて考えさせる |
| 知識を伝える | 体験させて実感させる |
| 間違いを指摘する | 「私も同じ間違いをした」と共感する |
| 急いで進める | 相手のペースに合わせる |

### 2-6. 口癖・特徴的なセリフ

**基本パターン**：
- 「数字は嘘をつかない。でも、見方を知らないと何も語ってくれない」
- 「君はどう思う？」（穏やかに、急かさず）
- 「いい質問だね」（肯定から入る）
- 「私も昔、同じ間違いをしたよ」（共感を示す）
- 「そう、そこなんだよ」（静かに、しかし確信を持って）

**場面別のセリフ例**：

| 場面 | セリフ |
|------|--------|
| ミサキが質問したとき | 「いい質問だね。実はね——」 |
| ミサキが間違えたとき | 「惜しいね。もう少し考えてみよう」 |
| ミサキが正解したとき | 「そう、そこに気づけたのは大きいよ」 |
| 核心を伝えるとき | 「大事なのはね——」（少し間を置いて） |
| 過去を振り返るとき | 「私も昔……」（遠い目をして） |

### 2-7. ミサキとの関係

**師弟関係だが、上下関係ではない**

安藤はミサキを「教え子」ではなく「一緒に考える相手」として扱う。ミサキの疑問を尊重し、彼女自身が答えにたどり着くのを待つ。

**安藤がミサキに見出したもの**：
- 本質を突く質問ができる素直さ
- わからないことを「わからない」と言える正直さ
- かつての自分にはなかった謙虚さ

### 2-8. 物語における役割

1. **知識の提供者**：財務分析のスキルを伝える
2. **気づきの促進者**：問いかけによってミサキを導く
3. **失敗の共有者**：自分の失敗談を通じて「数字の向こう側」を伝える
4. **最後の背中押し**：第12話でミサキの決断を支える

## 3. 黒田専務（敵役）

### 3-1. 基本プロフィール

| 項目 | 内容 |
|------|------|
| **名前** | 黒田慎太郎（くろだ しんたろう） |
| **年齢** | 60歳 |
| **役職** | 東洋テック株式会社 専務取締役 海外事業本部長 |
| **経歴** | 東洋テック一筋40年。営業→海外駐在→本部長→専務 |
| **社内評価** | 「海外事業を立て直した功労者」として次期社長の最有力候補 |

### 3-2. 外見・印象

- 長身で威圧感がある。銀縁眼鏡
- 声が大きく、会議では場を支配する
- 自信に満ちた立ち居振る舞い
- 部下には厳しいが、上層部には愛想がいい

### 3-3. 表の顔

**「海外事業を立て直した男」**

- 5年前、赤字だった海外事業部門を黒字化
- 経営会議では常に好調な数字を報告
- 社長からの信頼が厚い
- 「結果を出す男」として社内で一目置かれている

### 3-4. 裏の顔

**数字を操作して自分の手柄を作っていた**

| 不正の手口 | 財務上の兆候 | 学べるスキル |
|-----------|-------------|-------------|
| 架空売上の計上 | 売掛金の異常な膨張 | 売上債権回転期間の分析 |
| 在庫の水増し | 棚卸資産の急増 | 在庫回転期間の分析 |
| 費用の先送り | 利益とCFの乖離 | CF計算書の読み方 |
| 関連会社への付け替え | 関連当事者取引の急増 | 有報の注記の読み方 |

### 3-5. 動機

**「次期社長になるために、どんな手段も使う」**

- 若い頃から出世欲が強い
- 海外赴任時代、業績が振るわず焦った
- 「数字さえ良ければ評価される」と学んだ
- 最初は小さな操作だったが、徐々にエスカレート
- 今では後戻りできないほど深みにはまっている

### 3-6. 性格

| 特徴 | 具体的な描写 |
|------|-------------|
| **自己中心的** | 会社より自分の出世が大事 |
| **狡猾** | 証拠を残さないよう慎重に動く |
| **威圧的** | 反論する部下を黙らせる |
| **表裏がある** | 上層部には愛想がいいが、部下には冷酷 |
| **追い詰められると弱い** | 証拠を突きつけられると動揺する |

### 3-7. 物語での登場

| 話数 | 登場シーン |
|------|-----------|
| 第1話 | 経営会議で好調な数字を報告（伏線） |
| 第10話 | ミサキの前で自信満々に振る舞う |
| 第11話 | ミサキが黒田の部門の数字に違和感を覚える |
| 第12話 | ミサキと安藤が証拠を集める（黒田は登場しないが存在感） |
| 第13話 | 経営会議で告発され、最終的に土下座 |

### 3-8. 告発シーン（第13話）での描写

**段階的な崩壊**：

1. **余裕** → ミサキの発表を軽視
2. **動揺** → 数字の矛盾を指摘されて顔色が変わる
3. **反論** → 「それは誤解だ」と否定
4. **崩壊** → 証拠を突きつけられて言葉を失う
5. **土下座** → 社長の追及に耐えられず崩れ落ちる

## 7. 執筆時の注意点

### ミサキを描くとき
- 「数字が苦手」を誇張しすぎない。読者が共感できる程度に
- 成長は少しずつ。一気に賢くなりすぎない
- 心の声を効果的に使って、読者との一体感を作る

### 安藤先生を描くとき
- 穏やかさと核心を突く鋭さのバランス
- 「教える」のではなく「問いかける」
- 過去の失敗への後悔を時々見せる

### 黒田専務を描くとき
- 序盤は「有能な経営者」として描く（読者を騙す）
- 不正の兆候は数字に現れる（読者も気づけるように）
- 最後の崩壊は一気に。カタルシスを重視
