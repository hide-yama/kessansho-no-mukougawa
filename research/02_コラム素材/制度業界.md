## 1.2 レポートが作られた背景：当時の日本企業の問題点

### 主な課題

1. **低収益性の長期化**
   - 日本企業のROE・売上高営業利益率は欧米企業の約半分
   - この傾向が20年以上継続

2. **株式リターンの低迷**
   - 過去20年間で上場企業約1,600社のうち、株式リターンがプラスになった企業は約200社のみ

3. **ダブルスタンダード経営**
   - 投資家向けにはROEをアピールしながら、社内では異なる指標を使用
   - 企業と投資家の認識ギャップが拡大

4. **長期投資家の不在**
   - 低収益→長期投資家が育たない→イノベーションが生まれにくい、という悪循環

---

## 1.3 「ROE8%」の根拠

### なぜ8%なのか

伊藤レポートでは、**株主資本コスト（COE: Cost of Equity）** を根拠としています。

```
【ROE8%の算出根拠】

■ 株主資本コスト（COE）の調査結果
  - 日本株に投資する機関投資家が期待するリターン：約7%以上

■ 価値創造の条件
  - ROE > COE（株主資本コスト）であれば、企業は株主価値を創造している
  - ROE < COE であれば、株主価値を毀損している

■ 結論
  - 最低ライン：ROE 8%（第1ステップ）
  - 目標水準：ROE 10%以上（第2ステップ）
```

### 国際比較（当時のデータ）

| 地域 | 平均ROE | ROE8%未満の企業割合 |
|------|---------|---------------------|
| 日本（TOPIX500） | 約5% | 約40% |
| 米国（S&P500） | 約15% | 約14% |
| 欧州（STOXX600） | 約12% | 約19% |

**要因分析**：日本企業の低ROEの主因は「レバレッジ」ではなく、「売上高利益率（事業の収益力）」の低さ。

## 1.4 主な提言内容

### 企業への提言

1. **資本コストを意識した経営**
   - ROE8%を最低水準として目指す
   - 手厚い内部留保の見直し

2. **CFO人材の育成**
   - 経営者とともに企業価値向上を目指す有能なCFOの確保

3. **投資家との対話強化**
   - 「緊張と協調」の両側面を満たした双方向の対話

### 投資家への提言

1. 単にROE目標を押しつけるだけでなく、中長期的な観点から対話を深める
2. 企業価値を評価する能力を高める
3. 「静かな決別」（対話なき撤退）を回避する

## 1.7 批判や課題

### 主な批判

1. **「ROE8%」の一人歩き**
   - 本来の趣旨（長期的な企業価値向上）ではなく、数値目標だけが注目された
   - 短期的なROE向上策（自社株買い、リキャップCB等）に走る企業も

   - 従業員・取引先など他のステークホルダーへの配慮が不足するのではないか

3. **達成期限の曖昧さ**
   - 伊藤教授自身は「中長期的な目標」と説明したが、短期目標と誤解された

- 米国企業との格差は依然として約2倍
- ROE15%以上の企業割合：日本19% vs 米国61% vs 欧州49%
- 形式的なガバナンス整備から「実質化」への移行が必要

# 2. PBR1倍問題と東証の要請（2023年〜）

## 2.1 問題の背景

### PBRとは

**PBR（Price Book-value Ratio：株価純資産倍率）** = 株価 ÷ 1株当たり純資産（BPS）

- **PBR1倍**：株価 = 純資産（解散価値と等価）
- **PBR1倍割れ**：株価 < 純資産 → 「会社を解散したほうが株主にとって得」という評価

### 東証プライム市場の実態（2023年時点）

| 指標 | プライム市場 | スタンダード市場 |
|------|--------------|------------------|
| PBR1倍割れ企業の割合 | **約50%** | **約60%** |
| ROE8%未満企業の割合 | 約40% | — |

### なぜこれが問題なのか

1. **資本効率の低さの象徴**
   - 株主から預かった資本を効率的に活用できていない

2. **国際比較での見劣り**
   - 米国・欧州ではPBR1倍割れ企業は少数派

3. **市場改革の停滞**
   - 2015年のコーポレートガバナンス・コード以降も、本質的な改善が進まない

---

## 2.2 東証の要請内容（2023年3月31日）

### 正式名称

「資本コストや株価を意識した経営の実現に向けた対応について」

### 要請の3つの柱

| ステップ | 内容 |
|----------|------|
| **①現状分析** | 自社の資本コスト・資本収益性を把握・分析 |
| **②計画策定・開示** | 改善に向けた方針・具体的取組・進捗状況を開示 |
| **③株主との対話** | 投資家との建設的な対話を通じて取組を磨き上げ |

### 対象

- プライム市場・スタンダード市場の**全上場企業**
- PBR1倍以上の企業も対象（「PBR1倍割れ企業だけ」ではない点に注意）

### 開示状況の公表

- 2024年1月より、対応を開示している企業の一覧表を毎月公表
- 「見える化」により企業の取組を後押し

---

# 4. なぜ銀行の自己資本比率は低いのか

## 4.1 銀行のビジネスモデルの特性

### 一般企業との違い

| 項目 | 銀行 | 製造業 |
|------|------|--------|
| **主な資産** | 貸出金（他人の預金を原資） | 工場・設備・在庫 |
| **負債の性質** | 預金（顧客から預かったお金） | 借入金・社債 |
| **レバレッジ** | **極めて高い**（預金で資金調達） | 相対的に低い |
| **収益モデル** | 金利差（預金金利 < 貸出金利） | 製品販売 |

### 銀行の自己資本比率が低い理由

```
【銀行のバランスシート構造】

資産（100%）           負債・純資産（100%）
├─ 貸出金：70%        ├─ 預金：90%（他人資本）
├─ 有価証券：20%      ├─ 借入金：5%
└─ その他：10%        └─ 自己資本：5%（純資産）

→ 預金という「他人資本」を活用して貸出を行うビジネスモデル
→ 自己資本比率は構造的に低くなる
```

---

## 4.2 BIS規制（バーゼル規制）の概要

### BIS規制とは

**BIS規制**（バーゼル規制）とは、バーゼル銀行監督委員会が策定した、国際的に活動する銀行の自己資本比率等に関する国際統一基準。

### バーゼルI、II、IIIの違い

| 規制 | 公表年 | 主な内容 |
|------|--------|----------|
| **バーゼルI** | 1988年 | 自己資本比率8%以上の基準を設定。信用リスクのみ考慮 |
| **バーゼルII** | 2004年 | 信用リスクの計測方法を精緻化。オペレーショナルリスクを追加 |
| **バーゼルIII** | 2010年〜 | 自己資本の「質」を厳格化。流動性比率・レバレッジ比率を導入 |

### 自己資本比率の計算方法

```
           自己資本
自己資本比率 = ─────────────────
         リスク・アセット

■ 分子：自己資本
  ├─ CET1（普通株式等Tier1）：普通株式、内部留保
  ├─ その他Tier1：優先株式等
  └─ Tier2：劣後債、一般貸倒引当金

■ 分母：リスク・アセット
  = 資産 × リスクウェイト（資産の種類ごとに設定）
  
  例）国債：0%、事業法人：100%、住宅ローン：35%
```

---

## 4.3 規制上の「正常値」

### 国際統一基準（海外営業拠点を有する銀行）

| 項目 | 最低基準 |
|------|----------|
| 総自己資本比率 | **8%以上** |
| Tier1比率 | 6%以上 |
| CET1比率 | 4.5%以上 |
| 資本保全バッファー | +2.5%（上乗せ） |

→ 実質的には**10.5%以上**が求められる

### 国内基準（海外営業拠点を持たない銀行）

- 自己資本比率：**4%以上**

| 比較項目 | 銀行 | 製造業 |

## 5.2 業種別の特徴と解説

### 製造業

- **自己資本比率が高い理由**：設備投資のための安定した財務基盤が必要
- **ROEの目安**：8〜12%（伊藤レポートの基準をクリア）
- **営業利益率**：素材系は低め（3〜5%）、精密機器は高め（10〜15%）

### 小売業

- **自己資本比率**：在庫回転が早く、借入依存度は業態により異なる
- **営業利益率が低い理由**：薄利多売モデル、価格競争が激しい
- **ROE**：効率的な在庫管理ができれば高くなる

### 銀行

- **自己資本比率が低い理由**：預金を活用したレバレッジ経営
- **ROAが低い理由**：総資産（貸出金）が巨額なため、分母が大きい
- **ROE**：レバレッジにより、ROAの割にROEは高くなりやすい

### 不動産

- **営業利益率が高い理由**：賃貸収入は限界費用が低い
- **自己資本比率**：有利子負債を活用した投資が一般的
- **景気感応度**：不動産市況により業績変動が大きい

### IT・通信

- **自己資本比率が高い理由**：固定資産が少なく、キャッシュリッチな企業が多い
- **営業利益率が高い理由**：限界費用が低い（特にソフトウェア）
- **ROEの高さ**：資本効率が良く、高収益企業が多い

### 商社

- **ROEが高い理由**：投資事業・資源事業からの配当収入、レバレッジ活用
- **営業利益率が低い理由**：取引仲介（口銭ビジネス）が中心
- **特徴**：近年は事業投資型に移行し、ROE向上
