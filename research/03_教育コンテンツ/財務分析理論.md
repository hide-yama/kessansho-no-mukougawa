# 財務分析の基礎理論

財務分析の核心は「利益」と「現金」という2つの視点から企業を立体的に理解することにあります。本レポートでは、**ROEを3つの要素に分解するデュポン分析**、**キャッシュフロー計算書の実践的な読み方**、**財務三表の連動関係**、そして**業種ごとの適正水準**を、日本企業の具体的な数値を用いて解説します。

---

# 第1章：デュポン分解の理論と実践

## 1.1 なぜ「デュポン」と呼ぶのか ― 100年前の革新

デュポン分解（DuPont Analysis）は、1910年代にアメリカの化学大手**DuPont社**で開発された財務分析手法です。

### 開発者：F.ドナルドソン・ブラウン

| 項目 | 内容 |
|------|------|
| **開発者** | F. Donaldson Brown（1885-1965） |
| **入社** | 1908年、デュポン社に入社 |
| **開発時期** | 1912年に効率性分析のレポートを上司に提出、その中にROI計算式の原型を記載 |
| **体系化** | 1914年に計算式を正式に体系化 |
| **普及** | 1921年、GM（ゼネラルモーターズ）のCFOとして導入。アルフレッド・スローンCEOの経営改革と結びつき全米に普及 |
| **評価** | 2022年、アメリカ会計殿堂入り |

ブラウンが開発したこの手法は、**「20世紀アメリカ企業経営における最も重要な革新の一つ」**と評価されています。

---

## 1.2 計算式の構造：売上高で「分解」、総資産で「つなぐ」

### 基本式

```
ROE = 売上高純利益率 × 総資産回転率 × 財務レバレッジ
```

### 各要素の計算式

| 要素 | 計算式 | 測定するもの |
|------|--------|-------------|
| **売上高純利益率** | 当期純利益 ÷ 売上高 | 収益性（稼ぐ力） |
| **総資産回転率** | 売上高 ÷ 総資産 | 効率性（回す力） |
| **財務レバレッジ** | 総資産 ÷ 自己資本 | 資本構成（借りる力） |

### 数学的証明

この3要素を掛け合わせると、途中の「売上高」と「総資産」が約分され、最終的に**当期純利益÷自己資本=ROE**になります。

```
   純利益      売上高      総資産       純利益
  ────── × ────── × ────── = ────── = ROE
   売上高      総資産     自己資本     自己資本
```

この分解により、「ROEが高い（または低い）理由」が一目でわかるようになります。

---

## 1.3 各要素の意味と改善方法

### ① 売上高純利益率（収益性）

「売上1円あたりどれだけ最終利益を残せるか」を示します。

| 項目 | 内容 |
|------|------|
| **改善方法** | 商品の付加価値向上、原価率の低減、販管費の効率化 |
| **業種別目安** | 高級ブランド・IT：10〜30%、小売業：2〜5% |

### ② 総資産回転率（効率性）

「資産をどれだけ効率よく売上に転換しているか」を測ります。

| 項目 | 内容 |
|------|------|
| **改善方法** | 在庫回転率の向上、売掛金回収の早期化、遊休資産の売却 |
| **業種別目安** | 小売・卸売業：2〜4回転、製造業・不動産業：0.3〜1回転 |

### ③ 財務レバレッジ（資本構成）

「自己資本の何倍の資産を持っているか」を表します。

| 自己資本比率 | 財務レバレッジ |
|-------------|---------------|
| 50% | 2倍 |
| 25% | 4倍 |
| 10% | 10倍 |

**注意点**：レバレッジが高いほどROEは向上しますが、借入依存度が上がり財務リスクも増大します。

## 1.4 トヨタ自動車のデュポン分解（2024年3月期）

製造業の代表としてトヨタ自動車を分析します。

### 基礎データ

| 項目 | 金額 |
|------|------|
| 売上高（営業収益） | **45兆953億円** |
| 当期純利益 | **4兆9,449億円** |
| 総資産 | **90兆1,143億円** |
| 自己資本 | **35兆2,394億円** |
| ROE（実績） | **15.8%** |

### デュポン分解の計算

| 要素 | 計算 | 結果 |
|------|------|------|
| 売上高純利益率 | 4.9兆÷45.1兆 | **10.97%** |
| 総資産回転率 | 45.1兆÷90.1兆 | **0.50回** |
| 財務レバレッジ | 90.1兆÷35.2兆 | **2.56倍** |
| 検証（3要素の積） | 10.97%×0.50×2.56 | **約14%** |

### 分析結果

- **売上高純利益率10.97%**：自動車業界として非常に高い水準。ハイブリッド車の高収益性と原価改善努力の成果
- **総資産回転率0.50回**：製造業として標準的。大規模工場設備と金融子会社のリース資産が総資産を押し上げ
- **財務レバレッジ2.56倍**：自己資本比率38%に相当。金融事業を含む製造業として健全な水準

## 1.5 ファーストリテイリングのデュポン分解（2024年8月期）

小売業の代表としてユニクロを展開するファーストリテイリングを見てみましょう。

### 基礎データ

| 項目 | 金額 |
|------|------|
| 売上収益 | **3兆1,038億円** |
| 当期純利益 | **3,720億円** |
| 総資産 | **3兆5,875億円** |
| 自己資本 | **2兆165億円** |
| ROE（実績） | **18.45%** |

### デュポン分解の計算

| 要素 | 計算 | 結果 |
|------|------|------|
| 売上高純利益率 | 3,720億÷31,038億 | **11.98%** |
| 総資産回転率 | 31,038億÷35,875億 | **0.87回** |
| 財務レバレッジ | 35,875億÷20,165億 | **1.78倍** |
| 検証（3要素の積） | 11.98%×0.87×1.78 | **18.5%** |

### 分析結果

- **売上高純利益率11.98%**：SPA（製造小売）モデルにより中間マージンを排除、粗利益率約54%を実現
- **財務レバレッジ1.78倍**：自己資本比率56%と保守的な財務戦略
- **特徴**：高い収益性と効率性で**ROE 18%超**を達成

# 第2章：キャッシュフロー計算書を読み解く

## 2.1 直接法と間接法：なぜ日本企業の99%が間接法を採用するのか

キャッシュフロー計算書には**直接法**と**間接法**の2つの作成方法があります。

### 直接法

| 項目 | 内容 |
|------|------|
| **概要** | 営業活動による現金収支を取引ごとに総額で表示 |
| **イメージ** | 「営業収入○○円、商品仕入支出△△円」のような「企業版家計簿」 |
| **メリット** | 現金の流れが直接わかる |
| **デメリット** | 作成に手間がかかる |
| **推奨** | 国際会計基準（IFRS）では推奨 |

### 間接法

| 項目 | 内容 |
|------|------|
| **概要** | 税引前当期純利益を出発点に、非現金項目や運転資本の増減を加減算して営業CFを算出 |
| **メリット** | 「利益とキャッシュのズレ」を可視化できる、作成の負担が軽い |
| **採用率** | 日本の上場企業の**約99%**が採用 |

---

## 2.2 営業キャッシュフロー：「本業で稼ぐ力」の真実

### 間接法による営業CFの主な構成要素

| 項目 | 加減算 | 理由 |
|------|--------|------|
| 税引前当期純利益 | 起点 | P/Lの利益からスタート |
| 減価償却費 | **＋加算** | 現金支出を伴わない費用 |
| 売上債権の増加 | **△減算** | 売上は計上したが現金未回収 |
| 棚卸資産の増加 | **△減算** | 仕入で現金が流出済み |
| 仕入債務の増加 | **＋加算** | 仕入はしたが現金未払い |

### 営業CFのプラス・マイナスの意味

| 状態 | 意味 |
|------|------|
| **プラス** | 本業で現金を稼げている。設備投資や借入返済に回す余力がある |
| **マイナス** | 本業で現金を生み出せていない。継続すると資金繰りが悪化 |

### 「営業利益と営業CFは異なる」― 具体例

**例**：売上100万円を全額掛け売り（翌月回収）した場合

| 財務諸表 | 影響 |
|----------|------|
| P/L | 利益が計上される |
| 営業CF | **ゼロ**（売掛金が増加した分、利益から差し引かれる） |

**結論**：利益が出ていても、現金が増えているとは限らない

---

## 2.3 投資キャッシュフロー：マイナスが「良い」とき「悪い」とき

### 投資CFの主な構成要素

- 有形固定資産の取得・売却
- 投資有価証券の取得・売却
- 子会社株式の取得

### 投資CFの解釈

| パターン | 状況 | 良い/悪い |
|----------|------|-----------|
| **マイナス（良い）** | 成長投資・設備投資を積極的に実施 | ◎ 成長企業は将来の収益拡大のためマイナスが通常 |
| **マイナス（悪い）** | 投資効果が見込めない非効率な支出 | × 回収困難な投資 |
| **プラス（良い）** | 計画的な事業再編・戦略的な資産売却 | ○ 選択と集中 |
| **プラス（悪い）** | 資金繰り悪化による資産の投げ売り | × リストラによる切り売り |

**重要**：投資CFのプラス・マイナスだけでなく、**「なぜそうなったか」**を分析することが重要

---

## 2.4 財務キャッシュフロー：資金調達と株主還元のバランス

### 財務CFの主な構成要素

- 借入金の調達・返済
- 社債の発行・償還
- 配当金の支払い
- 自社株買い

### 財務CFの解釈

| 状態 | 意味 | 注意点 |
|------|------|--------|
| **プラス** | 資金調達（借入増加、増資）が返済を上回る | 成長投資のための調達か、資金繰り悪化による借入増加かを見極める |
| **マイナス** | 返済や株主還元が調達を上回る | 一般的には**マイナスが健全**（借入返済や配当ができている） |

---

## 2.5 フリーキャッシュフロー：企業が「自由に使える」お金

### 計算式

```
フリーキャッシュフロー（FCF） = 営業CF + 投資CF
```

### 具体例

| 項目 | 金額 |
|------|------|
| 営業CF | ＋800万円 |
| 投資CF | △300万円 |
| **FCF** | **＋500万円** |

→「本業で稼いだ800万円から投資300万円を引いても、500万円の余裕資金がある」

### FCFが重要な理由

| 理由 | 説明 |
|------|------|
| **財務健全性の指標** | 借入返済・配当支払い・新規投資に回せる余力を示す |
| **企業価値評価（DCF法）** | 将来のFCFを現在価値に割り引いて企業価値を算出 |

---

## 2.6 企業のライフサイクルをCFパターンで読み解く

営業CF・投資CF・財務CFの組み合わせパターンから、企業のライフサイクル段階を推測できます。

### CFパターン一覧表

| ライフサイクル | 営業CF | 投資CF | 財務CF | 特徴 |
|---------------|:------:|:------:|:------:|------|
| **創業期** | △ | △ | ＋ | 本業は赤字、積極投資、借入・増資で資金調達 |
| **成長期** | ＋ | △ | ＋ | 本業で稼ぎ始めるが、成長投資のため更に資金調達 |
| **成熟期** | ＋ | △ | △ | 本業で安定的に稼ぎ、投資も賄い、返済・株主還元も実施 |
| **衰退期①** | △ | ＋ | △ | 本業が厳しく、資産売却で現金確保、返済優先 |
| **衰退期②** | △ | ＋ | ＋ | 本業が厳しく、資産売却しつつ存続のため借入増加 |

### 最も健全な「成熟期」パターン

```
営業CF：＋（本業で十分な現金を稼ぐ）
投資CF：△（その範囲内で投資を行う）
財務CF：△（借入返済や配当も実施）
```

# 第3章：財務三表はこうつながっている

## 3.1 P/L → B/S：利益は純資産に蓄積される

損益計算書（P/L）で計算された**当期純利益**は、貸借対照表（B/S）の**利益剰余金**に加算されます。

### 計算式

```
期末利益剰余金 = 期首利益剰余金 + 当期純利益 − 配当金
```

### 具体例

| 項目 | 金額 |
|------|------|
| 期首利益剰余金 | 1,000万円 |
| 当期純利益 | 200万円 |
| 配当金 | 50万円 |
| **期末利益剰余金** | **1,150万円** |

### ポイント

- 黒字を続ければ利益剰余金が積み上がり、自己資本が充実
- 赤字が続けば利益剰余金が減少し、最悪の場合は債務超過に

---

## 3.2 B/S → C/F：資産負債の増減が現金の動きを説明する

間接法によるC/F作成では、B/S各科目の増減を調整することで現金の増減を算出します。

### 基本ルール

| B/Sの変動 | CFへの影響 | 理由 |
|-----------|-----------|------|
| 資産が増加 | CFは**マイナス** | 現金が他の資産に変わった |
| 資産が減少 | CFは**プラス** | 他の資産が現金に変わった |
| 負債が増加 | CFは**プラス** | 現金を払わずに済んだ |
| 負債が減少 | CFは**マイナス** | 現金を支払った |

### 具体例

| B/S科目の変動 | CFへの影響 | 理由 |
|--------------|-----------|------|
| 売掛金が50万円増加 | 営業CF **△50万円** | 売上は計上したが現金未回収 |
| 棚卸資産が20万円減少 | 営業CF **＋20万円** | 在庫が売れて現金化された |
| 買掛金が20万円増加 | 営業CF **＋20万円** | 仕入はしたが現金未払い |
| 借入金が100万円減少 | 財務CF **△100万円** | 返済で現金が流出 |

---

## 3.3 C/F → B/S：現金残高は必ず一致する

### 計算式

```
期首現金 + 営業CF + 投資CF + 財務CF = 期末現金
```

### 具体例

| 項目 | 金額 |
|------|------|
| 期首現金 | 100万円 |
| 営業CF | ＋80万円 |
| 投資CF | △30万円 |
| 財務CF | △20万円 |
| **期末現金** | **130万円** |

この130万円がB/Sの「現金及び預金」と一致することで、三表の整合性が確認できます。

---

## 3.4 具体的な取引で三表の連動を確認する

### 取引1：商品（原価60万円）を100万円で掛け販売

| 財務諸表 | 影響 |
|----------|------|
| **P/L** | 売上＋100万円、原価＋60万円 → 利益＋40万円 |
| **B/S** | 売掛金＋100万円、棚卸資産△60万円、利益剰余金＋40万円 |
| **C/F** | 営業CF：利益＋40万円、売掛金増加△100万円、在庫減少＋60万円 → **実質±0** |

**ポイント**：利益は出ているが現金は増えていない ― これが「利益≠現金」の典型例

### 取引2：機械1,000万円を現金で購入

| 財務諸表 | 影響 |
|----------|------|
| **P/L** | 購入時は影響なし（減価償却時に費用化） |
| **B/S** | 現金△1,000万円、固定資産＋1,000万円 |
| **C/F** | 投資CF△1,000万円 |

### 取引3：減価償却費100万円を計上

| 財務諸表 | 影響 |
|----------|------|
| **P/L** | 減価償却費＋100万円 → 利益△100万円 |
| **B/S** | 固定資産△100万円、利益剰余金△100万円 |
| **C/F** | 営業CF：利益△100万円に減価償却費＋100万円を加算 → **現金影響なし** |

**ポイント**：減価償却は**現金支出を伴わない費用**のため、CFでは加算して調整。これが間接法の核心

## 3.5 三表連動の図解

```
┌─────────────────────────────────────────────────────────────┐
│                    【財務三表の連動関係】                     │
└─────────────────────────────────────────────────────────────┘

┌──────────────┐     当期純利益      ┌──────────────┐
│    P/L      │ ─────────────────→ │    B/S      │
│ （損益計算書）│   利益剰余金に蓄積   │ （貸借対照表）│
└──────────────┘                     └──────┬───────┘
                                            │
                                            │ 資産・負債の
                                            │ 増減を調整
                                            ↓
                                    ┌──────────────┐
                                    │    C/F      │
                                    │（キャッシュ  │
                                    │ フロー計算書）│
                                    └──────┬───────┘
                                            │
                                            │ 現金の期末残高
                                            │ が一致
                                            ↓
                            ┌───────────────────────────┐
                            │  B/Sの「現金及び預金」   │
                            └───────────────────────────┘
```

---

## 3.6 三表連動の理解がもたらすメリット

| メリット | 説明 |
|----------|------|
| **利益と現金のズレの理解** | 「なぜ利益が出ているのに現金がないのか」が解消される |
| **黒字倒産リスクの発見** | P/L上は黒字でも、営業CFがマイナス続きなら資金ショートの危険 |
| **企業の真の健全性把握** | 三表を総合的に見ることで、表面的な数字に騙されない |

# 第4章：業種別「正常値」の読み解き方

## 4.1 業種別財務指標の一覧

日本の上場企業約3,700社のデータに基づく業種別の財務指標の目安です。

| 業種 | 自己資本比率 | ROE | ROA | 営業利益率 |
|------|:------------:|:---:|:---:|:---------:|
| **製造業** | 45-55% | 6-8% | 4-6% | 4-5% |
| **小売業** | 30-40% | 7-9% | 2-4% | 2-3% |
| **銀行・金融** | 4-6% | 6-9% | 0.3-0.5% | ―※ |
| **不動産** | 30-35% | 8-11% | 2-3% | 10-15% |
| **IT・通信** | 30-40% | 10-14% | 3-5% | 14-18% |
| **商社** | 30-40% | 12-17% | 4-6% | 3-5%※※ |
| **サービス業** | 45-55% | 8-10% | 5-8% | 8-12% |

※銀行は経常利益基準で評価  
※※商社は持分法投資利益が大きく、営業利益率だけでは収益力を測れない

## 4.2 製造業：装置産業ゆえの特性

### なぜこの水準なのか

| 指標 | 水準 | 理由 |
|------|------|------|
| 自己資本比率 45-55% | 高め | 工場・機械への設備投資が必要な装置産業でありながら、景気変動リスクに備える必要がある |
| 総資産回転率 0.5〜1回 | 低め | 工場設備という巨額の固定資産を抱えている |

### 代表企業の実例

| 企業名 | 自己資本比率 | ROE | 営業利益率 | 特徴 |
|--------|:-----------:|:---:|:---------:|------|
| トヨタ自動車 | 38% | 15.8% | 9.1% | 金融事業含むため自己資本比率やや低め |
| ソニー | 35-40% | 12-15% | 10-12% | エンタメ・金融の複合企業 |
| キーエンス | **96.7%** | 14-16% | **55%超** | 無借金経営、高付加価値製品に特化 |

## 4.3 小売業：薄利多売モデルの宿命

### なぜこの水準なのか

| 指標 | 水準 | 理由 |
|------|------|------|
| 営業利益率 2-3% | 低め | 仕入れて販売するビジネスモデルの宿命。粗利益率25-35%でも人件費・店舗賃料が重い |
| 総資産回転率 | 高め | 在庫を早く回転させることで利益を確保 |

### 代表企業の実例

| 企業名 | 自己資本比率 | ROE | 営業利益率 | 特徴 |
|--------|:-----------:|:---:|:---------:|------|
| ファーストリテイリング | 56% | **19%** | 13% | SPA（製造小売）で異例の高収益 |
| セブン＆アイ | 40% | 6-8% | 5-6% | コンビニ中心の安定経営 |
| イオン | **13%** | 2-3% | 2-3% | 金融事業でレバレッジ高め |

## 4.4 銀行・金融：レバレッジ経営の典型

### なぜこの水準なのか

| 指標 | 水準 | 理由 |
|------|------|------|
| 自己資本比率 4-6% | 極めて低い | 預金（負債）を集めて貸出（資産）で利ざやを稼ぐビジネスモデル |
| ROA 0.3-0.5% | 低い | 総資産が巨大なため。財務レバレッジ効果でROEを6-9%に |

### 健全性の評価基準

銀行の健全性は会計上の自己資本比率ではなく、**BIS規制のTier1資本比率**で評価

| 基準 | 要求水準 |
|------|----------|
| 国際基準行 | 普通株式Tier1比率 7%以上 |
| G-SIBs | さらに上乗せ |

### 代表企業の実例

| 企業名 | 自己資本比率 | ROE | Tier1資本比率 |
|--------|:-----------:|:---:|:------------:|
| 三菱UFJ | 5% | 9% | 14.2% |
| 三井住友FG | 5% | 8-9% | 12.9% |
| みずほFG | 4-5% | 6-7% | 11-12% |

## 4.5 IT・通信：ストック型ビジネスの強み

### なぜこの水準なのか

| 指標 | 水準 | 理由 |
|------|------|------|
| 営業利益率 14-18% | 高め | 月額課金のストック型ビジネスで安定収益、参入障壁が高い寡占市場 |
| 自己資本比率 30-40% | 中程度 | 基地局・光ファイバー・データセンターなど設備投資が必要 |

### 代表企業の実例

| 企業名 | 自己資本比率 | ROE | 営業利益率 |
|--------|:-----------:|:---:|:---------:|
| NTT | 約30% | 14% | 14% |
| KDDI | 28% | 14% | 17% |
| ソフトバンク（通信子会社） | 15-18% | **21%** | 14% |

**注意**：ソフトバンクのROE 21%は通信3社で最高だが、自己資本比率が低く財務レバレッジ効果が大きい

## 4.6 商社：投資会社への進化

### なぜこの水準なのか

| 指標 | 水準 | 理由 |
|------|------|------|
| ROE 12-17% | 高め | トレーディングから事業投資会社へ進化、適度なレバレッジ活用 |
| 営業利益率 3-5% | 低く見える | 利益の多くが「持分法投資利益」など営業外で計上されるため |

### 代表企業の実例

| 企業名 | 自己資本比率 | ROE | 特徴 |
|--------|:-----------:|:---:|------|
| 伊藤忠商事 | 35% | **16%** | 「ROE 15%以上を継続」を経営目標 |
| 三菱商事 | 38% | 10-12% | 資源ビジネスの比重大 |
| 三井物産 | 40% | 14% | LNG・鉄鉱石に強み |

## 4.7 サービス業：アセットライト経営の利点

### なぜこの水準なのか

| 指標 | 水準 | 理由 |
|------|------|------|
| 自己資本比率 45-55% | 高め | 人材が主要な経営資源で固定資産が少ない「アセットライト」なビジネスモデル |
| ROA 5-8% | 全業種で最高水準 | 少ない資産で利益を生み出せる |

### 代表企業の実例

| 企業名 | 自己資本比率 | ROE | 営業利益率 |
|--------|:-----------:|:---:|:---------:|
| リクルートHD | 50-55% | 15-20% | 13-15% |
| オリエンタルランド | **70%超** | 10-12% | **20-25%** |

**特徴**：オリエンタルランドはディズニーブランドの独占的運営で営業利益率20%超、自己資本比率70%超

## 4.8 業種特性を踏まえた分析の重要性

### 分析のポイント

| ポイント | 説明 |
|----------|------|
| **同業種間での比較** | 銀行の自己資本比率5%を製造業の基準で「危険」と判断するのは誤り |
| **ROEだけで判断しない** | ROAと併せて見ることで「レバレッジ依存」かどうかを確認 |
| **3-5年の推移で判断** | 一過性の減損や特別利益を排除し、企業の真の実力を把握 |

# まとめ：財務分析の実践的フレームワーク

## 4つのテーマの要点

| テーマ | 学べること |
|--------|-----------|
| **デュポン分解** | ROEを「収益性」「効率性」「レバレッジ」の3要素に分解。同じROE 15%でも「質」が異なる |
| **キャッシュフロー計算書** | 「利益」と「現金」のズレを可視化。営業CFが継続的にプラスであることが健全な企業の第一条件 |
| **財務三表の連動関係** | 「利益が出ているのに現金がない」「赤字なのに現金が増えている」の理由が明確に |
| **業種別の正常値** | 銀行の自己資本比率5%は危機ではなく業態特性。業種特性を踏まえた分析が重要 |
