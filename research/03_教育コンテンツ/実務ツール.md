# 実務ツール

財務分析を始めたい人が「これさえあれば今日から実践できる」ことを目指した実務ガイドです。**無料で使えるデータベース**から**Excel関数**、**分析レポートの書き方**まで、必要な知識とツールを網羅的に整理しました。まずは関心のあるセクションから読み始めてください。

---

## 1. 無料で使える財務データベース（日本）

日本には上場企業の財務情報を無料で入手できる優れたサービスが複数あります。それぞれ特徴が異なるため、目的に応じて使い分けることが効率的です。

### 主要サービス一覧

| サービス名 | URL | 主な用途 | 料金 |
|-----------|-----|---------|------|
| EDINET | https://disclosure.edinet-fsa.go.jp/ | 有価証券報告書の閲覧 | 完全無料 |
| TDnet | https://www.release.tdnet.info/inbs/I_main_00.html | 決算短信・適時開示の確認 | 完全無料 |
| Yahoo!ファイナンス | https://finance.yahoo.co.jp/ | 株価・ニュース・基本情報 | 完全無料 |
| バフェット・コード | https://www.buffett-code.com/ | 財務分析・企業比較 | 基本無料 |
| 株探（かぶたん） | https://kabutan.jp/ | 決算速報・テーマ株発掘 | 基本無料 |
| みんかぶ | https://minkabu.jp/ | AI株価診断・投資家コミュニティ | 基本無料 |
| 日経会社情報DIGITAL | https://www.nikkei.com/nkd/ | ニュース連携・業界分析 | 有料（日経電子版契約要） |

### 各サービスの詳細

#### EDINET（金融庁）

**概要**：金融庁が運営する法定開示書類の電子閲覧システム。有価証券報告書、半期報告書、大量保有報告書など、企業が法律に基づいて提出する書類を**24時間365日、登録不要**で閲覧できます。

**使い方**：
1. サイトにアクセスし「書類簡易検索」をクリック
2. 企業名または証券コードを入力
3. 「有価証券報告書」にチェックを入れて検索
4. PDFで閲覧・ダウンロード

**メリット**：公式情報源で信頼性が最も高い／**過去10年分**を閲覧可能／API機能でデータ取得も可能

**デメリット**：情報量が膨大で初心者には難解／分析ツールや可視化機能がない／株価情報は含まれない

---

#### TDnet（東京証券取引所）

**概要**：東証が運営する適時開示情報伝達システム。決算短信や重要な経営情報を**開示と同時にリアルタイム**で確認できます。

**使い方**：
1. 「適時開示情報閲覧サービス」にアクセス
2. 会社名、証券コード、開示日等で検索
3. PDFで閲覧・ダウンロード

**メリット**：最速の情報源（決算発表をリアルタイムで確認可能）／完全無料・登録不要

**デメリット**：無料閲覧は**31日分のみ**／分析ツールなし

---

#### Yahoo!ファイナンス

**概要**：日本最大級の投資・金融情報ポータル。個人投資家にとって必要な情報を網羅的に提供し、ほとんどの機能を無料で利用可能です。

**使い方**：
1. 検索窓に企業名または証券コードを入力
2. 「株価」「チャート」「ニュース」「企業情報」タブで切り替え
3. ポートフォリオ機能はYahoo! IDでログインして利用

**メリット**：**完全無料**で初心者に優しいUI／リアルタイム株価／証券会社アプリ連携

**デメリット**：財務分析機能は基本的／長期業績データは他サービスより劣る／広告が多い

---

#### バフェット・コード

**概要**：投資銀行レベルの高機能な企業分析ツールを個人投資家向けに提供。約**3,600社の日本企業**と**6,000社の米国企業**の財務分析が可能です。

**使い方**：
1. 検索窓に企業名または証券コードを入力
2. 企業ページで業績推移をグラフで確認
3. 「条件検索」でスクリーニング、「企業比較」で競合分析

**メリット**：**無料で高機能**（ROE、EBITDA等を自動計算）／過去10年のグラフ表示／CSVダウンロード可能

**デメリット**：5年以上の詳細データは有料（月額990円〜）／初心者には機能が多すぎる場合あり

---

#### 株探（かぶたん）

**概要**：決算速報の速さとテーマ株検索に強みを持つ株式情報サイト。AIによる**決算サプライズ判定**が特徴です。

**使い方**：
1. 銘柄コードまたは企業名で検索
2. 「決算速報」で最新発表をチェック
3. 「銘柄探検」でテーマからスクリーニング

**メリット**：**決算速報が早い**／四半期業績の長期データが無料で確認可能／AIサプライズ判定

**デメリット**：1ヶ月前以上の情報は有料（月額2,178円）／リアルタイム株価は有料

---

#### みんかぶ

**概要**：ユーザー参加型の資産形成情報ポータル。**AI株価診断**と投資家コミュニティ機能が特徴です。

**使い方**：
1. 企業名または証券コードで検索
2. 「みんかぶ目標株価」と「買い」「売り」予想を確認
3. 初心者向けコンテンツで学習

**メリット**：AI株価診断で理論株価を提示／投資家コミュニティで他者の見方を参考にできる

**デメリット**：高度な機能は有料（月額1,320円）／売買予想は個人見解で信頼性は保証なし

## 2. 有価証券報告書の読み方

有価証券報告書（有報）は上場企業が年に1度提出する**最も詳細な公式開示資料**です。監査法人の監査を受けており、情報の信頼性が高いのが特徴。100〜300ページに及ぶ分量ですが、ポイントを絞れば効率的に読めます。

### 入手方法（EDINET操作手順）

1. **EDINET**（https://disclosure.edinet-fsa.go.jp/）にアクセス
2. 「書類簡易検索」をクリック
3. 「提出者/発行者/ファンド」欄に**企業名**または**証券コード**を入力
4. 書類種別で「有価証券報告書」にチェック
5. 検索実行 → PDFで閲覧・ダウンロード

**ポイント**：正式名称で検索（例：「NEC」→「日本電気」）／過去10年分を閲覧可能

### 有価証券報告書の全体構成

```
有価証券報告書
├─ 第一部【企業情報】 ← 企業分析のメイン
│   ├─ 第1【企業の概況】
│   ├─ 第2【事業の状況】 ← 特に重要
│   ├─ 第3【設備の状況】
│   ├─ 第4【提出会社の状況】
│   └─ 第5【経理の状況】 ← 財務諸表
├─ 第二部【提出会社の保証会社等の情報】
└─ 監査報告書
```

### 各セクションの解説

#### 第1【企業の概況】

| 項目 | 内容 | 読むべきポイント |
|------|------|-----------------|
| 主要な経営指標等の推移 | 直近5年間の売上高、利益、自己資本比率など | **最初に見るべき項目**。業績トレンドが一目瞭然 |
| 沿革 | 会社設立からの歴史、M&A履歴 | 企業文化・成長パターンを理解 |
| 事業の内容 | ビジネスモデル、事業セグメント | **必読**。何をやっている会社か把握 |
| 従業員の状況 | 従業員数、平均年齢、**平均年間給与** | 就活・転職でも注目。労働集約型か確認 |

#### 第2【事業の状況】★最重要セクション

| 項目 | 内容 | 読むべきポイント |
|------|------|-----------------|
| 経営方針・対処すべき課題 | 中期経営計画、KPI | 会社の方向性・価値観 |
| サステナビリティ | ESG情報、気候変動対応 | 2023年以降追加の重要項目 |
| **事業等のリスク** | 想定リスクと対応策 | **投資判断で必読** |
| **MD&A** | 経営者視点での業績分析 | 数字の背景理解に必須 |
| 研究開発活動 | R&D投資額、注力分野 | 将来の成長性予測 |

#### 第3【設備の状況】

| 項目 | 内容 |
|------|------|
| 設備投資等の概要 | 当期の設備投資額と内訳 |
| 主要な設備の状況 | 工場・事業所の所在地（製造業で特に重要） |
| **設備の新設・除却等の計画** | 今後の投資計画 → **将来の成長予測**に直結 |

#### 第4【提出会社の状況】

| 項目 | 内容 |
|------|------|
| 株式等の状況 | 発行済株式数、大株主リスト |
| 配当政策 | 配当方針、配当性向の目標 |
| 役員の状況 | 取締役・監査役の経歴、報酬 |
| コーポレート・ガバナンス | 取締役会構成、内部統制 |

#### 第5【経理の状況】

| 項目 | 内容 |
|------|------|
| 連結財務諸表 | BS（貸借対照表）、PL（損益計算書）、CF（キャッシュ・フロー計算書） |
| **財務諸表注記** | 会計方針、セグメント情報、関連当事者取引など → **重要情報の宝庫** |
| 個別財務諸表 | 提出会社単体の財務諸表 |

### 特に重要な箇所の詳細

#### 事業等のリスク

企業が認識しているリスク要因と対応策が記載されています。

**チェックポイント**：
- **冒頭に記載されているリスクほど重要**（経営者の認識順）
- 一般的なリスク（景気変動等）より**企業固有のリスク**に注目
- 「可能性が高い」「重大な影響」などの表現に注意
- **前年からの変化**（新規追加・削除されたリスク）を確認

#### MD&A（経営者による分析）

経営者自身が数字の背景や原因を説明するセクション。単なる数字の羅列ではなく、**「経営者の目を通して」**企業を見ることができます。

**見るべきポイント**：
- 業績の増減要因（なぜ売上が増えた/減ったのか）
- セグメント別分析（どの事業が好調/不調か）
- 経営目標の達成状況
- 将来の見通し（ただし推測を含む点に注意）

#### セグメント情報

複数事業を営む企業が事業別・地域別に業績を開示。**連結財務諸表だけでは見えない事業ごとの収益性**が分かります。

**掲載場所**：第5【経理の状況】の連結財務諸表注記内

**活用例**：楽天グループの場合、全社で赤字でも「楽天市場・金融は黒字、モバイルが大幅赤字」とセグメント情報で判明

#### 財務諸表注記

見落としがちですが**重要情報の宝庫**です。

| 注記項目 | 重要性 |
|---------|-------|
| 重要な会計方針 | 会計処理の前提を理解 |
| 会計上の見積りに関する注記 | 減損、引当金等の根拠。将来リスクの兆候 |
| 関連当事者取引 | 親会社・子会社間取引。利益操作の可能性チェック |
| 偶発債務 | 訴訟リスク、債務保証など。BSに載らないリスク |
| 後発事象 | 決算日後の重要事項。直近の変化を確認 |

### 効率的な読み方のコツ

#### 時間別の優先順位

**10分で読む場合**（最優先3項目）：
1. 主要な経営指標等の推移（5年間の業績トレンド）
2. 事業の内容（何をやっている会社か）
3. セグメント情報（どの事業が儲かっているか）

**30分で読む場合**（上記＋追加4項目）：
4. 事業等のリスク（企業固有リスクに注目）
5. MD&A（業績変動の理由）
6. 従業員の状況（平均年収、従業員数推移）
7. 設備の新設・除却等の計画（将来の投資計画）

#### 推奨される読み順

```
① 主要な経営指標等の推移 → 全体像をつかむ
② 事業の内容 → ビジネスモデル理解
③ セグメント情報 → 収益構造の把握
④ MD&A → 数字の背景理解
⑤ 事業等のリスク → 投資リスクの確認
⑥ 経理の状況（財務諸表） → 詳細な財務分析
```

**NG**：いきなり財務諸表から読み始める → ビジネスモデルを理解していないと数字の意味が分からない

#### PDF内検索で効率化

**Ctrl+F（Mac: Cmd+F）で検索するキーワード**：

| 目的 | 検索キーワード |
|------|---------------|
| リスク把握 | 「リスク」「訴訟」「係争」 |
| 業績変動の理由 | 「増加」「減少」「要因」「主な理由」 |
| 将来計画 | 「計画」「予定」「見込み」「目標」 |
| 問題の兆候 | 「減損」「引当金」「のれん」 |

### 見落としがちなポイント

| 見落としやすい箇所 | なぜ重要か |
|------------------|-----------|
| 注記の「重要な会計上の見積り」 | 減損リスク、引当金の妥当性が分かる |
| セグメント情報の「調整額」 | 本社費用の重さが分かる |
| 「偶発債務」「後発事象」 | BSに載らないリスク、最新動向 |
| 「継続企業の前提」 | 記載があれば倒産リスクの兆候 |
| 「関連当事者取引」注記 | 親子会社間の不正取引の可能性 |

## 3. 決算短信の読み方

決算短信は有価証券報告書の**「速報版」**という位置づけ。決算日から30〜45日以内に公開され、投資家が最も早く決算内容を確認できる資料です。

### 基本情報

| 項目 | 内容 |
|------|------|
| 開示タイミング | 決算日から30〜45日以内（「45日ルール」） |
| 入手先 | TDnet、企業IRページ |
| 分量 | 10〜20ページ程度 |
| 監査 | **監査対象外**（速報のため推測含む場合あり） |

### 構成

```
決算短信
├─ 表紙（サマリー情報）← 1〜2ページ ★最重要
└─ 添付資料 ← 3ページ目以降
    ├─ 経営成績等の概況
    ├─ 会計基準の選択
    └─ 連結財務諸表及び注記
```

### サマリーページの見方（最重要）

表紙1〜2ページに要点が凝縮されています。**ここだけ見れば企業の業績概要が把握可能**です。

#### 経営成績の主要項目

| 項目 | 意味 | チェックポイント |
|------|------|-----------------|
| 売上高 | 本業での収入総額 | 前年同期比で増加しているか |
| 営業利益 | 本業で稼いだ利益 | 売上高営業利益率も確認 |
| 経常利益 | 営業利益＋営業外損益 | 金融収支なども反映 |
| 親会社株主に帰属する当期純利益 | 最終的な利益 | 配当の原資になる |
| 1株当たり当期純利益（EPS） | 純利益÷発行済株式数 | 株価評価の基準 |

#### 財政状態

| 項目 | 意味 | 目安 |
|------|------|------|
| 総資産 | 会社が持つ資産の総額 | 規模感の把握 |
| 純資産 | 資産から負債を引いた額 | 実質的な会社の価値 |
| 自己資本比率 | 純資産÷総資産×100 | **40%以上**が安心 |

#### キャッシュ・フロー

| 種類 | 内容 | 健全な状態 |
|------|------|-----------|
| 営業CF | 本業で稼いだ現金 | **プラス**が基本 |
| 投資CF | 設備投資や事業投資 | マイナスが通常（成長投資） |
| 財務CF | 借入・返済・配当など | 状況による |

**理想的なパターン**：営業CF＝プラス、投資CF＝マイナス、財務CF＝マイナス

### 業績予想の読み方

#### 進捗率の計算

```
進捗率(%) = 四半期累計実績 ÷ 通期予想 × 100
```

**進捗率の目安**（3月決算企業）：

| 決算 | 経過月数 | 単純計算での目安 |
|------|---------|-----------------|
| 第1四半期（6月） | 3ヶ月 | 25% |
| 第2四半期（9月） | 6ヶ月 | 50% |
| 第3四半期（12月） | 9ヶ月 | 75% |

**注意**：業種によって稼ぐ時期が異なるため、**前年同期の進捗率と比較**することが重要

#### 業績予想の信頼性判断

- **過去の予想精度を確認**：保守的な予想を出す傾向か、強気か
- **会社四季報の記者予想との比較**：乖離がある場合は要注意
- **セグメント別動向**：添付資料で予想の妥当性を検証

### 有価証券報告書との違い

| 項目 | 決算短信 | 有価証券報告書 |
|------|---------|---------------|
| 根拠法令 | 証券取引所規程 | 金融商品取引法 |
| 開示タイミング | 30〜45日以内 | **3ヶ月以内** |
| 速報性 | **高い** | 低い |
| 情報量 | 少ない（10〜20ページ） | 多い（100〜300ページ） |
| 監査 | **不要** | **必須**（監査済み） |
| 正確性 | 推測含む場合あり | 監査済みで信頼性高い |

**使い分け**：
- **決算短信**：速報確認、業績予想の把握
- **有価証券報告書**：詳細な財務分析、リスク情報の確認

### 10分でできる決算短信チェック

1. **1分目**：経営成績で売上高・営業利益の前年同期比を確認
2. **2分目**：業績予想を確認し、成長見通しを把握
3. **3分目**：配当の状況を確認
4. **4〜5分目**：添付資料の「経営成績の概況」で業績変動の理由を把握
5. **6〜8分目**：「今後の見通し」で来期の展望を確認
6. **9〜10分目**：セグメント情報で事業別の好不調を確認

### 便利なExcel関数（コピペ可能）

#### CAGR（年平均成長率）の計算

```excel
【方法1: べき乗演算子】
=(最終年セル/初年セル)^(1/年数)-1
例: =(G2/B2)^(1/5)-1

【方法2: POWER関数】
=POWER(最終年セル/初年セル,1/年数)-1
例: =POWER(G2/B2,1/5)-1

【方法3: RRI関数（Excel 2013以降）】
=RRI(年数,初年セル,最終年セル)
例: =RRI(5,B2,G2)
```

**実例**：2020年売上1,000万円 → 2025年売上1,500万円（5年）
`=POWER(1500/1000,1/5)-1` → **結果: 8.45%**

#### 構成比計算

```excel
【対売上高比率】
=項目セル/$売上高セル
例: =B5/$B$2  ※$記号で絶対参照に

【実例】
売上高       1,000  | =B2/$B$2 → 100.0%
売上原価       650  | =B3/$B$2 →  65.0%
売上総利益     350  | =B4/$B$2 →  35.0%
営業利益       150  | =B6/$B$2 →  15.0%
```

#### 前年比（YoY）計算

```excel
【前年比（何倍か）】
=今年のセル/前年のセル
例: =C3/B3 → 結果: 1.20 (または120%)

【増減率（何%増減したか）】
=(今年のセル-前年のセル)/前年のセル
例: =(C3-B3)/B3 → 結果: 0.20 (20%増)

または
=今年のセル/前年のセル-1
例: =C3/B3-1 → 結果: 0.20

【エラー回避付き】
=IFERROR(C3/B3-1,"-")
```

#### その他便利な関数

```excel
【条件分岐（IF関数）】
=IF(営業利益率>=10%,"良好",IF(営業利益率>=5%,"普通","要改善"))
例: =IF(D5>=0.1,"良好",IF(D5>=0.05,"普通","要改善"))

【増減判定】
=IF(C3>B3,"↑増加",IF(C3<B3,"↓減少","→横ばい"))

【四捨五入】
=ROUND(数値,小数点以下桁数)
例: =ROUND(B3/A3*100,1)  ※小数点1桁まで
```

## 5. 分析レポートのテンプレート

### 標準的な構成例

| セクション | 内容 |
|-----------|------|
| 1. エグゼクティブサマリー | 分析結果の要約、投資判断の結論を1ページに |
| 2. 企業概要 | 沿革、事業内容、経営陣、主要製品 |
| 3. 業界分析 | 市場規模、競合環境、ポジショニング |
| 4. 財務分析 | 収益性・安全性・効率性・成長性の4分類 |
| 5. バリュエーション | DCF法、PER/PBR比較、理論株価 |
| 6. リスク分析 | 事業・財務・外部環境リスク |
| 7. 結論・投資判断 | 総合評価、推奨アクション |

### 各セクションで使う主要指標

#### 収益性分析

| 指標 | 計算式 | 目安 |
|------|-------|------|
| 売上高営業利益率 | 営業利益÷売上高 | **5%超で優良** |
| ROE | 当期純利益÷自己資本 | **8%以上**が目安 |
| ROA | 当期純利益÷総資産 | 資産活用効率 |
| ROIC | 税引後営業利益÷投下資本 | 近年注目の指標 |

#### 安全性分析

| 指標 | 計算式 | 目安 |
|------|-------|------|
| 流動比率 | 流動資産÷流動負債 | 200%以上が理想 |
| 当座比率 | 当座資産÷流動負債 | 100%以上で問題なし |
| 自己資本比率 | 自己資本÷総資産 | **40%以上で優良** |

#### 効率性分析

| 指標 | 内容 |
|------|------|
| 総資産回転率 | 売上高÷総資産 |
| 売上債権回転期間 | 売上債権÷（売上高÷365） |
| 棚卸資産回転期間 | 棚卸資産÷（売上原価÷365） |

### 良いレポートの特徴

**ベストプラクティス**：
- **比較を含める**：前年比較、同業他社比較、予算比較を必ず実施
- **図表を活用**：BS・PLを1枚の図にまとめて視覚化
- **根拠を明示**：分析に使用した定義・算出方法を記載
- **コメントを付加**：数値の変動理由を具体的に解説
- **デュポン分解**：ROEを3要素（収益性×効率性×財務レバレッジ）に分解

**避けるべき失敗**：
- 数字の羅列だけで分析コメントがない
- 単年度のみの分析（最低3〜5期の推移を見る）
- 業界特性を無視した画一的な分析
- 結論があいまいで具体的なアクションが示されていない
