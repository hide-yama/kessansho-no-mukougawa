# バリュエーション

## 株式バリュエーション完全ガイド

株価の割高・割安を判断する力は、投資で成果を出すための最も基本的なスキルである。**PER15倍は割安か？PBR0.5倍の銀行株は買いか？** 本章では、バリュエーション（企業価値評価）の代表的な手法を、日本企業の具体例とともに解説する。「数字の意味」を理解すれば、投資判断の精度は格段に上がる。

---

# 第1章：PER（株価収益率）——利益の何年分で株を買うか

## 1.1 定義と計算式

PER（Price Earnings Ratio：株価収益率）は、株価が1株あたり利益の何倍まで買われているかを示す指標である。最も広く使われるバリュエーション指標の一つだ。

### 計算式

```
PER（倍）＝ 株価 ÷ EPS（1株あたり純利益）

または

PER（倍）＝ 時価総額 ÷ 純利益
```

### 具体例：キーエンス（6861）

| 項目 | 数値 |
|------|------|
| 株価 | 60,000円 |
| EPS（1株あたり純利益） | 約1,600円 |
| **PER** | **60,000 ÷ 1,600 ≒ 37.5倍** |

PER37.5倍とは、「現在の利益水準が続くなら、投資回収に**37.5年かかる**」という意味だ。言い換えれば、投資家は「将来の利益成長」を織り込んで、今の利益の37年分を前払いしている。

---

## 1.2 「利益の何年分」という意味

PERの本質的な意味を理解するために、具体例で考えてみよう。

### 例：PER15倍の企業

- 株価：1,500円
- EPS：100円
- PER：15倍

この場合、「投資回収に15年かかる」と解釈できる。ただし、これは**利益が一定**という前提での話だ。

### 成長企業の場合

利益が毎年10%成長するなら、投資回収期間は大幅に短縮される。これが「高PERでも割安になりうる」理由だ。

| 年数 | EPS（10%成長） | 累積EPS |
|------|---------------|---------|
| 1年目 | 100円 | 100円 |
| 5年目 | 161円 | 610円 |
| 10年目 | 259円 | 1,594円 |
| 15年目 | 418円 | 3,177円 |

→ 10%成長が続けば、約10年で投資回収可能（PER15倍でも）

---

## 1.3 PERの目安——日本市場の現在水準

### 日経平均・TOPIXのPER推移

| 指標 | PER水準 | 備考 |
|------|---------|------|
| **日経平均（2025年1月）** | **約17〜18倍** | 加重平均ベース |
| **TOPIX（2025年1月）** | **約15〜17倍** | 12ヶ月先予想ベース |
| 過去平均レンジ | 14〜16倍 | アベノミクス以降の適正水準 |
| 割安の目安 | 14倍以下 | 押し目買いが入りやすい |
| 割高の目安 | 18倍以上 | 利益確定売りが出やすい |
| バブル期（1989年） | 60〜70倍 | 異常な過大評価（参考） |

### 過去のPER水準との比較

| 時期 | 日経平均PER | 背景 |
|------|------------|------|
| 1989年（バブル絶頂） | 60〜70倍 | 資産インフレ、投機過熱 |
| 2003年（ITバブル崩壊後） | 100倍超 | 企業業績悪化でEPS低下 |
| 2008年（リーマンショック） | 赤字企業続出 | 算出不能な時期も |
| 2020年（コロナショック） | 20〜25倍 | 業績悪化先取り |
| 2024〜2025年 | 15〜18倍 | 適正水準に回帰 |

---

## 1.4 業種別の平均PER——比較は「同業種内」で

PERの「高い・低い」は業種によって大きく異なる。IT企業のPER30倍と銀行のPER10倍を単純比較しても意味がない。**必ず同業種内で比較すること**が鉄則だ。

### 業種別平均PER一覧（2024〜2025年）

| 業種 | 平均PER | 特徴・理由 |
|------|---------|-----------|
| **全業種平均** | 約15〜16倍 | 中央値は約14.9倍 |
| **情報・通信（IT）** | **25〜30倍** | 成長期待で高水準、スケーラビリティ |
| **小売業** | 20〜25倍 | 消費回復期待、優待人気銘柄で高め |
| **製造業（電機・機械）** | 15〜20倍 | 半導体関連は高め、重工業は低め |
| **不動産業** | 10〜15倍 | 資産株として安定的、金利感応度高 |
| **金融（銀行業）** | **12〜13倍** | 2024年に8.7倍→12.8倍へ上昇（金利上昇期待） |
| **電力・ガス** | 10〜15倍 | 規制産業、成長期待薄 |

### なぜ業種でPERが異なるのか

| 要因 | 高PER業種 | 低PER業種 |
|------|-----------|-----------|
| 成長期待 | IT、バイオ | 銀行、電力 |
| 利益の安定性 | 食品、医薬品 | 素材、鉄鋼 |
| 景気感応度 | 低い業種が高PER | 高い業種が低PER |
| 資本集約度 | 低い（IT等） | 高い（製造業等） |

---

## 1.5 高PERの意味——市場は「成長」に賭けている

PERが高い企業には、市場が「将来の利益成長」を期待している。

### 高PER企業の代表例

| 企業名 | PER | 高PERの理由 |
|--------|-----|-------------|
| キーエンス | 約37倍 | 営業利益率50%超、FA市場成長 |
| ファーストリテイリング | 約38倍 | 海外ユニクロ高成長、グローバル展開 |
| 東京エレクトロン | 約25倍 | 半導体製造装置の世界シェア |
| ソニーグループ | 約25倍 | ゲーム・音楽・映画の複合成長 |

### 高PERが許容される条件

| 条件 | 説明 |
|------|------|
| 持続的な利益成長 | 年率10%以上の成長が見込める |
| 高い参入障壁 | 競合が参入しにくい独自の強み |
| 高ROE | 資本効率が高く、複利で成長できる |
| グローバル展開 | 国内市場に依存しない成長余地 |
| 再投資機会 | 成長投資の余地が十分にある |

**警告：** 成長期待が剥落すればPERは急落する。「高PER＝割高」ではなく、「成長が続くかどうか」が判断の核心だ。

---

## 1.6 低PERの意味と罠——「安い」と「割安」は違う

PERが低い企業は一見「お買い得」に見える。しかし、**低PERには必ず理由がある**。

### 低PERの主な理由

| 理由 | 説明 | 例 |
|------|------|-----|
| **業績悪化リスク** | 将来の減益を市場が織り込んでいる | 構造不況業種 |
| **成熟産業** | 成長余地がなく、利益成長が見込めない | 地方銀行 |
| **一時的な特別利益** | 特別利益でEPSが一時的に膨らんでいる | 資産売却益 |
| **市場の無関心** | 時価総額が小さく機関投資家が参入できない | 小型株 |
| **景気循環のピーク** | 景気敏感株で利益がピークにある | 素材、鉄鋼 |

### バリュートラップの警告

**例：** PER8倍の株が「割安」だと飛びついたら、翌年には利益が半減してPER16倍になった——これは珍しくない。

```
【バリュートラップの典型パターン】

今期：純利益100億円、時価総額800億円 → PER8倍（割安に見える）
　↓
来期：純利益50億円に減少、時価総額800億円 → PER16倍（実は割高だった）
```

**教訓：** 現在のPERではなく、**将来のPERで考える**習慣をつけよう。

---

## 1.7 PERだけで判断してはいけない理由

PERには本質的な限界がある。

### PERの限界

| 限界 | 説明 |
|------|------|
| **赤字企業には使えない** | 純利益がマイナスだと計算不能 |
| **一時的要因に左右される** | 特別利益・損失で大きく歪む |
| **会計処理の影響** | 減価償却方法、引当金計上で変動 |
| **成長性を反映しにくい** | 将来の利益成長を見ていない |
| **業種間比較に不向き** | 業種ごとに適正水準が異なる |

### 実務的な使い方

| 使い方 | 説明 |
|--------|------|
| **同業他社との比較** | クロスセクション分析（横比較） |
| **過去の自社との比較** | 時系列分析（縦比較） |
| **PBR、ROEとの組み合わせ** | 複数指標で総合判断 |
| **予想PERの活用** | 来期予想利益ベースで計算 |

# 第2章：PBR（株価純資産倍率）——解散したらいくら戻るか

## 2.1 定義と計算式

PBR（Price Book-value Ratio：株価純資産倍率）は、株価が1株あたり純資産の何倍まで買われているかを示す指標である。

### 計算式

```
PBR（倍）＝ 株価 ÷ BPS（1株あたり純資産）

または

PBR（倍）＝ 時価総額 ÷ 純資産（株主資本）
```

### 具体例：三菱UFJフィナンシャル・グループ（8306）

| 項目 | 数値 |
|------|------|
| 株価 | 約1,900円 |
| BPS（1株あたり純資産） | 約1,400円 |
| **PBR** | **1,900 ÷ 1,400 ≒ 1.36倍** |

---

## 2.2 「解散価値との比較」という意味

### PBR1倍の意味

PBR1倍とは、株価と解散価値が等しい状態を意味する。

| PBR水準 | 意味 | 市場の評価 |
|---------|------|-----------|
| **PBR1倍** | 株価＝解散価値 | 事業継続と解散が同価値 |
| **PBR2倍** | 株価＝解散価値の2倍 | 事業継続で価値が2倍になる |
| **PBR0.5倍** | 株価＝解散価値の半分 | 「この会社に将来性はない」 |

### 「解散価値」の考え方

会社を今すぐ解散して、全資産を売却し、負債を返済し、残った純資産を株主に分配する——その金額が「解散価値」だ。

```
【解散価値の計算イメージ】

総資産 1,000億円
－ 負債  600億円
＝ 純資産 400億円 ← これが解散価値

発行済株式数 4億株 の場合
BPS = 400億円 ÷ 4億株 = 100円/株

株価が100円なら PBR1倍
株価が50円なら PBR0.5倍（解散した方が得）
```

### 要請の背景

### 業種別平均PBR一覧（2024〜2025年）

| 業種 | 平均PBR | 特徴・理由 |
|------|---------|-----------|
| **情報・通信（IT）** | **2.3〜2.5倍** | 無形資産主体、高成長期待 |

| **不動産業** | 0.8〜1.2倍 | 含み資産で変動 |

## 2.5 低PBRが「許容される」業種とその理由

すべての低PBRが問題というわけではない。**構造的に低PBRになりやすい業種**がある。

### 銀行・金融セクター

| 理由 | 説明 |
|------|------|
| **自己資本規制** | バーゼルⅢ規制で自社株買いによる資本圧縮が困難 |
| **清算時の価値毀損** | 銀行が清算すると取引先も悪化、貸出債権の価値が大幅毀損 |
| **市場の認識** | 「今の純資産は清算時には維持できない」 |

### 資産含み益のある企業

| 理由 | 説明 |
|------|------|
| **含み益の非顕在化** | 保有株式・不動産に含み益があっても売却しなければ利益にならない |
| **「眠っている資産」** | 投資家は実現可能性を割り引いて評価 |

### 景気敏感株・重厚長大産業

| 理由 | 説明 |
|------|------|
| **減益リスク** | 景気後退時の大幅減益を織り込んだ株価形成 |
| **設備投資負担** | 巨額の設備投資が必要でROEが低くなりがち |

---

## 2.6 PBRの具体的な使い方

### PBRの活用シーン

| シーン | 使い方 |
|--------|--------|
| **資産株の評価** | 不動産、含み益のある企業の下値目安 |
| **金融機関の評価** | 銀行・保険会社の主要指標 |
| **割安株スクリーニング** | PBR1倍割れ銘柄の抽出 |
| **M&A価格の検討** | 買収価格の下限目安 |

### PBRとROEの関係

PBRとROEには密接な関係がある。理論的には以下の式が成り立つ。

```
PBR ≒ ROE ÷ 株主資本コスト

例：
ROE 10%、株主資本コスト 8% の場合
理論PBR ≒ 10% ÷ 8% = 1.25倍
```

**ROEが株主資本コストを上回れば、PBRは1倍を超える**——これがPBR改善の本質だ。

# 第3章：成長企業のバリュエーション——赤字でも評価できる指標

## 3.1 なぜ赤字企業にPERが使えないか

PERは「純利益」を分母に使う。赤字企業は純利益がマイナスなので、PERは計算できない（マイナスのPERは意味をなさない）。

### 赤字企業の例

| 企業 | 純利益 | PER | 評価方法 |
|------|--------|-----|----------|
| 黒字企業A | +100億円 | 15倍 | PERで評価可能 |
| 赤字企業B | △50億円 | **計算不能** | PSR、EV/EBITDAを使用 |

スタートアップやIT企業の多くは、成長投資を優先して意図的に赤字を続ける。こうした企業を評価するには、**利益以外の指標**が必要だ。

---

## 3.2 PSR（株価売上高倍率）——売上があれば計算できる

### 定義と計算式

```
PSR（倍）＝ 時価総額 ÷ 年間売上高

または

PSR（倍）＝ 株価 ÷ 1株あたり売上高
```

### 具体例：マネーフォワード（2017年上場時）

| 項目 | 数値 |
|------|------|
| 想定時価総額 | 243億円 |
| 今期予想売上高 | 27億円 |
| **PSR** | **約9倍**（赤字のままIPO） |

### なぜ赤字企業に使えるか

売上高は利益と違い、会計処理や一時的損失の影響を受けにくい。赤字でも計算可能なのが最大の利点だ。

### PSRの目安

| PSR倍率 | 判断目安 | 該当企業イメージ |
|---------|----------|-----------------|
| **0.5倍以下** | 割安 | 製造業大型株など |
| **0.7倍** | 日経225企業の中央値 | 一般的な大企業 |
| **1〜5倍** | 一般的な水準 | 成長企業〜成熟企業 |
| **10〜20倍** | 高成長期待 | SaaS企業、テック株 |
| **20倍以上** | 割高ゾーン | 売り検討の目安 |

### PSRの注意点

| 注意点 | 説明 |
|--------|------|
| **業種差が大きい** | IT企業は利益率が高くPSRが高くなりやすい |
| **利益率を無視** | 売上はあっても赤字が続く企業もある |
| **同業種内で比較** | 必ず同業他社と比較する |

---

## 3.3 EV/EBITDA——M&Aの世界標準

### EVとは（Enterprise Value：企業価値）

企業を買収する際の実質的コスト。株式時価総額に負債を加え、現金を差し引く。

```
EV ＝ 時価総額 ＋ 有利子負債 － 現預金
```

### EBITDAとは

EBITDA（Earnings Before Interest, Taxes, Depreciation and Amortization）は、利息・税金・減価償却費控除前利益。本業の「稼ぐ力」を示す。

```
EBITDA ＝ 営業利益 ＋ 減価償却費
```

### EV/EBITDA倍率の計算

```
EV/EBITDA倍率 ＝ EV ÷ EBITDA
```

この倍率は「買収コストを何年で回収できるか」を意味する。

### 具体例

| 項目 | 金額 |
|------|------|
| 時価総額 | 5億円 |
| 有利子負債 | 2,000万円 |
| 現預金 | 5,000万円 |
| **EV** | **4億7,000万円** |
| 営業利益 | 5,000万円 |
| 減価償却費 | 1,000万円 |
| **EBITDA** | **6,000万円** |
| **EV/EBITDA** | **7.8倍**（約7.8年で投資回収） |

### 業種別のEV/EBITDA目安

| 業種 | EV/EBITDA倍率 | 特徴 |
|------|---------------|------|
| 情報・通信（IT） | 8〜17倍 | 高めに評価 |
| サービス業 | 8〜15倍 | 人材依存 |
| **全業種平均** | **6〜8倍** | 標準水準 |
| 製造業（鉄鋼等） | 5〜6倍 | 設備集約 |
| 中小企業M&A | 3〜5倍 | 小規模プレミアム |

### PERとの違い

| 項目 | PER | EV/EBITDA |
|------|-----|-----------|
| 分子 | 時価総額のみ | 時価総額＋負債－現金 |
| 分母 | 純利益 | EBITDA（営業利益＋減価償却） |
| 税金の影響 | 受ける | 受けない |
| 減価償却の影響 | 受ける | 受けない |
| 国際比較 | 難しい | 容易 |
| 主な用途 | 株式投資 | M&A、国際比較 |

---

## 3.4 DCF法（ディスカウント・キャッシュフロー法）——理論的に最も正確

### 基本的な考え方

企業が将来生み出すキャッシュフローを、現在の価値に割り引いて合計する方法。「今日の100万円」と「5年後の100万円」は価値が違うという考えに基づく。

### 「現在価値」の考え方

```
【例】割引率5%の場合

1年後の100万円の現在価値 = 100万円 ÷ 1.05 = 95.2万円
5年後の100万円の現在価値 = 100万円 ÷ 1.05^5 = 78.4万円
10年後の100万円の現在価値 = 100万円 ÷ 1.05^10 = 61.4万円
```

### 計算の流れ（簡略版）

```
【Step1】将来のフリーキャッシュフロー（FCF）を予測（3〜5年分）

FCF = 営業利益 ×（1－税率）＋ 減価償却費 － 設備投資 ± 運転資本増減

【Step2】割引率（WACC）を設定

一般的に5〜7%程度（企業のリスクに応じて調整）

【Step3】各年のFCFを現在価値に割り引く

現在価値 = FCF ÷（1 + 割引率）^年数

【Step4】ターミナルバリュー（予測期間以降の永続価値）を計算

ターミナルバリュー = 最終年のFCF ×（1 + 永久成長率）÷（割引率 － 永久成長率）

【Step5】すべてを合計 → 企業価値
```

### DCF法の限界と注意点

| 限界 | 説明 |
|------|------|
| **前提条件への依存** | 成長率・割引率を少し変えると結果が大きく変動 |
| **予測の困難さ** | 5年後の業績は誰にもわからない |
| **専門性の必要性** | 素人が行うと誤った結果になりやすい |
| **主観の介入** | 楽観的な前提を置けば高い価値が出る |

**結論：** DCFは「唯一の正解」ではなく「参考値」として使う。複数のシナリオで計算し、レンジで捉えることが重要。

---

## 3.5 各手法の使い分け

| 企業の状態 | 推奨手法 | 理由 |
|------------|----------|------|
| **赤字・スタートアップ** | PSR | 売上があれば計算可能 |
| **赤字だがEBITDA黒字** | EV/EBITDA | キャッシュベースで評価 |
| **黒字安定企業** | PER、EV/EBITDA | 一般的な指標が使える |
| **M&A対象企業** | EV/EBITDA、DCF | 買収コスト回収を評価 |
| **高成長・将来性重視** | DCF | 将来CFを直接評価 |

# 第4章：バリュートラップ——「割安」に見えて株価が上がらない罠

## 4.1 バリュートラップとは何か

バリュートラップとは、PERやPBRで「割安」と判断される銘柄に投資したにもかかわらず、**いつまでも割安なまま放置される**、または株価がさらに下落する状態を指す。「割安の罠」とも呼ばれる。

### なぜ起こるのか

| 原因 | 説明 |
|------|------|
| **将来の業績悪化を織り込み** | 市場は「現在」ではなく「将来」を見ている |
| **流動性の低さ** | 出来高が少なく、機関投資家が参入できない |
| **成長期待の欠如** | 事業の成熟化、構造的問題 |
| **株価を意識しない経営** | 株主還元に消極的 |
| **カタリストの不在** | 株価上昇のきっかけがない |

---

## 4.2 典型的なパターンと特徴

### バリュートラップのパターン

| パターン | 特徴 | 具体例 |
|----------|------|--------|
| **業績下降トレンド** | 現在PER8倍でも将来減益で実質割高 | ゲーム会社（ヒット作後） |
| **過剰な内部留保** | 現金を溜め込み成長投資・株主還元せず | 多くの日本企業 |
| **構造的衰退業界** | 業界全体に成長余地なし | 地方銀行、印刷業 |
| **政策保有株の存在** | 資本効率を下げる持ち合い株式 | 伝統的大企業 |
| **ガバナンス不全** | 株主価値を意識しない経営 | オーナー企業 |

---

## 4.3 日本企業の具体的な事例

### 事例1：地方銀行——上場73行中72行がPBR1倍割れ

地方銀行は日本株バリュートラップの典型例だ。2024年時点で**7割以上がPBR0.5倍以下**という惨状にある。

| 銀行名 | PBR | ROE | 特徴 |
|--------|-----|-----|------|
| 栃木銀行 | 0.11倍 | 約3% | 超割安だが株価低迷 |
| 筑波銀行 | 0.12倍 | 約2% | 構造的低PBR |
| 愛知銀行 | 0.17倍 | 約3% | 競合激化 |
| 豊和銀行 | 0.08倍 | 約1% | 極端な低評価 |

**バリュートラップの理由：**

| 理由 | 説明 |
|------|------|
| 人口減少・地方経済衰退 | 貸出需要が構造的に減少 |
| 超低金利環境 | 利ザヤ縮小（2024年以降は改善傾向） |
| ROE平均4%未満 | 資本コスト8%を大幅に下回る |
| デジタル化対応の遅れ | フィンテックとの競争激化 |

### 事例2：MIXI（2019年）——PER8倍の罠

| 時期 | 状況 |
|------|------|
| 2013-2016年 | 「モンスターストライク」大ヒットで株価急騰 |
| 2019年3月 | 株価ピークから半値以下、PER8倍に低下 |
| その後 | 「割安」に見えたが、**さらに株価は半値に** |

**教訓：** 低PERは「将来の減益」を市場が織り込んでいた。現在の数値ではなく、**将来の業績で割安かどうかを判断すべき**だった。

### 事例3：三菱鉛筆——実質「タダで買える」のに株価低迷

2022年時点のバリュエーション：

| 項目 | 金額 |
|------|------|
| 時価総額 | 約780億円 |
| 現金＋有価証券 | 622億円 |
| **事業価値（EV）** | **約158億円** |
| 経常利益 | 約66億円 |
| EV/経常利益 | **約2.4倍** |

理論上「ほぼ無料で買収可能」な状態だったが、過剰な内部留保、成長投資・株主還元への消極性、株価を意識しない経営により、割安なまま放置された。

## 4.4 バリュートラップの見分け方——確認すべきポイント

### チェックリスト

**【財務指標】**

| チェック項目 | 危険サイン |
|--------------|-----------|

| 業績トレンド | 減収減益が続いている |

**【将来性】**

| チェック項目 | 危険サイン |

**【経営の質】**

| チェック項目 | 危険サイン |
|--------------|-----------|

| 東証要請への対応 | 開示していない |

**【カタリストの有無】**

| チェック項目 | ポジティブサイン |
|--------------|-----------------|

| アクティビスト介入 | 可能性がある |

## 5.3 金融セクター——PBR重視の理由

金融機関は「他人のお金（預金）を運用して稼ぐ」ビジネスモデルのため、資産規模と自己資本の関係が重要になる。

| **PBR** | ◎ | 純資産との比較が本質 |
| **ROE** | ◎ | 資本効率の指標 |
| 自己資本比率 | ○ | 健全性の指標 |
| 不良債権比率 | ○ | 資産の質 |

## 5.4 景気敏感セクター——景気循環とPERの逆転現象

### 重要な注意点

景気敏感株（素材、化学、鉄鋼など）は、**PERが低いときに買ってはいけない**。

### 景気循環とPERの関係

| 景気局面 | 利益水準 | PER | 投資判断 |
|----------|---------|-----|----------|
| **景気ピーク** | 最大 | **最低** | 売り時 |
| 景気後退 | 減少 | 上昇 | 様子見 |
| **景気底** | 最小 | **最高** | 買い時 |
| 景気回復 | 増加 | 低下 | 保有継続 |

一般的な「低PER＝割安」という発想は通用しない。景気循環の位置を見極める必要がある。

### 代表銘柄

| 企業 | 業種 | 特徴 |
|------|------|------|
| 日本製鉄 | 鉄鋼 | 景気・鋼材価格連動 |
| 住友化学 | 化学 | 石化市況連動 |
| コマツ | 建機 | 資源価格・建設投資連動 |

# 結論——バリュエーションの本質

株価の「高い・安い」を見極める力は、一朝一夕には身につかない。しかし、本章で解説した指標の「意味」を理解すれば、判断の精度は格段に上がる。

## 最も重要な3つの原則

### 原則1：「安い」と「割安」は違う

低PER・低PBRには必ず理由がある。構造的問題を抱えた企業は、いくら指標が割安でも株価は上がらない。

### 原則2：同業種内で比較する

IT企業のPER30倍と銀行のPER10倍を単純比較しても意味がない。業種特性を理解し、同業他社との相対比較で判断する。

### 原則3：複数指標で総合判断する

PERだけ、PBRだけで判断するのは危険。ROE、成長性、カタリストの有無まで含めて総合的に評価する。
