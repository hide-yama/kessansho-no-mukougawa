# 付録B：決算書分析チェックリスト

本書で学んだ分析手法を実践するためのチェックリストです。
決算書を読む際にご活用ください。

---

## 1. 収益性チェック（第1話・第2話）

### 営業利益率の確認
- [ ] 営業利益率を計算したか（営業利益÷売上高×100）
- [ ] 過去3〜5年の推移を確認したか
- [ ] 同業他社と比較したか
- [ ] 業界平均と比較したか

### セグメント分析
- [ ] 有価証券報告書のセグメント情報を確認したか
- [ ] 事業別・地域別の売上高構成比を把握したか
- [ ] 各セグメントの営業利益率を比較したか
- [ ] 「稼ぎ頭」と「足を引っ張る事業」を識別したか

---

## 2. キャッシュフロー分析（第3話）

### 3つのCFバランス
- [ ] 営業CFはプラスか（本業で稼いでいるか）
- [ ] 投資CFの内訳を確認したか（成長投資か資産売却か）
- [ ] 財務CFの内訳を確認したか（借入増か返済か）

### パターン判定
| 営業CF | 投資CF | 財務CF | 企業の状態 |
|:---:|:---:|:---:|---|
| ＋ | − | − | 健全な成熟企業 |
| ＋ | − | ＋ | 積極投資中の成長企業 |
| ＋ | ＋ | − | リストラ・資産売却中 |
| − | − | ＋ | 要注意（借入で投資） |
| − | ＋ | ＋ | 危険信号（資産売却＋借入で生存） |

### 利益との整合性
- [ ] 営業利益と営業CFを比較したか
- [ ] 大きな乖離がある場合、原因を特定したか
- [ ] フリーキャッシュフロー（営業CF+投資CF）を確認したか

---

## 3. 財務健全性チェック（第4話）

### 貸借対照表の基本分析
- [ ] 自己資本比率を計算したか（自己資本÷総資産×100）
- [ ] 流動比率を計算したか（流動資産÷流動負債×100）
- [ ] 有利子負債の金額と推移を確認したか

### 目安となる水準
| 指標 | 安全圏 | 要注意 |
|---|---|---|
| 自己資本比率 | 30%以上 | 20%未満 |
| 流動比率 | 150%以上 | 100%未満 |

### 資産の質チェック
- [ ] 売上債権（売掛金）の増減を確認したか
- [ ] 棚卸資産（在庫）の増減を確認したか
- [ ] 売上高の伸びと比較して異常な増加はないか

---

## 4. 銀行財務チェック（第5話）

銀行を分析する際の特殊なポイント：

- [ ] 自己資本比率（BIS基準）を確認したか
- [ ] 預貸率を確認したか
- [ ] 与信コスト（貸倒引当金繰入額）の推移を確認したか
- [ ] 不良債権比率を確認したか

---

## 5. 効率性・収益性チェック（第6話）

### ROE分析
- [ ] ROEを計算したか（当期純利益÷自己資本×100）
- [ ] 同業他社と比較したか
- [ ] 日本企業平均（約8%）と比較したか

### デュポン分析（ROEの3分解）
- [ ] 売上高利益率（当期純利益÷売上高）
- [ ] 総資産回転率（売上高÷総資産）
- [ ] 財務レバレッジ（総資産÷自己資本）
- [ ] ROEが高い理由（低い理由）を特定したか

### ROA分析
- [ ] ROAを計算したか（当期純利益÷総資産×100）
- [ ] ROEとROAの差（レバレッジ効果）を確認したか

---

## 6. バリュエーションチェック（第7話）

### 株価指標
- [ ] PERを確認したか（株価÷EPS）
- [ ] 同業他社のPERと比較したか
- [ ] PBRを確認したか（株価÷BPS）
- [ ] PBR1倍割れの場合、その理由を考察したか

### 割安・割高の判断
- [ ] 成長期待を織り込んだ評価か考察したか
- [ ] 業績の見通しを確認したか
- [ ] 「割安には理由がある」可能性を検討したか

---

## 7. 成長企業分析（第8話）

### ユニットエコノミクス
- [ ] 売上高成長率を確認したか
- [ ] 赤字の場合、投資フェーズか確認したか
- [ ] 決算説明資料でKPIを確認したか

### チェックポイント
- [ ] 顧客獲得コスト（CAC）は適正か
- [ ] 顧客生涯価値（LTV）との比率はどうか
- [ ] チャーンレート（解約率）は許容範囲か
- [ ] 黒字化の見通しはあるか

---

## 8. 成熟企業分析（第9話）

### 株主還元
- [ ] 配当利回りを確認したか
- [ ] 配当性向を確認したか
- [ ] 自社株買いの実績を確認したか
- [ ] 総還元性向を計算したか

### 安定性チェック
- [ ] 過去10年の業績推移を確認したか
- [ ] 景気変動への耐性を確認したか
- [ ] 市場シェアの安定性を確認したか

## 10. 不正・異常発見チェック（第11話・第12話）

### 危険信号（レッドフラッグ）
- [ ] 売上債権回転期間が異常に長くないか
- [ ] 在庫回転期間が異常に長くないか
- [ ] 営業利益と営業CFに大きな乖離がないか
- [ ] 売上高は増えているのにCFは減っていないか

### 関連当事者取引チェック
- [ ] 有価証券報告書の「関連当事者との取引」を確認したか
- [ ] 取引金額の急増がないか
- [ ] 取引条件に不自然な点がないか
- [ ] 役員が関係する会社との取引がないか

### 異常検知の計算式
```
売上債権回転期間 = 売上債権 ÷ 売上高 × 365日
在庫回転期間 = 棚卸資産 ÷ 売上原価 × 365日
```
