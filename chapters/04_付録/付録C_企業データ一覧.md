# 付録C：本書で取り上げた企業データ一覧

本書で分析事例として登場した企業の概要と、学習のポイントをまとめました。

## 1. ファーストリテイリング（ユニクロ）

### 企業概要
| 項目 | 内容 |
|---|---|
| 証券コード | 9983 |
| 業種 | 小売業（アパレル） |
| 主要ブランド | ユニクロ、GU、Theory |
| 決算期 | 8月 |

### 本書での登場
| 話数 | 分析テーマ |
|---|---|
| 第1話 | 営業利益率の国際比較 |
| 第2話 | セグメント分析（地域別業績） |
| 第10話 | 競合分析（vsしまむら） |

### 学びのポイント

- **海外展開**：セグメント情報で地域別成長を分析
- **規模の経済**：売上規模拡大と利益率向上の関係
