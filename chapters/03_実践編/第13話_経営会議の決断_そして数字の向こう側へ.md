## プロローグ：決戦の朝

13週目の月曜日、午前8時。

ミサキは経営会議室の前で深呼吸した。

手には、11週間の学びをすべて注ぎ込んだ発表資料。USBメモリが汗ばんだ手の中で滑る。

「田中さん」

振り返ると、経営企画部長の山田が立っていた。

「緊張してる？」

「……正直、はい」

山田部長は小声で言った。

「今日の発表、**黒田専務も出席する**。いつもは欠席が多いんだが、今日は『経営企画部の新人の発表を聞きたい』と」

ミサキの背筋が凍った。

「私の発表を……わざわざ？」

「何かあったのか？」

「いえ、何も」

山田部長が去った後、ミサキはスマートフォンを取り出した。

安藤先生からのメッセージ。

```
「数字は君の味方だ。
 何があっても、数字から目を逸らすな。
 君の11週間を、信じろ」
```

ミサキは深く息を吸い込んだ。

（逃げない。最後まで、自分の言葉で）


## コラム：「数字が読める」とは何か

13週間の旅が終わった。ミサキは何を手に入れたのだろうか。

**① 数字を「見る」から「読む」へ**

最初、決算書は「数字の羅列」だった。今は、一つひとつの数字が「意味」を持って見える。営業利益率16.1%——その向こうに、SPAというビジネスモデルが見える。売上債権回転期間98日——その向こうに、何かおかしなことが起きている気配がある。

**② 「比較」という武器**

単独の数字には、意味がない。競合と比べて、過去と比べて、業界平均と比べて——初めて「高い」「低い」「異常」がわかる。

**③ 「仮説」を立てる力**

数字を見たら、「なぜ？」と問う。仮説を立て、検証する。その繰り返しが、真実に近づく道だ。

**④ 行動する勇気**

数字を読めるだけでは、世界は変わらない。発見したことを、誰かに伝える。必要なら、声を上げる。それが、数字を読む本当の意味だ。
