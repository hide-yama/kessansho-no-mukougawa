# 第11話　業界分析と自社診断——不穏な数字

---

## プロローグ：宿題の答え、そして衝撃の発見

11週目の土曜日。

ミサキの手は震えていた。

「先生、東洋テックを競合と比較してきました。でも……」

声が詰まる。

「落ち着いて。何がわかった？」

安藤先生は静かに促した。

ミサキはノートを開いた。先週の宿題——売上債権回転期間、棚卸資産回転期間、営業CFマージンで競合と比較せよ。

その答えが、**信じたくない現実**を突きつけていた。

```
【東洋テック vs 電機業界平均 比較】

                      東洋テック    業界平均
───────────────────────────────────────────
売上債権回転期間        98日         65日
棚卸資産回転期間        85日         55日
営業CFマージン          2.1%         8.5%
```

「東洋テックだけ、**すべての指標で悪い**んです。しかも、3年前から急激に悪化している」

安藤先生は深く息を吐いた。

「予想はしていた。だが、これほどとは……」

---

## 第1章：業界分析の視点

「まず、冷静に分析手順を確認しよう」

安藤先生はホワイトボードに書いた。

```
【業界分析の3つの視点】

① 業界全体の構造
   → 成長率、競争環境、利益率の傾向

② 各社のポジション
   → シェア、強み、戦略の違い

③ 自社の位置づけ
   → 業界の中で相対的に優位か、劣位か
```

「電機業界全体の状況はどうだった？」

「総合電機メーカーは厳しい状況です。家電は中国・韓国勢に押され、重電はインフラ投資の減少。各社とも事業の選択と集中を進めています」

「日立は？」

「ITとインフラに特化して、上場子会社を整理。ROEは10%を超えるまで改善しました」

「パナソニックは？」

「車載電池とサプライチェーンに注力。家電依存から脱却を図っています」

「では、東洋テックは何に注力している？」

ミサキは言葉に詰まった。

「……わからないんです。海外事業を強化しているとは言っていますが、セグメント別の利益率は上がっていない」

---

## 第2章：売上債権回転期間の異常

「最初の指標、**売上債権回転期間**を詳しく見よう」

```
【売上債権回転期間とは】

売上債権回転期間 = 売上債権 ÷ 売上高 × 365日

意味：売上を現金として回収するのに何日かかるか
```

「東洋テックは98日。業界平均は65日。**1ヶ月以上長い**」

「これは何を意味する？」

ミサキは考えた。

「売掛金の回収が遅れている……お客さんがお金を払ってくれない？」

「それも一つの可能性だ。他には？」

安藤先生は鋭い目でミサキを見た。

「**売上を計上したけれど、実際には入金されない**——架空売上の可能性」

声にするだけで背筋が凍った。

「まだ断定はできない。でも、異常な数字には理由がある」

安藤先生は続けた。

```
【売上債権回転期間が長くなる原因】

① 正当な理由
   ・顧客の資金繰り悪化
   ・取引条件の変更（支払サイト延長）
   ・大型案件の増加（検収遅れ）

② 問題がある理由
   ・架空売上の計上
   ・押し込み販売（期末に無理な出荷）
   ・貸倒懸念のある債権を放置
```

「3年間で30日も延びている。正当な理由だけでは説明しにくい」

---

## 第3章：棚卸資産回転期間の異常

「次に、**棚卸資産回転期間**」

```
【棚卸資産回転期間とは】

棚卸資産回転期間 = 棚卸資産 ÷ 売上原価 × 365日

意味：在庫を販売するのに何日かかるか
```

ミサキはデータを示した。

「東洋テックは85日。業界平均は55日。こちらも**1ヶ月長い**」

「推移は？」

```
【東洋テック 棚卸資産回転期間の推移】

  3年前：58日（業界平均並み）
  2年前：68日
  1年前：76日
  今期 ：85日

  → 3年で27日も延長
```

「急激に悪化しています」

「在庫が増える原因は？」

「売れない商品が溜まっている……」

「もう一つ」

ミサキは気づいた。

「**在庫を水増し**している。実際より多く計上することで、売上原価を減らし、**利益を膨らませる**」

```
【在庫操作の仕組み】

  売上高        1,000
 -売上原価      ▲700  ← 在庫を増やすと原価が減る
 ─────────────────
  粗利益         300  ← 利益が増える

  カラクリ：
  期末在庫を水増し → 売上原価が減少 → 利益が増加
```

「在庫の金額を誤魔化すだけで、見かけの利益が増える……」

安藤先生は頷いた。

「古典的な粉飾手法の一つだ」

---

## 第4章：営業CFマージンの乖離

「三つ目の指標、**営業CFマージン**」

```
【営業CFマージンとは】

営業CFマージン = 営業CF ÷ 売上高 × 100

意味：売上の何%が現金として残るか
```

「東洋テックは2.1%。業界平均は8.5%。**4分の1以下**」

安藤先生は重要なポイントを示した。

「第4話で学んだね。**利益の質**を見る指標だ」

```
【営業利益 vs 営業CF 比較】

              東洋テック    業界平均
────────────────────────────────────
営業利益率      5.2%         6.0%
営業CFマージン  2.1%         8.5%
────────────────────────────────────
差              ▲3.1pt       +2.5pt
```

ミサキは愕然とした。

「業界平均は営業CFが営業利益を**上回っている**。減価償却費が加算されるから、普通はそうなる」

「でも東洋テックは営業CFが営業利益を**大幅に下回っている**。これは……」

「**利益が現金化されていない**。なぜだと思う？」

ミサキは今日の分析を振り返った。

「売掛金の異常な増加と、在庫の異常な増加。**利益は出ているのに、現金が入ってこない**」

「そういうことだ」

---

## 第5章：3つの異常が示すもの

「ここで、三つの異常を整理しよう」

安藤先生はホワイトボードにまとめた。

```
【東洋テックの3つの異常】

① 売上債権回転期間：98日（業界平均65日）
   → 売上の回収に異常に時間がかかる
   → 架空売上の可能性？

② 棚卸資産回転期間：85日（業界平均55日）
   → 在庫が異常に膨張している
   → 在庫水増しの可能性？

③ 営業CFマージン：2.1%（業界平均8.5%）
   → 利益が現金化されていない
   → 利益の「質」に問題？

【共通する特徴】
  ・3年前から急激に悪化
  ・業界平均と大きく乖離
  ・3つの指標が同時に悪化
```

「3つが同時に悪化している……偶然とは思えません」

「そうだ。一つだけなら説明がつくこともある。でも、三つ同時に、しかも3年連続で悪化——これは**構造的な問題**だ」

ミサキは震える声で言った。

「先生、これって……**粉飾決算**ですか？」

安藤先生は長い沈黙の後、答えた。

「可能性は否定できない。ただし、まだ断定はできない」

---

## 第6章：海外事業の謎

「もう一つ、気になるデータがある」

ミサキは追加の分析を示した。

```
【東洋テック セグメント別分析】

■ 国内事業
  売上成長率：+2%/年（横ばい）
  営業利益率：7.8%（安定）
  売上債権回転期間：62日（正常）

■ 海外事業
  売上成長率：+15%/年（急成長）
  営業利益率：3.2%（低い）
  売上債権回転期間：158日（異常）
```

安藤先生の目が光った。

「**海外事業だけ**異常が集中している」

「はい。国内事業は問題ありません。でも海外事業は——」

「売上債権回転期間158日。**5ヶ月以上**回収にかかる計算だ」

ミサキは有価証券報告書を調べた情報を伝えた。

「海外事業を統括しているのは……**黒田専務**です」

「黒田専務？」

「取締役会で『海外事業を立て直した功労者』と紹介されていました。次期社長の最有力候補だとも」

安藤先生の表情が険しくなった。

「売上が急成長しているのに、利益率は低く、売掛金は膨張——これは典型的な**粉飾パターン**だ」

## 第7章：なぜ3年前から？

「3年前に何があったか、調べた？」

「はい」

ミサキはメモを読み上げた。

```
【3年前の出来事】

■ 社内
  ・黒田専務が海外事業本部長に就任
  ・「海外売上高比率40%」の中期目標発表
  ・海外子会社の大幅増設

■ 業界
  ・中国市場の競争激化
  ・為替が円高に振れる
  ・同業他社が海外事業を縮小
```

「同業他社が**縮小**しているのに、東洋テックは**拡大**した」

「黒田専務の就任と同時に、海外事業の数字がおかしくなり始めた……」

安藤先生は静かに言った。

「**相関関係は因果関係ではない**。でも、無視できない符合だ」

## 第8章：ミサキの葛藤

「先生……私、どうすればいいんですか？」

ミサキの目には涙が浮かんでいた。

「経営会議で発表する分析に、これを入れるべきでしょうか。でも、黒田専務は次期社長候補です。私はただの若手社員で——」

「ミサキさん」

安藤先生は真剣な眼差しで言った。

「数字は**嘘をつかない**。でも、数字を**隠す**ことはできる。君が発見したことを黙っていれば、誰も気づかないかもしれない」

「……」

「でも、それで君は**自分を許せるか**？」

ミサキは自分の手を見た。9週間前は、決算書を開くことすら怖かった。今は、数字が何を語っているか、読めるようになった。

「私が見つけなければ、誰も気づかないかもしれない」

「そうだ」

「でも、間違っていたら……私のキャリアは終わりです」

安藤先生は優しく言った。

「だから、**もっと証拠を固める**必要がある。来週、徹底的に調べよう」

## 第9章：次週への準備

「経営会議まであと2週間。やるべきことを整理しよう」

```
【証拠固めのチェックリスト】

① 関連当事者取引の確認
   → 海外子会社との取引に不自然な点がないか

② 監査法人の意見を確認
   → 監査報告書に「懸念」が記載されていないか

③ 過去の修正申告を確認
   → 過去に会計処理の訂正がなかったか

④ 同業他社の詳細比較
   → 東洋テックだけの問題か、業界全体か

⑤ セグメント情報の精査
   → 海外子会社ごとの数字は取れるか
```

「来週までに、これらを調べてきてほしい」

ミサキは深く頷いた。

「わかりました。逃げません」

安藤先生は微笑んだ。

「君は強くなった。数字の向こう側にある**真実**を見つける力がついた」

ミサキは長い沈黙の後、答えた。

「わかりません。でも、**数字から逃げない**覚悟はあります」

「来週、**すべての証拠を揃えよう**。そして、君自身が判断すればいい」

## エピローグ：数字が語る真実

帰り道、ミサキの頭は混乱していた。

売上債権回転期間98日。棚卸資産回転期間85日。営業CFマージン2.1%。

**数字は嘘をつかない**——安藤先生の言葉が頭をよぎる。

でも、数字を**操作する人間**はいる。

3年前から海外事業の数字がおかしい。黒田専務が本部長に就任した年からだ。

（本当に……粉飾なの？）

ミサキは空を見上げた。

9週間前、自分は何も知らなかった。決算書は「難しい書類」でしかなかった。

今は違う。数字の向こう側に、**人の意思**が見える。

良い意思も、悪い意思も。

（私は、真実を見つけてしまった……）

ミサキは拳を握りしめた。

（逃げない。最後まで、自分の目で確かめる）

## コラム：粉飾決算の「早期発見シグナル」

粉飾決算——会社が意図的に決算書の数字を操作すること。投資家にとっては最大のリスクの一つだ。では、どうやって見抜くのか。

**5つの早期発見シグナル**

```
① 売掛金の異常な増加
   → 売上の伸び以上に売掛金が増えている
   → 「架空売上」のサイン

② 在庫の異常な増加
   → 売上原価の伸び以上に在庫が増えている
   → 「在庫水増し」のサイン

③ 利益とキャッシュフローの乖離
   → 利益は出ているのに現金が増えない
   → 「利益の質」に問題

④ 監査法人の交代
   → 特に理由なく監査法人が変わる
   → 「会計処理で揉めた」可能性

⑤ 経営陣の株式売却
   → 業績好調なのに経営者が株を売る
   → 「何かを知っている」可能性
```

**重要な心構え**

粉飾を見つけても、すぐに「不正だ」と断定してはいけない。正当な理由がある場合もある。

大切なのは、**異常を発見したら、その原因を徹底的に調べる**こと。

会社の説明、業界の動向、過去の事例——多角的に検証して、初めて判断できる。

数字は嘘をつかない。でも、数字を読む**私たちの姿勢**が問われている。
