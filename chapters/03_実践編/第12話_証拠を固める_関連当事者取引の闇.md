# 第12話　証拠を固める——関連当事者取引の闇

---

## プロローグ：決定的な証拠を求めて

12週目の土曜日。

経営会議まで、あと1週間。

ミサキは分厚い資料を抱えてカフェに入った。有価証券報告書、四半期報告書、そして——**関連当事者取引の注記**のコピー。

「先生、調べてきました」

安藤先生はコーヒーカップを置いて、資料を受け取った。

「関連当事者取引。ここに**すべての答え**がある可能性が高い」

ミサキは深呼吸した。

「覚悟はできています。真実を、見つけます」

---

## 第1章：関連当事者取引とは

「まず、**関連当事者取引**の基本を確認しよう」

```
【関連当事者取引とは】

会社と「特別な関係にある者」との取引

■ 関連当事者の例
  ・親会社、子会社
  ・関連会社
  ・役員とその近親者
  ・役員が支配する会社

■ なぜ開示が必要か
  ・通常の取引条件と異なる可能性
  ・利益相反の可能性
  ・株主が知るべき情報
```

「有価証券報告書には、関連当事者との重要な取引を**すべて開示**する義務がある」

「隠すことはできないんですね」

「法律上はね。でも、**金額が小さければ開示基準以下**になる。複数の取引に分散させれば、目立たなくなる」

---

## 第2章：東洋テックの関連当事者取引

「東洋テックの関連当事者取引を見てみよう」

ミサキは調べてきた内容を示した。

```
【東洋テック 関連当事者取引（抜粋）】

■ 海外子会社A（シンガポール）
  取引内容：製品販売
  取引金額：380億円
  売掛金残高：125億円

■ 海外子会社B（香港）
  取引内容：製品販売、業務委託
  取引金額：290億円
  売掛金残高：98億円

■ 海外子会社C（タイ）
  取引内容：製品販売
  取引金額：210億円
  売掛金残高：72億円

合計売掛金残高：295億円
```

安藤先生の目が鋭くなった。

「売掛金残高295億円——これは**東洋テック全体の売掛金の何%だ**？」

ミサキは計算した。

「全体の売掛金が約850億円なので……**35%**です」

「海外子会社向けの売掛金が、全体の3分の1を占めている」

## 第3章：異常の発見

「ここで、**回収状況**を確認しよう」

```
【海外子会社向け売上 vs 売掛金回収】

               売上    売掛金   回転期間
─────────────────────────────────────
子会社A        380億   125億    120日
子会社B        290億    98億    123日
子会社C        210億    72億    125日
─────────────────────────────────────
合計           880億   295億    122日

※通常の取引先：回転期間 60日
```

ミサキは息を呑んだ。

「**4ヶ月**も回収にかかっている……」

「外部の取引先なら、こんな条件では取引しない。でも子会社なら——」

「支払いを**わざと遅らせる**ことができる」

安藤先生は頷いた。

「子会社に売上を計上し、回収を遅らせる。親会社の売上は増え、利益も増える。でも現金は入ってこない」

「これが……**架空売上**の手口ですか？」

「完全な架空ではないかもしれない。でも、**実質的に回収不能な売上**を計上している可能性がある」

---

## 第4章：さらに深い闇

「もう一つ、気になることがある」

ミサキは追加の資料を出した。

```
【海外子会社の財務状況（推定）】

■ 子会社A（シンガポール）
  売上高：520億円
  営業利益：▲15億円（赤字）
  純資産：12億円

■ 子会社B（香港）
  売上高：410億円
  営業利益：▲22億円（赤字）
  純資産：8億円

■ 子会社C（タイ）
  売上高：280億円
  営業利益：▲8億円（赤字）
  純資産：5億円
```

安藤先生の表情が険しくなった。

「三社とも**赤字**。そして純資産は——」

「合計25億円。売掛金295億円に対して、**10分の1以下**です」

「つまり、この子会社たちには**返済能力がない**」

ミサキは震える声で言った。

「295億円の売掛金は、**回収できない可能性が高い**……」

「そういうことだ」

---

## 第5章：なぜ誰も気づかなかったのか

「でも先生、なぜこんな状態が放置されているんですか？監査法人は何をしていたんですか？」

「良い質問だ。監査報告書を見てみよう」

```
【監査報告書の確認ポイント】

■ 監査意見
  → 「適正意見」が付いている
  → ただし、これは「粉飾がない」という意味ではない

■ 強調事項
  → 「継続企業の前提に関する重要な不確実性」
  → 今回は記載なし

■ KAM（監査上の主要な検討事項）
  → 「海外子会社に対する債権の回収可能性」
  → 記載あり！
```

ミサキは目を見開いた。

「KAMに**記載されている**！」

「監査法人も気づいているんだ。でも、**適正意見を出す範囲内**だと判断した」

「どういうことですか？」

「会計基準上、売掛金の全額が回収不能だと**証明できない限り**、引当金を積む必要はない。子会社が存続している以上、『いずれ回収できる』という主張が通る」

ミサキは悔しそうに言った。

「**法律の範囲内**で、数字を操作している……」

「グレーゾーンだ。完全な違法ではない。でも、**投資家を誤解させる**意図があるなら、それは問題だ」

「ここで、**人**の動きを追ってみよう」

```
【黒田専務の経歴】

・25年前：東洋テック入社（営業部門）

・8年前：執行役員・アジア統括

【海外子会社の役員構成】
  子会社A：社長は黒田専務の元部下
  子会社B：社長は黒田専務が人選
  子会社C：社長は黒田専務の元部下
```

ミサキは気づいた。

「三社とも、**黒田専務の息のかかった人間**がトップ……」

「3年前、黒田専務が海外事業本部長に就任した年から、海外子会社向けの売上が急増している」

安藤先生は重要な指摘をした。

「偶然か、それとも——」

「意図的に、**業績を良く見せるため**に……」

## 第7章：もう一つの証拠

「最後に、**もう一つ**決定的な資料を見つけました」

ミサキは過去の有価証券報告書を開いた。

```
【過去の会計処理の訂正】

■ 2年前の第3四半期
  訂正内容：海外子会社向け売上の計上時期を訂正
  影響額：売上高▲42億円、営業利益▲18億円
  理由：「出荷基準から検収基準への変更に伴う調整」

■ 経営陣のコメント
  「会計基準の厳格な適用のため」
  「不正ではない」
```

安藤先生は深くため息をついた。

「**過去に一度、訂正している**。しかも海外子会社向けの売上で」

「これは……『バレかけた』ということですか？」

「かもしれない。そして、その後も**同じことを続けている**」

ミサキは確信した。

「先生、これは偶然ではありません。**組織的に、意図的に**行われている」

「そう考えざるを得ない」

## 第8章：告発の準備

「証拠は揃った。あとは——」

安藤先生はミサキを真剣に見つめた。

「君が**どう行動するか**だ」

ミサキは深呼吸した。

「経営会議で、この分析を発表します」

「黒田専務は、次期社長の最有力候補だ。君の発表は、彼の**地位を脅かす**ことになる」

「わかっています」

「会社は君を守ってくれないかもしれない。むしろ、**口封じ**に動く可能性すらある」

ミサキは拳を握りしめた。

「それでも、**数字は嘘をつかない**。私が見つけた真実を、黙っていることはできません」

安藤先生は微笑んだ。

「9週間前の君からは、想像できない言葉だ」

## 第9章：発表資料の構成

「では、発表資料の構成を考えよう」

```
【経営会議 発表資料（案）】

■ タイトル
  「東洋テックの財務分析——競合比較と課題抽出」

■ 構成
  1. 業界動向と競合比較
     → 表向きの分析（正当な内容）

  2. 自社の財務分析
     → 売上債権回転期間、棚卸資産回転期間、営業CFマージン

  3. 異常値の指摘
     → 業界平均との乖離、3年間の推移

  4. 海外事業の詳細分析
     → 関連当事者取引、子会社の財務状況

  5. 提言
     → 「調査委員会の設置」を提案

■ ポイント
  ・「不正」とは断定しない
  ・「異常値がある」「調査が必要」という表現
  ・感情的にならず、数字で語る
```

「**不正**とは言わないんですね」

「そうだ。断定は危険だ。数字の異常を指摘し、**第三者による調査**を提案する。それが君にできる最善の方法だ」

## コラム：内部告発と「声を上げる」ということ

企業の不正を発見したとき、社員はどうすべきか。これは非常に難しい問題だ。

**日本の現状**

2022年に改正された公益通報者保護法では、内部告発者の保護が強化された。しかし、現実には告発者が不利益を被るケースは少なくない。

```
【内部告発のジレンマ】

告発した場合：
  ・会社から報復を受ける可能性
  ・キャリアに傷がつく可能性
  ・「裏切り者」と見なされる可能性

告発しなかった場合：
  ・不正が継続する
  ・将来、より大きな問題になる
  ・自分も「共犯」になる可能性
```

**「異常の指摘」という方法**

本話でミサキが選んだのは、「不正を告発する」のではなく、「異常を指摘し、調査を提案する」という方法だ。

これは現実的なアプローチと言える。

・断定を避けることで、名誉毀損のリスクを減らす
・数字という客観的事実を示す
・第三者調査という「公正なプロセス」を求める

**数字の力**

財務分析のスキルは、こうした場面で**武器**になる。

「なんとなくおかしい」ではなく、「売上債権回転期間が業界平均より33日長い」と言える。感覚ではなく、**数字で語る**ことで、主張に説得力が生まれる。

声を上げることは勇気がいる。でも、数字という味方がいれば、一人でも戦える。
