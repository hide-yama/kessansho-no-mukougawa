## プロローグ：宿題の答え、そして実践への挑戦

10週目の土曜日。

「先生、ユニクロとしまむらの比較をしてきました」

ミサキはノートを開いた。この2週間で最も密度の濃い分析だった。

```
【ユニクロ vs しまむら 財務比較】

                    ユニクロ        しまむら
─────────────────────────────────────────────
売上高              3兆1,038億円    6,270億円
営業利益            5,009億円       430億円
営業利益率          16.1%           6.9%
粗利益率            50.3%           33.8%
販管費率            38.4%           26.9%
ROE                 約24%           約9%
店舗数              3,595店         2,200店
```

「同じアパレル業界なのに、営業利益率が**16%と7%**——2倍以上違います」

安藤先生は頷いた。

「なぜこんなに違うのか、仮説は立てた？」

「はい。ビジネスモデルの違いだと思います。ユニクロは**SPA**で自社企画・製造。しまむらは**仕入販売**で他社から買い付け」

「素晴らしい。今日は、その違いをさらに深掘りしよう」

安藤先生は続けた。

「今日から**第III部：実践編**だ。これまで学んだスキルを、**君自身が使いこなす**番だよ」

## 第1章：競合分析とは何か

「まず、競合分析の目的を明確にしよう」

安藤先生はホワイトボードに書いた。

```
【競合分析の目的】

① 自社の強み・弱みを相対化する
   → 比較してはじめて「強い」「弱い」がわかる

② 業界の構造を理解する
   → 利益の源泉はどこにあるか

③ 戦略の違いを見極める
   → 同じ業界でも「戦い方」は異なる
```

「重要なのは、**優劣をつけることではない**。違いを理解することだ」

「違いを理解する……」

「ユニクロが勝っていて、しまむらが負けている——そういう話ではない。それぞれに**異なる戦略**があり、**異なる強み**がある」

---

## 第2章：ビジネスモデルの違い

「まず、根本的なビジネスモデルの違いを見よう」

```
【SPA vs 仕入販売】

■ SPA（ユニクロ）
  企画 → 素材調達 → 製造 → 物流 → 販売

  特徴：
  ・全工程を自社でコントロール
  ・中間マージンを排除
  ・粗利益率が高い（50%）
  ・在庫リスクは自社が負う

■ 仕入販売（しまむら）
  仕入 → 販売

  特徴：
  ・メーカーから完成品を買い取り
  ・買い取り価格を抑えて利益確保
  ・粗利益率は低い（34%）
  ・仕入先との関係が重要
```

「SPAは粗利益率が高いけど、リスクも高いんですね」

「その通り。自分で企画・製造するから、売れなかったら在庫は全部自分持ち。しまむらは仕入れた商品が売れなくても、リスクは分散される」

---

## 第3章：粗利益率と販管費率

「財務構造を詳しく見てみよう」

```
【利益構造の分解】

                    ユニクロ        しまむら
─────────────────────────────────────────────
売上高              100%            100%
- 売上原価          ▲49.7%         ▲66.2%
─────────────────────────────────────────────
粗利益              50.3%           33.8%
- 販管費            ▲38.4%         ▲26.9%
─────────────────────────────────────────────
営業利益            16.1%           6.9%
```

ミサキは数字を見比べた。

「ユニクロは粗利益率が高いけど、販管費率も高い。しまむらは粗利益率が低いけど、販管費率も低い」

「その違いはどこから来ていると思う？」

「えっと……ユニクロは広告宣伝費やブランド投資にお金をかけている。しまむらはローコストオペレーションで運営している」

「正解だ」

安藤先生は詳細を示した。

```
【販管費の内訳推定】

■ ユニクロ
  ・広告宣伝費：売上の5〜7%（500億円規模）
  ・人件費：正社員中心
  ・店舗投資：都心の一等地に旗艦店

■ しまむら
  ・広告宣伝費：ほぼゼロ（折込チラシ中心）
  ・人件費：パート社員中心
  ・店舗投資：郊外ロードサイド
```

## 第4章：ポジショニングの違い

「二社の**ポジショニング**を整理しよう」

```
【ポジショニングマップ】

        高価格
          ↑
          │    ●ユニクロ
          │    （機能×ベーシック）
          │
低機能←──┼──→高機能
          │
          │    ●しまむら
          │    （トレンド×低価格）
          ↓
        低価格
```

「ユニクロは**機能性×ベーシック**。しまむらは**トレンド×低価格**」

「同じ『安い服』でも、価値の出し方が全く違うんですね」

「そう。ユニクロはヒートテックやエアリズムなど、**機能素材**で差別化する。しまむらは**最新トレンドを低価格で提供**することで差別化する」

---

## 第5章：強みと弱み

「ここで、両社の強みと弱みを整理してみよう」

ミサキは自分なりに考えた。

```
【ユニクロの強みと弱み】

強み：
  ・SPAによる高い粗利益率
  ・機能素材（ヒートテック、エアリズム）
  ・グローバル展開（海外売上55%）
  ・ブランド力

弱み：
  ・販管費が高い（広告、店舗投資）
  ・在庫リスクを全て自社が負う
  ・トレンド対応が弱い（ベーシック特化）
```

```
【しまむらの強みと弱み】

強み：
  ・徹底したローコストオペレーション
  ・広告費ほぼゼロでも集客できる
  ・パート社員活用の効率経営
  ・在庫リスクの分散

弱み：
  ・粗利益率が低い
  ・海外展開が限定的
  ・ブランド力が弱い
  ・成長余地が限られる
```

「どちらが『正解』というわけではない。**それぞれの戦略に合った財務構造**になっている」

## 第6章：なぜ利益率が2倍違うのか

「ここで核心の問いに答えよう」

安藤先生はホワイトボードにまとめた。

```
【利益率の差を生む要因】

① ビジネスモデルの違い
   ・SPA vs 仕入販売
   ・粗利益率50% vs 34%

② 投資戦略の違い
   ・ユニクロ：高投資→高リターン
   ・しまむら：低投資→安定リターン

③ ポジショニングの違い
   ・ユニクロ：機能×グローバル→プレミアム
   ・しまむら：トレンド×ローカル→効率

④ スケールメリット
   ・ユニクロ：3兆円規模で調達力が強い
   ・しまむら：6,000億円規模で規模の差
```

「結論として、**どちらが優れているかではなく、どちらも自社の戦略に最適化されている**」

ミサキは納得した。

「だから、利益率だけ見て『ユニクロの方が良い』とは言えないんですね」

「その通り。しまむらは低い利益率でも、ローコスト経営で**安定した収益**を上げ続けている。それも一つの成功モデルだ」

## 第7章：競合分析のフレームワーク

「今日学んだ分析手法を、フレームワークとして整理しよう」

```
【競合分析の5ステップ】

Step 1：財務数値の比較
   ・売上、利益、利益率
   ・粗利益率、販管費率の分解

Step 2：ビジネスモデルの理解
   ・どうやって稼いでいるか
   ・バリューチェーンの違い

Step 3：ポジショニングの把握
   ・顧客は誰か
   ・どんな価値を提供しているか

Step 4：強みと弱みの整理
   ・財務に表れる強み
   ・定性的な強み

Step 5：差の理由を突き止める
   ・なぜこの差が生まれたか
   ・戦略の違いか、実行力の違いか
```

「このフレームワークは、どんな業界でも使えるんですか？」

「基本的には使える。ただし、業界によって見るべきポイントは変わる」

## コラム：なぜ「ライバル」を研究すべきなのか

ビジネスの世界では「ライバルを研究しろ」とよく言われる。なぜ競合を知ることが重要なのか。

**① 自社の位置がわかる**

自社の営業利益率が10%だとする。これは高いのか、低いのか。業界平均が5%なら高い。15%なら低い。比較して初めて「自分の立ち位置」がわかる。

**② 改善のヒントが見つかる**

競合の方が利益率が高いなら、なぜかを調べる。原価率の違い？販管費の違い？理由がわかれば、改善のヒントになる。

**③ 差別化ポイントが明確になる**

競合と同じことをしていては勝てない。違いを理解することで、「自社は何で勝負すべきか」が見えてくる。

**④ 業界の変化を察知できる**

競合の動きは、業界全体の変化を映す鏡だ。新しい戦略、新しいビジネスモデル——競合を観察することで、変化の兆しをキャッチできる。

**注意点：数字だけでは見えないもの**

ただし、財務数値だけで競合分析は完結しない。顧客の声、従業員の士気、経営者のビジョン——定性的な情報も重要だ。

数字は「過去の結果」を映す。未来を予測するには、「なぜその結果になったか」を深掘りする必要がある。
