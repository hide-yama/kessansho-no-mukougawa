# 第4話　貸借対照表——企業の「体質」診断

---

## プロローグ：三つ目の地図

「先生、宿題をやってきました」

ミサキはノートを開いた。東洋テックのキャッシュフロー計算書を、過去3年分まとめてある。

「どうだった？」

「営業キャッシュフローはプラスでした。ただ……」

「ただ？」

「**利益の伸びに比べて、営業CFの伸びが小さい**気がして。何かが引っかかるんですけど、うまく説明できなくて」

安藤先生は静かにうなずいた。

「いい気づきだね。その違和感、覚えておいて。今日の学びが進めば、きっと理由がわかるようになる」

ミサキは首をかしげた。

「今日は何を学ぶんですか？」

「**三つ目の地図**を渡そう」

安藤先生はホワイトボードに大きく書いた。

```
財務三表

① 損益計算書（P/L）  → 1年間の「成績表」
② キャッシュフロー計算書（CF） → 1年間の「お金の流れ」
③ 貸借対照表（B/S）  → ある時点の「財産目録」
```

「損益計算書で利益を見た。キャッシュフロー計算書でお金の流れを見た。今日は**貸借対照表**——会社の『体質』を診断する方法を教えるよ」

## Scene 1：スナップショットとしてのB/S

「田中さん、健康診断を受けたことはある？」

「はい、毎年会社で」

「血液検査の結果を思い出してみて。**コレステロール値**や**血糖値**——あれは何を測っている？」

ミサキは考えた。

「その時点での……体の状態、ですか？」

「そう。**ある瞬間の体の状態**を数値で表している。貸借対照表も同じなんだ」

安藤先生は説明を続けた。

「損益計算書は**1年間の活動**を記録している。でも貸借対照表は違う。**決算日という一瞬**の状態を切り取った写真——スナップショットなんだ」

```
【三表の時間軸】

損益計算書（P/L）
├─ 4月1日 ──────────────────── 3月31日 ─┤
│         1年間の「流れ」を記録          │

キャッシュフロー計算書（CF）
├─ 4月1日 ──────────────────── 3月31日 ─┤
│         1年間の「流れ」を記録          │

貸借対照表（B/S）
                              │
                        3月31日時点
                        「瞬間」を記録
```

「だから貸借対照表を見るときは、**その瞬間、会社が何を持っていて、何を借りているか**がわかる」

ミサキはメモを取った。

「体の状態を測る健康診断……会社版の健康診断みたいなものですね」

「そう。そして健康診断と同じで、**数値が正常範囲かどうか**を見ることが大切なんだ」

## Scene 2：資産・負債・純資産

「貸借対照表は、**三つのパート**に分かれている」

安藤先生は図を描いた。

```
┌─────────────────────────────────────────────┐
│              貸借対照表（B/S）              │
├──────────────────┬──────────────────────────┤
│                  │                          │
│      資産        │         負債             │
│    （Assets）    │     （Liabilities）      │
│                  │                          │
│  ・現金          │  ・借入金                │
│  ・売掛金        │  ・買掛金                │
│  ・在庫          │  ・社債                  │
│  ・工場・設備    ├──────────────────────────┤
│  ・土地          │                          │
│                  │       純資産             │
│                  │   （Net Assets）         │
│                  │                          │
│                  │  ・資本金                │
│                  │  ・利益剰余金            │
│                  │                          │
└──────────────────┴──────────────────────────┘

【大原則】
資産 ＝ 負債 ＋ 純資産
（持っているもの）＝（借りているお金）＋（自分のお金）
```

「左側が**資産**——会社が持っているもの。現金、売掛金、在庫、工場、土地など」

「右側は二つに分かれているんですね」

「そう。上が**負債**——銀行からの借金や、仕入れ先への未払金など、**いつか返さなきゃいけないお金**」

「下が**純資産**——株主から出資されたお金や、過去の利益の蓄積。これは**返さなくていいお金**」

ミサキは理解した。

「つまり、会社が持っている資産は、**借りたお金**と**自分のお金**で賄われている……ということですか」

「その通り。だから必ず**資産＝負債＋純資産**という等式が成り立つ。これを『貸借一致の原則』と言う」

## Scene 3：自己資本比率——会社の「筋肉質度」

「さて、ここで重要な指標を教えよう」

安藤先生は計算式を書いた。

```
自己資本比率 ＝ 純資産 ÷ 総資産 × 100

（自分のお金）÷（持っているもの全部）× 100
```

「自己資本比率は、**資産のうち、どれだけが自分のお金で賄われているか**を示す」

「借金に頼っていない度合い……ですか？」

「そう言い換えてもいい。自己資本比率が高いほど、**借金への依存度が低い**。つまり**財務的に安定している**」

安藤先生は目安を示した。

```
【自己資本比率の目安（一般企業）】

50%以上  → 非常に健全
30〜50%  → 健全
20〜30%  → やや注意
20%未満  → 要注意（借金依存度が高い）

※業種によって「正常値」は異なる
```

「ただし、**業種によって基準は違う**。たとえば銀行は自己資本比率10%でも健全とされる。これは次回詳しく話そう」

ミサキは質問した。

「自己資本比率が低いと、何が問題なんですか？」

「いい質問だね。借金が多いということは、**毎月の返済負担が重い**ということ。景気が悪くなったとき、利益が減っても返済は待ってくれない」

「黒字倒産のリスクが高まる……」

「その通り。逆に自己資本比率が高い会社は、**不況に耐える体力がある**。だから私は自己資本比率を『会社の筋肉質度』と呼んでいるんだ」

## Scene 5：財務レバレッジの光と影

「ところで田中さん、借金は悪いことだと思う？」

突然の質問にミサキは戸惑った。

「えっと……できれば少ない方がいい、ですか？」

「本当にそうかな？」

安藤先生は例を出した。

「君が100万円持っていて、年利5%で運用できるとする。1年後にいくらになる？」

「105万円です」

「じゃあ、銀行から100万円を年利2%で借りて、合計200万円を年利5%で運用したらどうなる？」

ミサキは計算した。

「200万円×5%で10万円の利益。借金の利息が100万円×2%で2万円。差し引き……8万円の利益！」

「そう。**自分のお金だけなら5万円の利益**だったのに、**借金を使ったら8万円**になった。これが**財務レバレッジ**の効果だ」

安藤先生は図を描いた。

```
【財務レバレッジの効果】

【借金なしの場合】
自己資金 100万円 × 利回り5% ＝ 利益 5万円
ROE（自己資本利益率）＝ 5%

【借金ありの場合】
自己資金 100万円 ＋ 借金 100万円 ＝ 運用資金 200万円
運用益 200万円 × 5% ＝ 10万円
利息 100万円 × 2% ＝ 2万円
純利益 ＝ 8万円
ROE（自己資本利益率）＝ 8万円 ÷ 100万円 ＝ 8%

→ レバレッジをかけるとROEが上がる
```

「借金を『てこ（レバー）』にして、**少ない自己資本で大きなリターンを得る**。これがレバレッジ経営だ」

「じゃあ、借金は多い方がいいんですか？」

「**うまくいっているときは**ね。でも、逆もある」

安藤先生の声が低くなった。

「もし運用利回りがマイナス5%になったらどうなる？」

ミサキは計算した。

「借金なしなら5万円の損失。借金ありなら……10万円の損失に利息2万円で、12万円の損失！」

「そう。**レバレッジは両刃の剣**なんだ。儲かるときは大きく儲かるが、損するときも大きく損する」

```
【レバレッジの光と影】

　　　　　　好況時　　　　不況時
借金なし　　利益 小　　　損失 小
借金あり　　利益 大　　　損失 大

→ 借金が多いほど「振れ幅」が大きくなる
```

「だから自己資本比率が低い会社——借金が多い会社は、**景気変動に弱い**。業績が悪化したとき、一気に経営危機に陥るリスクがある」

ミサキは理解した。

「トヨタが自己資本比率39%を維持しているのは、**リスクを取りすぎない**ためなんですね」

「そう。儲けを最大化するなら、もっと借金を増やしてレバレッジをかければいい。でもトヨタは**安定性を優先**している。過去の危機から学んだ教訓だね」

## Scene 6：健全性のチェックポイント

「貸借対照表を見るとき、いくつかチェックポイントがある」

安藤先生は箇条書きで説明した。

```
【B/S健全性チェックリスト】

□ 自己資本比率は業界水準以上か？
  → 30%以上なら一般的に健全

□ 現金・預金は十分か？
  → 月商の1〜3ヶ月分が目安

□ 売掛金が異常に増えていないか？
  → 売上に比べて売掛金だけ膨らんでいたら要注意

□ 在庫が異常に増えていないか？
  → 売れ残りが積み上がっている可能性

□ 借入金の返済期限は適切か？
  → 短期借入に依存しすぎていないか
```

「特に**売掛金**と**在庫**の動きは重要だ」

「どうしてですか？」

「覚えているかな。前回話した**利益とキャッシュの乖離**——その原因がここに隠れていることが多い」

安藤先生は説明した。

「売掛金が増えるということは、**売上は計上したけど、まだお金をもらっていない**ということ。利益は出ているのにキャッシュが増えない」

「在庫も同じですか？」

「そう。在庫が増えるということは、**仕入れでお金が出ていったけど、まだ売れていない**ということ。これもキャッシュを圧迫する」

ミサキはハッとした。

「さっき私が言った違和感……東洋テックの**利益は増えているのに営業CFの伸びが小さい**というのは……」

「売掛金か在庫が膨らんでいる可能性があるね。次のレッスンまでに、B/Sもチェックしてみるといい」

## コラム：なぜトヨタは10兆円の現金を持つのか

トヨタ自動車の「総資金量」——金融事業を除いた現金、定期預金、公社債等を合計すると、約**21兆円**に達する。なぜこれほど巨額の資金を保有しているのだろうか。

**危機の記憶**

トヨタは創業以来、数々の危機を乗り越えてきた。

- **オイルショック（1970年代）**：燃費重視の小型車戦略へ転換
- **リーマンショック（2008年）**：創業以来初の営業赤字4,610億円
- **大規模リコール問題（2009-2010年）**：世界で900万台規模のリコール
- **東日本大震災（2011年）**：サプライチェーン寸断で生産停止
- **コロナ禍（2020年）**：世界的な販売減少

特にリーマンショックは衝撃的だった。拡大路線の中で在庫が積み上がり、需要が急減したとき対応できなかった。この反省から、トヨタは「**何があっても潰れない財務体質**」を最優先課題として構築してきた。

**サプライチェーンを守る責任**

トヨタには約3万社の部品メーカーとの取引関係がある。危機時に自社だけでなく、サプライヤーを資金面で支援できる体制を持つことで、サプライチェーン全体の崩壊を防ぐ役割も担っている。

2020年のコロナ禍では、トヨタは1兆円のコミットメントライン（銀行融資枠）を設定し、「自動車産業全体の資金は潤沢だ」というメッセージを発信した。

**未来への投資余力**

電動化、ソフトウェア、全固体電池——巨額の研究開発投資が必要な時代において、十分な内部資金があることで、機動的かつ大胆な投資が可能になる。

トヨタの財務戦略は「**守りを固めることで、攻めの自由度を確保する**」という哲学に基づいている。一見すると「お金を貯めすぎ」に見えるかもしれないが、それは過去の危機から学んだ、生き残りの知恵なのだ。
