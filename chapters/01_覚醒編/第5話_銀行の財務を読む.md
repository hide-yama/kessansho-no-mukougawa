## Scene 1：銀行のビジネスモデル

「まず、銀行がどうやってお金を稼いでいるか、わかるかい？」

ミサキは考えた。

「えっと……預金を集めて、それを貸し出して、金利の差で儲ける……ですか？」

「基本はその通り。でも、ここに**銀行特有の構造**がある」

安藤先生は図を描いた。

```
【銀行のビジネスモデル】

┌─────────────────────────────────────────────────────┐
│                                                     │
│   預金者 ──(預金)──→ 銀行 ──(融資)──→ 借り手      │
│          ←(利息0.02%)←     ←(利息1.5%)←           │
│                                                     │
│   銀行の利益 ＝ 貸出金利 − 預金金利                 │
│              ＝ 1.5% − 0.02%                        │
│              ＝ 1.48%（利ざや）                     │
│                                                     │
└─────────────────────────────────────────────────────┘
```

「この**貸出金利と預金金利の差**を、**利ざや（NIM: Net Interest Margin）**と呼ぶ。銀行の最も基本的な収益源だ」

「製造業で言う『粗利率』みたいなものですか？」

「近いね。ただし、ここで重要なのは——**預金は銀行にとって『借金』である**ということだ」

ミサキは驚いた。

「預金が借金？」

「そう。預金者から見れば、銀行にお金を預けている。でも銀行から見れば、**預金者からお金を借りている**んだ。いつか返さなきゃいけない」

「だから貸借対照表では、預金は**負債**に計上される……」

「その通り。製造業にとって借入金は減らしたい負債だけど、銀行にとって預金は**増やしたい負債**なんだ。ここが根本的に違う」

```

「あれ、**売上高**じゃなくて**経常収益**って書いてありますね」

「いい気づきだ。銀行の損益計算書は、一般企業とは項目名が違う」

安藤先生は説明した。

「製造業の『売上高』に相当するのが、銀行の『**経常収益**』。でも中身が全然違う」

```
【銀行の経常収益の内訳】

┌─────────────────────────────────────────────────┐
│                                                 │
│  ① 資金利益（約50%）                           │
│     → 貸出金利息 − 預金利息                   │
│     → いわゆる「利ざや」からの収入             │
│                                                 │
│  ② 役務取引等利益（約40%）                     │

│                                                 │

│     → 国債売買益、為替差益など                 │
│                                                 │
└─────────────────────────────────────────────────┘
```

「面白いのは、一般企業なら『営業外収益』に計上される**受取利息**が、銀行では**本業の中核収益**になっていることだね」

ミサキはメモを取った。

「銀行の『売上』は、利息と手数料でできている……」

「そう。そして三井住友FGの純利益**1兆1,780億円**は、邦銀として**初めて1兆円を超えた**歴史的な数字なんだ」

## Scene 3：利ざやという生命線

```

貸出金利回り          0.72%          5.25%

────────────────────────────────────────────

「国内の利ざやが**0.69%**……？　すごく低くないですか？」

「低い。アメリカの銀行は**2.5〜3.5%**ある。日本は約3分の1しかない」

「なぜこんなに差があるんですか？」

「まず、**長年のゼロ金利・マイナス金利政策**。2016年から2024年まで、日銀がマイナス金利を続けていた。貸出金利を上げられないのに、預金金利はゼロに近い」

「金利が上がれば、利ざやも増える……？」

「その通り。2024年3月に日銀がマイナス金利を解除して、2025年1月には政策金利が**0.50%**まで上がった。三井住友FGの試算では、**政策金利0.25%上昇で年間+1,000億円**の増益効果がある」

ミサキは目を丸くした。

「金利が上がるだけで、1,000億円も儲かるんですか？」

「銀行は**金利ビジネス**だからね。金利環境が変わると、業績が大きく動く。これが銀行株を分析するときの重要なポイントだ」

## Scene 5：BIS規制という世界基準

「じゃあ、銀行の健全性はどうやって測るんですか？」

「ここで登場するのが、**BIS規制**——正式にはバーゼル規制と呼ばれる国際基準だ」

安藤先生は説明を始めた。

「2008年のリーマンショックを覚えているかい？」

「はい。トヨタが創業以来初の赤字に転落した年ですよね。先生に教えていただきました」

「よく覚えているね。あの時、世界中の銀行も危機に陥った。その反省から、国際的な銀行の自己資本規制が強化されたんだ」

```
【BIS規制（バーゼルIII）の自己資本比率】

      　　　　　自己資本
自己資本比率 ＝ ────────── × 100
      　　　　リスクアセット

※リスクアセット ≠ 総資産
```

「ここで重要なのは、分母が**総資産ではなくリスクアセット**ということ」

「リスクアセット？」

「資産の種類によって、リスクの重みづけをした数字だ」

安藤先生は具体例を示した。

```
【リスクウェイトの例】

国債（日本国債）   → リスクウェイト 0%
住宅ローン         → リスクウェイト 35%
一般企業向け融資   → リスクウェイト 100%

【計算例：資産100億円の場合】
・国債 50億円 × 0% ＝ 0億円
・住宅ローン 30億円 × 35% ＝ 10.5億円
・企業向け融資 20億円 × 100% ＝ 20億円
────────────────────────────────────
リスクアセット合計 ＝ 30.5億円

自己資本 5億円の場合：
・会計上の自己資本比率：5億円÷100億円 ＝ 5%
・BIS規制上の自己資本比率：5億円÷30.5億円 ＝ 16.4%
```

ミサキの目が輝いた。

「なるほど！　国債はリスクがゼロだから、分母に入らない。だから**同じ自己資本でも、BIS規制上は高い比率になる**んですね」

「その通り。三井住友FGの場合——」

```
【三井住友FG 自己資本比率の二つの見方】

会計上の自己資本比率      ：約4.8%（純資産÷総資産）
BIS規制上の自己資本比率   ：約15.2%（総自己資本÷リスクアセット）
CET1比率（中核自己資本）  ：約12.4%

→ 国際基準の最低要件（8%）を大幅に上回り、健全
```

「BIS規制で見れば**15%超**。十分に健全なんです」

「だから『銀行は自己資本比率10%で健全』と言われるのは、**BIS規制上の基準**ということなんですね」

「そういうこと。製造業の自己資本比率と銀行のそれは、**計算方法が根本的に違う**。単純比較してはいけないんだ」

## Scene 6：レバレッジの魔法——ROA 0.4%でROE 8%

「ここで、面白いことを見せよう」

安藤先生は三井住友FGとトヨタの比較表を出した。

```
【三井住友FG vs トヨタ自動車（2025年3月期）】

              三井住友FG    トヨタ
────────────────────────────────────────
純利益        1.18兆円      4.77兆円
総資産        306兆円       94兆円
純資産        14.8兆円      36.9兆円
────────────────────────────────────────
ROA           0.38%         5.1%
ROE           8.0%          13.3%
自己資本比率   4.8%          39%
財務レバレッジ 20.8倍        2.6倍
```

「ROAを見てごらん。三井住友FGは**0.38%**、トヨタは**5.1%**。約13倍も違う」

「銀行は効率が悪いんですか？」

「そうじゃない。**ビジネスモデルが違う**んだ。デュポン分解してみよう」

```
【デュポン分解】

ROE ＝ 利益率 × 資産回転率 × 財務レバレッジ

三井住友FG：8.0% ＝ 11.6% × 0.033 × 20.8
トヨタ    ：13.3% ＝ 9.9% × 0.51 × 2.6
```

「三井住友FGの資産回転率は**0.033回**。トヨタの**15分の1**しかない」

「そんなに低い……」

「巨大な預金を受け入れて、それを長期で貸し出すビジネスだからね。資産がなかなか回転しない」

「でも、ROEはトヨタとそこまで変わらない。なぜですか？」

「**財務レバレッジ**を見てごらん。三井住友FGは**20.8倍**、トヨタは**2.6倍**。約8倍も違う」

ミサキは計算した。

「つまり、資産回転率は15分の1だけど、レバレッジが8倍だから……結果的にROEは似たような水準になる！」

「そう！　これが**レバレッジビジネスとしての銀行**の本質なんだ。ROAは低くても、高レバレッジでROEを引き上げる」

## コラム：なぜ銀行株は長く「割安」だったのか

銀行株のPBR（株価純資産倍率）は、長年**1倍を下回る**状態が続いていた。PBR1倍割れとは、「会社を解散して資産を分配した方が、株を持っているより得」という評価を意味する。なぜこんなことが起きていたのか。

**ROEが株主資本コストを下回っていた**

理論的には、PBRは以下の式で説明できる。

```
PBR ≒ ROE ÷ 株主資本コスト
```

株主資本コストとは、「投資家が銀行株に期待するリターン」のこと。一般に8〜9%程度と言われる。

日本の銀行のROEは長年5〜7%程度で、株主資本コストを下回っていた。つまり「投資家の期待に応えられていない」状態だったのだ。

**金利環境の激変**

2024年3月、日銀がマイナス金利を解除。2025年1月には政策金利が0.50%に引き上げられた。これにより銀行の利ざやが改善し、業績が急回復している。

三井住友FGのROEは**8.0%**に到達し、PBRも**1.3倍程度**まで回復した。中島社長は「欧米主要銀行並みのROE 15%を長期目標とする」と表明している。

**投資家へのメッセージ**

銀行株を見るときは、**金利環境**と**ROE**の関係を意識することが大切だ。金利が上がればROEが改善し、株価も上がる。逆に金利が下がれば……という具合に、銀行株は金利に連動しやすい。

2024年から日本の金利は「正常化」に向かっている。これは銀行業にとって30年ぶりの追い風であり、銀行株の再評価につながっている。
