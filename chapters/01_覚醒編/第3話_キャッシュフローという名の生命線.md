# 第3話　キャッシュフローという名の生命線

---

## プロローグ：宿題の答え、そして新たな疑問

「先生、宿題をやってきました」

1週間後のレッスン。ミサキはノートを開いた。

「東洋テックの決算書を見てきたんですね。どうだった？」

「はい。海外事業のセグメントを中心に見ました」

ミサキは調べてきた数字を読み上げた。

「海外事業部門は売上高が2,800億円、営業利益率は8.2%。国内事業の6.5%より高いです。黒田専務が立て直した、と社内でも評判なんですよね」

「数字の上では、悪くないね」

「はい。でも……」

ミサキは言葉を探した。

「何か引っかかることがある？」

「うまく言えないんですが……数字は問題なさそうなのに、安藤先生が『気になることがある』とおっしゃっていたのが頭に残っていて」

安藤先生は穏やかにうなずいた。

「その違和感、大切にしておいて。今日のレッスンが終わる頃には、もう少しはっきりするかもしれない」

「え？」

「その前に一つ、質問させてくれ」

安藤先生はコーヒーカップを置いた。

「東洋テックの海外事業は、利益が出ている。じゃあ、**利益が出ていれば、その会社は安全**——そう思うかい？」

ミサキは少し考えてから答えた。

「……そう思います。利益が出ているなら、儲かっているということですよね？」

安藤先生の目が、少し鋭くなった。

「じゃあ、逆に聞くよ。**黒字なのに潰れた会社**を、知っているかい？」

## Scene 1：黒字倒産という恐怖

ミサキは首をかしげた。

「黒字なのに潰れる……？　そんなことあるんですか？」

「ある。しかも、珍しくない」

安藤先生は手元のタブレットを操作し、ある記事を見せた。

「2010年、日本航空——JALが経営破綻した。負債総額2兆3,000億円。戦後最大の倒産だった」

「JALって……あの飛行機の？」

「そう。でもね、JALは破綻直前の決算でも、**一応は黒字だった**んだよ」

ミサキは混乱した。

「どういうことですか？　黒字なら、儲かってるってことですよね？」

「うん、**損益計算書の上では**ね」

安藤先生はホワイトボードに向かった。

「ここで、今日のテーマだ。**キャッシュフロー計算書**——会社の『生命線』を見る地図だよ」

---

## Scene 2：利益とキャッシュは違う

「田中さん、一つ聞くよ」

「はい」

「君が友達に1万円を貸したとする。3ヶ月後に返してもらう約束だ。この瞬間、君のお財布には何円ある？」

ミサキは考えた。

「……1万円減っています」

「そうだね。じゃあ、君の**資産**はどうなった？」

「えっと……現金は減ったけど、『貸付金』という権利が増えた。だから資産の総額は変わらない……？」

「素晴らしい。じゃあ、ここで問題。友達が『悪いけど、返せなくなった』と言ってきたらどうなる？」

「……1万円の損失になります」

「**帳簿上は資産があっても、現金がなければ何も買えない**。これが『利益とキャッシュの違い』なんだ」

安藤先生はホワイトボードに大きく書いた。

```
利益（P/L）＝ 権利ベース
キャッシュ（CF）＝ 現金ベース
```

「損益計算書は『稼ぐ権利』を記録している。でも、権利だけでは従業員に給料は払えない。仕入れ先への支払いもできない」

「つまり……**現金がなくなったら、黒字でも潰れる**ってことですか？」

「その通り。昔からこう言うんだよ——」

---

## Scene 3：勘定あって銭足らず

安藤先生は古い言葉を口にした。

「**勘定あって銭足らず**」

「かんじょう……あって、ぜにたらず？」

「そう。帳簿の上では利益が出ている。でも手元に現金がない。だから支払いができず、倒産する。これを**黒字倒産**と呼ぶ」

ミサキは背筋が寒くなった。

「どうして、そんなことが起きるんですか？」

「いくつかパターンがある」

安藤先生は箇条書きで説明した。

```
【黒字倒産のパターン】

1. 売掛金の回収遅れ
   → 商品は売れた（利益計上）が、お金はまだもらっていない

2. 在庫の積み上げ
   → 仕入れでお金が出ていくが、売れるまで利益にならない

3. 設備投資の過剰
   → 将来のために工場を建てたが、現金がなくなった

4. 借金の返済
   → 毎月の返済が利益を上回っている
```

「JALの場合は、長年の過剰投資——特に大型機材への投資が重荷になっていた。利益は出ていても、**借金の返済に現金が消えていく**状態だったんだ」

ミサキは手帳にメモを取りながら言った。

「つまり、**利益だけ見ていても会社の健康状態はわからない**んですね」

「そう。だから私たちは、**キャッシュフロー計算書**を見る。お金の流れ——『どこから入って、どこへ出ていくか』を追うんだ」

---

## Scene 4：三つのキャッシュフロー

「キャッシュフロー計算書は、**3つのパート**に分かれている」

安藤先生はホワイトボードに図を描いた。

```
┌────────────────────────────────────────┐
│           キャッシュフロー計算書        │
├────────────────────────────────────────┤
│                                        │
│  ① 営業キャッシュフロー（営業CF）     │
│     → 本業で稼いだお金                 │
│                                        │
│  ② 投資キャッシュフロー（投資CF）     │
│     → 設備投資や資産売却に使ったお金   │
│                                        │
│  ③ 財務キャッシュフロー（財務CF）     │
│     → 借入・返済・配当に関するお金     │
│                                        │
└────────────────────────────────────────┘
```

「まず**営業CF**。これが最も重要だ」

「本業で稼いだお金……ですか」

「そう。商品を売って、お金を回収して、仕入れ代金を払って——日々の事業活動で**実際にいくらお金が残ったか**を示している」

「損益計算書の利益とは違うんですね」

「違う。たとえば100万円の商品を売っても、代金が来月入金なら、**今月の営業CFはゼロ**だ。でも損益計算書には100万円の売上が計上される」

ミサキは頭を整理した。

「営業CFがプラスなら、本業でちゃんとお金を稼げている……ということですね」

「その通り。**営業CFがずっとマイナスの会社は危険**だ。本業でお金を失い続けているわけだからね」

次に安藤先生は投資CFを説明した。

「**投資CF**は、工場を建てたり、設備を買ったりするお金。通常は**マイナス**になる」

「マイナスが普通なんですか？」

「うん。会社は成長のために投資する。投資にはお金がかかる。だから投資CFは**マイナスが健全**なんだ」

「逆にプラスだと……？」

「工場を売ったり、事業を縮小したりしている可能性がある。もちろん、**経営判断として正しい場合もある**けどね」

最後は財務CFだった。

「**財務CF**は、お金の調達と返済。銀行から借りればプラス、返済すればマイナス。株主への配当もマイナス」

「なるほど……三つのCFで、お金の入りと出を追うんですね」

「そして、この中で最も大切な数字がある。**フリーキャッシュフロー**——略してFCFだ」

---

## Scene 5：トヨタの巨大な金庫

「フリーキャッシュフローとは何か？」

安藤先生は計算式を書いた。

```
フリーキャッシュフロー（FCF）
＝ 営業CF − 投資CF（の絶対値）
＝ 本業で稼いだお金 − 成長投資に使ったお金
＝ 「自由に使えるお金」
```

「本業で稼いで、必要な投資を済ませて、**それでも手元に残ったお金**。これが『自由に使える』という意味でフリーキャッシュフローと呼ばれる」

「自由に使える……」

「この FCF があれば、借金を返せる。株主に配当を出せる。新しい事業に挑戦できる。逆に FCF がマイナス続きだと、**お金が流出し続けている**ことになる」

安藤先生はトヨタの数字を見せた。

```
【トヨタ自動車 2025年3月期 キャッシュフロー】

営業キャッシュフロー  ＋3兆6,969億円
投資キャッシュフロー  ▲1兆4,126億円
────────────────────────────────────
フリーキャッシュフロー 約2兆2,800億円

現金及び現金同等物    約9兆円
```

ミサキは数字の大きさに息をのんだ。

「フリーキャッシュフローだけで2兆円……？」

「そう。**毎年2〜4兆円のFCFを生み出す**。だから現金が約9兆円も貯まっている。これがトヨタの『金庫』だよ」

「なぜこんなに現金を持っているんですか？」

安藤先生の表情が少し曇った。

「トヨタは過去に何度も危機を経験している。2008年のリーマンショックでは、創業以来初の営業赤字——4,610億円の赤字に転落した」

「トヨタが赤字に……」

「その経験から、**何があっても潰れない財務体質**を最優先で作り上げてきた。9兆円の現金は、『保険』なんだ」

ミサキはメモを取りながら言った。

「利益率だけじゃなくて、**キャッシュの厚み**も見なきゃいけないんですね」

「そう。**利益は意見、キャッシュは事実**——という言葉がある。利益は会計処理で変わる余地があるが、現金の増減は誰にも操作できない」

## Scene 6：メルカリのフィンテック戦略

「じゃあ、第1話で見たメルカリはどうでしょう？」

ミサキは気になっていた。

「いい視点だね。メルカリを見てみよう」

安藤先生は新しいデータを開いた。

「メルカリには**フィンテック事業**がある。メルカード——聞いたことあるかい？」

「なんとなく……」

「メルカリが発行するクレジットカードだ。340万枚も発行されていて、**債権残高が1,872億円**もある」

「債権残高って……？」

「簡単に言うと、**ユーザーが『あと払い』で使ったお金**だ。メルカリが立て替えて、後から回収する」

ミサキは何かに気づいた。

「ちょっと待ってください。その1,872億円は……」

「そう。**メルカリから出ていったお金**だ。将来返ってくる『債権』ではあるけど、今この瞬間は手元にない」

「じゃあ、キャッシュフローにはマイナスの影響が……？」

「鋭いね。フィンテック事業が成長するほど、**一時的に現金は減る**。これを『成長投資』として許容できるかどうかが、投資家の判断になる」

安藤先生は補足した。

「メルカリの回収率は**99.2%**。ほとんど取りっぱぐれがない。だからこの1,872億円は、将来ほぼ確実に戻ってくる。**リスクを取って成長している**わけだ」

ミサキは感心した。

「損益計算書の利益だけ見てたら、こういう動きはわからないですね」

「そう。**CFを見ると、会社の戦略が見えてくる**んだよ」

「財務CFがプラスだったのは、金融事業の借入が増えたから。でも本業のトヨタは③の成熟期パターンに近い」

「営業CFで稼いで、投資して、まだお金が余って、株主に配当を出す……」

「そう。**理想的な成熟企業のCFパターン**だ」

ミサキはふと疑問を持った。

「①の成長期パターン——営業CFがマイナスでも、いいんですか？」

「いい質問だね。メルカリは2020年頃まで、まさにこのパターンだった。本業は赤字、でも成長のために投資を続け、株式発行や借入で資金を調達していた」

「赤字でも許されるんですか？」

「**将来の成長が見込める限りは**ね。投資家は『今の赤字』より『将来の黒字』を見ている。でも——」

安藤先生の声が低くなった。

「**成長が止まったら、話は別だ**。いつまでも赤字を垂れ流す会社は、いずれ市場から見放される」

レッスンも終盤に差しかかった頃、安藤先生がふと言った。

ミサキは背筋を正した。先生の表情がいつもと違った。

「2008年のことだ。リーマンショックが起きる直前——」

「私は証券会社のアナリストだった。ある銘柄を担当していてね。損益計算書は問題なかった。利益は出ていた。だから私は『問題ない』とレポートに書いた」

「……」

「でも、私は**キャッシュフローを軽視していた**。運転資本が膨らんでいることに、気づかなかったんだ」

運転資本——売掛金や在庫のことだとミサキは理解した。

「リーマンショックが来て、その銘柄は暴落した。私のレポートを信じて投資した人たちが、大損した」

安藤先生の声が震えた。

「ある投資家から電話があった。個人の方でね。『安藤さん、あなたを信じていたのに——』と、泣きながら話す声を聞いた」

ミサキは言葉が出なかった。

「その時、気づいたんだ。**私は数字だけを見ていた。数字の向こうにある人間を見ていなかった**」

沈黙が流れた。

「数字は嘘をつかない。でも、**数字の読み方を間違えると、人を傷つける**。それ以来、私はキャッシュフローを誰よりも重視するようになった」

安藤先生は顔を上げ、穏やかな表情に戻った。

「だから田中さん、覚えておいて。**利益は操作できる。でもキャッシュは嘘をつけない**。会社の本当の健康状態を知りたければ、必ずCFを見ること」

ミサキは深くうなずいた。

「先生がなぜ『数字の向こう側』と言うのか……少しわかった気がします」

「ありがとう。さて、今日の宿題だ」

## コラム：JALはなぜ破綻したのか——黒字倒産の教科書

2010年1月19日、日本航空（JAL）が会社更生法の適用を申請した。負債総額2兆3,221億円。戦後最大の企業倒産だった。

驚くべきことに、JALは**破綻直前まで黒字決算**を続けていた。ではなぜ、黒字の会社が倒産したのか？

答えは**キャッシュフロー**にあった。

**過剰な設備投資**

JALは長年、ボーイング747（ジャンボジェット）などの大型機材を大量に保有していた。機材の購入には巨額の借入が必要で、その返済が毎年重くのしかかっていた。

**損益計算書の罠**

航空機は「減価償却」という会計処理で、費用が何十年にも分散される。だから損益計算書上は、毎年の費用負担は小さく見える。

しかし**借金の返済は待ってくれない**。キャッシュは毎月、銀行へ流出していく。

**営業CFと利益の乖離**

JALの決算書を詳しく見ると、**営業利益が出ているのに、営業CFがマイナス**という年が続いていた。本業でお金を稼げていなかったのだ。

さらに悪いことに、地方路線の維持や関連会社への投資で、現金は出ていく一方だった。

**最後の砦が崩れたとき**

2008年のリーマンショックで航空需要が激減。それでも借金の返済は止まらない。ついに資金繰りが行き詰まり、倒産に至った。

**教訓**

JALの破綻は、**利益だけを見ていると会社の本当の姿を見誤る**という教訓を残した。

決算書を読むとき、必ずキャッシュフロー計算書を開くこと。特に**営業CFがプラスかどうか**——これが会社の「生命線」だ。

黒字でも、お金がなくなれば会社は終わる。

## 次回予告

「貸借対照表か——正直、まだよくわかっていません」

ミサキは正直に言った。
