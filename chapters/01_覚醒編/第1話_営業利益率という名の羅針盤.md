# 第1話　営業利益率という名の羅針盤

**核心の問い：なぜユニクロはトヨタより「効率的」に稼げるのか？**

---

## プロローグ：数字が怖い

会議室の空気が、鉛のように重い。

「田中、これどう思う？」

上司の声が遠くで聞こえる。目の前には、びっしりと数字が並んだ決算資料。売上高、営業利益、経常利益、当期純利益——。

（また数字だ）

私、田中ミサキ、24歳。大手電機メーカー「東洋テック」の経営企画部に異動して半年。数字が怖い。数字を見ると、頭が真っ白になる。

「田中？」

「あ、すみません。もう少し精査が必要かと——」

逃げるような返答。情けない。

会議室の向かいでは、先輩の高橋さんがすらすらと資料を読み上げている。「この営業利益率の推移を見ると……」「キャッシュフローの観点から言えば……」

なぜあの人は、あんなに自然に数字が読めるんだろう。

私には永遠にできない気がする。

---

## Scene 1：運命の出会い

その日の夕方、私は会議室で一人、資料と格闘していた。

「経営会議まで3ヶ月ある。それまでに、うちの会社の分析レポートを作れ」

部長からそう言われたのは、先週のこと。分析レポート。自社の財務を読み解いて、経営陣の前で発表する。

（無理。絶対、無理）

頭を抱えていると、会議室のドアがノックされた。

「失礼します。田中さんですか？」

振り返ると、銀縁眼鏡をかけた初老の男性が立っていた。白髪交じりの短髪。穏やかな目元。落ち着いたジャケット姿。

「佐藤部長から聞いてきました。財務分析で困っているとか」

「あ、はい……。安藤さん、ですか？」

社外アドバイザーとして入ってきた人がいるとは聞いていた。元証券アナリストで、近々社外取締役になるかもしれないとか。

「安藤誠一です。よろしく」

安藤さんは私の向かいに座り、テーブルの上の資料をちらりと見た。

「なるほど。決算書が並んでいますね」

「はい。でも、どこから手をつけていいか……」

正直に言った。わからないことを「わかったふり」するのは、私にはできない性分だ。

安藤さんは、少し間を置いてから言った。

「数字は嘘をつかない。でも、見方を知らないと何も語ってくれない」

「……え？」

「逆に言えば、見方さえ知っていれば、数字は雄弁に語り始める。企業の物語をね」

物語。決算書に、物語がある？

「田中さん。私に週1回、2時間だけくれませんか。3ヶ月で、君を変えてみせます」

その穏やかな声に、私は思わずうなずいていた。

---

## Scene 2：最初の問い——売上高の罠

翌週。私たちの最初のレッスンが始まった。

「さて」と安藤さんは言った。「最初に一つ質問をしていいかな」

「はい」

「ユニクロとトヨタ。どっちが『すごい』会社だと思う？」

突然の質問に、私は目を瞬かせた。

「えっと……トヨタ、ですか？」

「理由は？」

「売上高が……トヨタは48兆円くらいで、ユニクロは3兆円くらいですよね。15倍以上違います」

安藤さんは静かにうなずいた。

「いい答えだね。売上高を見るのは自然なことだ。でも——」

そう言って、安藤さんはホワイトボードに数字を書き始めた。

```
【売上高】
トヨタ：48兆367億円
ユニクロ：3兆1,038億円

【営業利益】
トヨタ：4兆7,955億円
ユニクロ：5,009億円
```

「営業利益を見てみよう。トヨタは4兆8,000億円、ユニクロは5,000億円。やはりトヨタのほうが約10倍大きいね」

「はい」

「じゃあ、これを見たらどう思う？」

安藤さんは、数字の横に別の数字を書き加えた。

```
【営業利益率】
トヨタ：9.9%
ユニクロ：16.1%
```

私は思わず声を上げた。

「あれ……ユニクロのほうが高い」

「そう。売上高ではトヨタが15倍も大きい。でも、『本業で稼ぐ効率』では、ユニクロのほうが上なんだ」

「本業で稼ぐ効率……」

「これが営業利益率。売上高の何パーセントが、本業の儲けとして手元に残るか。言い換えれば、100円売ったときに何円が営業利益になるか、という指標だよ」

私は計算してみた。

トヨタは100円売ると、約10円が営業利益。
ユニクロは100円売ると、約16円が営業利益。

「つまり……」

「君はどう思う？」

安藤さんは穏やかに問いかけた。答えを教えるのではなく、私に考えさせようとしている。

「売上高が大きい会社が、必ずしも『効率的に稼いでいる』わけではない……ということですか？」

安藤さんの目が、少し輝いた。

「そう、そこなんだよ。売上高だけを見ていると、企業の本当の姿を見誤る。営業利益率は、企業の『稼ぐ力』を測る羅針盤のようなものなんだ」

---

## Scene 3：羅針盤を手に入れる

「でも、安藤さん」

私は疑問をぶつけた。

「なぜユニクロのほうが、営業利益率が高いんですか？　トヨタは世界一の自動車メーカーですよね。効率化だってものすごいはず……」

「いい質問だね」

安藤さんは腕を組んで、少し考えてから言った。

「答えを先に言うのは簡単だけど、まずは君に仮説を立ててもらいたい。なぜだと思う？」

仮説。そう言われても、私には見当もつかない。

「えっと……ユニクロのほうが、コストを抑えているとか？」

「どんなコストを？」

「……わかりません」

安藤さんは微笑んだ。

「正直でいいね。わからないことを『わからない』と言えるのは、大きな強みだよ」

そう言って、安藤さんは新しい図を描き始めた。

```
【営業利益の構造】

売上高　　　　　　100%
　│
　├─ 売上原価　　▲X%　←「作る・仕入れるコスト」
　│
売上総利益（粗利）　Y%
　│
　├─ 販管費　　　▲Z%　←「売る・管理するコスト」
　│
営業利益　　　　　　●%
```

「営業利益は、売上高から『売上原価』と『販売費及び一般管理費（販管費）』を引いたものだ」

「売上原価と販管費……」

「売上原価は、商品を作ったり仕入れたりするためのコスト。トヨタなら鉄鋼や部品の仕入れ、工場の人件費。ユニクロなら服の素材や縫製工場への支払い」

「なるほど」

「販管費は、商品を売ったり会社を運営したりするためのコスト。広告宣伝費、店舗の家賃、本社の人件費、物流費用……そういったものだね」

私はノートにメモを取りながら聞いた。

「つまり、営業利益率を上げるには……」

「君はどう思う？」

「売上原価を下げるか、販管費を下げるか、あるいは両方……ですか？」

「そう。もしくは、同じコストでより多く売るか。いずれにしても、営業利益率の違いは、**ビジネスモデルの違い**から生まれるんだ」

ビジネスモデルの違い。

「ユニクロとトヨタは、そもそもビジネスモデルが違う……」

「そう。だから単純に『どちらがすごいか』は比較できない。大事なのは、**なぜその営業利益率なのか**を理解すること。それがわかれば、その会社の戦略が見えてくる」

私は、少しだけ霧が晴れた気がした。

---

## Scene 4：仮説を立てる技術

「では、実際に数字を見てみよう」

安藤さんは、ユニクロとトヨタの財務データを並べた。

```
【収益構造の比較】

　　　　　　　　ユニクロ　　　　トヨタ
売上高　　　　　3兆1,038億円　　48兆367億円
売上原価　　　　▲約1.5兆円　　　▲約39兆円
─────────────────────────────────
売上総利益　　　約1.5兆円　　　　約9兆円
（粗利率）　　　（約50%）　　　　（約18%）
販管費　　　　　▲約1兆円　　　　▲約4兆円
（販管費率）　　（約34%）　　　　（約8%）
─────────────────────────────────
営業利益　　　　5,009億円　　　　4兆7,955億円
（営業利益率）　（16.1%）　　　　（9.9%）
```

「ちょっと待ってください」

私は数字を見て、思わず声を上げた。

「粗利率が、ユニクロは50%で、トヨタは18%……。これ、すごく違いますよね」

「いい気づきだね。なぜだと思う？」

（また『君はどう思う？』だ）

でも、今度は少し考えることができた。

「えっと……ユニクロは服を売っていて、トヨタは車を売っている。車のほうが、部品とか材料費が高いから……？」

「惜しいね。方向性は合っている」

安藤さんは立ち上がり、窓際に歩いていった。

「自動車は、何万点もの部品を組み合わせて作る。鉄鋼、ガラス、プラスチック、電子部品、タイヤ……。一台の車を作るための原価は、販売価格の80%以上になることも珍しくない」

「80%以上！」

「一方、ユニクロは『SPA』というビジネスモデルを採用している。企画から製造、販売までを一貫して自社でコントロールする。中間業者を通さないから、マージンを抑えられる」

「だから粗利率が高いんですね」

「そう。でも、話はここで終わらない」

安藤さんは振り返った。

「面白いのは、販管費率のほうなんだ。ユニクロは34%、トヨタは8%。こちらはトヨタのほうがずっと低い」

私は表をもう一度見た。確かに、トヨタの販管費率は驚くほど低い。

「なぜトヨタは、販管費率がこんなに低いんですか？」

「仮説を立ててみようか」

---

## Scene 5：原価率という手がかり

「トヨタの販管費率が低い理由……」

私は必死に考えた。

「トヨタは……店舗を持たない？」

「そう。トヨタ自身は、基本的に車を直接消費者に売らない。販売店（ディーラー）が売るんだ」

「ああ！　だから店舗の人件費とか家賃とかがかからない」

「その通り。トヨタは製造会社であり、販売は別会社が担当する。だから販管費率は低く抑えられる」

「一方、ユニクロは……」

「自社で店舗を運営している。世界中に3,500店舗以上。店舗の家賃、スタッフの人件費、広告宣伝費——これらがすべて販管費に計上される」

私は理解し始めていた。

「つまり、ユニクロは粗利率が高いけど、販管費率も高い。トヨタは粗利率が低いけど、販管費率も低い。結果として——」

「結果として？」

「ユニクロは 50% - 34% = 16%、トヨタは 18% - 8% = 10%……あ、でも実際のトヨタの営業利益率は9.9%だから、ちょっと計算が合わないですね」

「細かい数字は他にも要素があるからね。でも、構造としてはそういうことだ」

安藤さんは、満足そうにうなずいた。

「いいかい、田中さん。同じ『小売業』でも、ビジネスモデルによって財務構造はまったく違う。同じ『製造業』でもそう。大事なのは、**なぜその数字になっているのか**を考えること」

「なぜその数字になっているのか……」

「数字を見て『高い・低い』と判断するだけでは、本質は見えてこない。『なぜ高いのか』『なぜ低いのか』を考える。そうすることで、数字の向こうにある**物語**が見えてくるんだ」

---

## Scene 6：販管費の罠——メルカリの謎

「もう一つ、面白い例を見せよう」

安藤さんは、新しい数字を書き加えた。

```
【メルカリの収益構造】

売上収益　　　　1,874億円　　100%
売上原価　　　　▲580億円　　▲31%
─────────────────────────────
売上総利益　　　1,294億円
（粗利率）　　　　　　　　　（69%）
販管費　　　　　▲1,119億円　▲60%
─────────────────────────────
営業利益　　　　175億円
（営業利益率）　　　　　　　（9.3%）
```

「粗利率69%！」

私は驚いた。ユニクロの50%よりさらに高い。

「でも、営業利益率は9.3%。ユニクロより低いですね」

「そう。なぜだと思う？」

「販管費率が……60%。すごく高い」

「何に使っていると思う？」

私は考えた。メルカリといえば……。

「広告？　テレビCMとか、よく見ます」

「正解。メルカリは創業以来、広告宣伝費に莫大な投資をしてきた。2024年6月期だけで442億円。売上高の約24%を広告に使っている」

「売上の4分の1を広告に……」

「彼らは『今』の利益より、『将来』のシェアを取ることを優先してきたんだ。プラットフォームビジネスは、ユーザーが増えれば増えるほど価値が上がる。だから、まず規模を取りに行く」

「だから粗利率が高いのに、営業利益率は低いんですね」

「そういうこと。粗利率が高い＝儲かっている、とは限らない。**粗利をどう使っているか**が重要なんだ」

私はノートに大きく書いた。

> **粗利率が高いのに営業利益率が低い → 販管費に何を使っているか確認**

「これは、成長企業を分析するときの重要なポイントになる。覚えておいてね」

## [コラム] 柳井正が毎朝最初に見る数字

ファーストリテイリング創業者・柳井正は、どんな数字を重視しているのだろうか。

柳井は毎朝5時に起き、6時過ぎには出社する。そこで最初にするのは、前日の売上と在庫のチェックだ。

「経営者には2つしか仕事がない」と柳井は言う。「一番大事なのは教育すること。もう1つが利益責任。利益がなければ誰も幸福になれません」

この「利益責任」という言葉に、彼の経営哲学が凝縮されている。

ユニクロが営業利益率16%を維持できるのは、偶然ではない。柳井は「利益率が下がったら、何かが間違っている」と考える。在庫が増えすぎていないか。値引きに頼りすぎていないか。無駄なコストが発生していないか。

利益率は、経営の「健康診断」なのだ。

興味深いのは、柳井が「売上高」ではなく「利益」を重視する点だ。

「売上は増やそうと思えば増やせる。値引きすればいい。でも、それでは会社は強くならない」

売上の追求ではなく、利益率の維持。これがユニクロの成長を支える哲学だ。

柳井は読書家としても知られる。愛読書は『プロフェッショナルマネジャー』（ハロルド・ジェニーン著）。ITTという複合企業を率いた経営者の回顧録で、柳井はこれを「最高の教科書」と呼ぶ。

その中に、こんな一節がある。

「経営の目的は利益を上げることである。その他のすべては、そのための手段にすぎない」

厳しい言葉だ。でも、この原則を愚直に守り続けたからこそ、ユニクロは山口県の小さな紳士服店から、世界第3位のアパレル企業へと成長できたのだろう。

数字を読むとは、経営者の哲学を読むことでもある。
