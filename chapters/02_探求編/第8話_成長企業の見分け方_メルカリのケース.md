# 第8話　成長企業の見分け方——メルカリのケース

---

## プロローグ：宿題の答え、そして発見

8週目の土曜日。

「先生、メルカリの決算書を読んできました」

ミサキはノートを開いた。びっしりと書き込まれたページには、メルカリの財務データと自分なりの分析が並んでいる。

「どうだった？」

「正直、最初は意味がわかりませんでした」

ミサキは苦笑した。

「**粗利率が69%もある**のに、**営業利益率は9%しかない**。トヨタは粗利率18%で営業利益率10%なのに。なぜ粗利率が高いのに利益率が低いのか——」

「いい疑問だ。答えは見つかった？」

「**広告宣伝費**です。売上の23%、400億円以上を広告に使っている。粗利益のほとんどを成長投資に回しているから、営業利益が少ない」

安藤先生は満足そうに頷いた。

「では、なぜ赤字だったのに高く評価されたのか——その仮説は？」

ミサキは深呼吸した。

「**国内フリマ事業だけ見れば、営業利益率は40%**です。メルカリ本体は稼いでいる。でも、米国事業やFintech事業に投資しているから、連結では赤字だった。つまり——」

「つまり？」

「**赤字は『悪いこと』ではなく、『成長への投資』だった**んじゃないかと思います」

安藤先生は静かに拍手した。

「素晴らしい。今日は、その仮説をさらに深掘りしていこう」

---

## 第1章：赤字企業が評価される理由

「まず、常識を疑おう」

安藤先生はホワイトボードに書いた。

```
【一般的な常識】
  利益が出ている ＝ 良い企業
  赤字 ＝ 悪い企業

【成長企業の論理】
  赤字 ＝ 成長への投資中
  黒字化できない ＝ 悪い企業
```

「成長企業は、**今の利益よりも将来の成長を優先**する。だから、利益を犠牲にして投資を続ける」

「でも、いつまでも赤字じゃダメですよね？」

「もちろんだ。だから投資家は、**黒字化のポテンシャル**を見ている」

```
【投資家が見ているポイント】

① 単体では黒字か？
   → メルカリ国内事業は営業利益率40%

② 赤字の原因は投資か？
   → 広告費・人件費など、止めれば黒字化できる

③ 市場は成長しているか？
   → CtoC市場、Fintech市場は拡大中

④ 競争優位はあるか？
   → MAU2,260万人、後発が追いつけない
```

「つまり、**赤字の質**を見ているんですね」

「その通り。すべての赤字が同じではない」

---

## 第2章：ユニットエコノミクス——成長の健全性を測る

「成長企業を評価する上で重要な概念がある。**ユニットエコノミクス**だ」

```
【ユニットエコノミクス】

  1人（または1取引）あたりの収益性を見る

  LTV（顧客生涯価値） ＞ CAC（顧客獲得コスト）

  → 健全な成長
```

「LTVとCACですか？」

「**LTV**は、1人の顧客が生涯にわたってもたらす利益の合計。**CAC**は、その顧客を獲得するのにかかったコストだ」

安藤先生は具体例を示した。

```
【メルカリの推定ユニットエコノミクス】

■ LTV（顧客生涯価値）の推計
  ・年間売上貢献：約4,400円
  ・平均利用年数：5年（推定）
  ・粗利率：69%
  → LTV ≒ 15,000円

■ CAC（顧客獲得コスト）の推計
  ・広告宣伝費：442億円
  ・新規ユーザー獲得数：約300万人/年（推定）
  → CAC ≒ 15,000円

■ LTV/CAC比率 ≒ 1.0
```

「LTVとCACがほぼ同じ……これはどうなんですか？」

「一般的に、**LTV/CAC比率は3以上が望ましい**と言われる。1.0はギリギリのライン」

「じゃあ、メルカリは……」

「新規ユーザー獲得だけで見ると厳しい。でも、メルカリには**別の武器**がある」

---

## 第3章：プラットフォームの力

「メルカリがなぜ成長を続けられるのか——その理由を考えてみよう」

```
【プラットフォームビジネスの特徴】

① 在庫リスクがない
   ・売り手と買い手をマッチングするだけ
   ・商品を仕入れる必要がない

② 限界費用がほぼゼロ
   ・ユーザーが増えても、追加コストは微小
   ・規模が大きくなるほど利益率が上がる

③ ネットワーク効果
   ・ユーザーが増えるほど、価値が上がる
   ・後発が追いつけない「勝者総取り」
```

「これが**粗利率69%**の理由ですね」

「そう。製造業のトヨタは粗利率18%。鉄やアルミを買って、工場で加工して、車を作る。物理的なコストがかかる」

「メルカリは物を作らないから、原価が低い」

「さらに重要なのが③のネットワーク効果だ」

安藤先生は図を描いた。

```
【ネットワーク効果の循環】

  出品者が増える → 商品が増える
        ↓               ↓
  買い手が集まる ← 選択肢が増える
        ↓
  さらに出品者が増える ...
```

「この循環が回り始めると、**後発は追いつけない**。MAU2,260万人というユーザー基盤は、それ自体が競争優位なんだ」

## 第4章：なぜ粗利率69%で営業利益率9%なのか

「ここで、君が発見した謎に戻ろう」

```
【メルカリの利益構造（2024年6月期）】

  売上収益      1,874億円   100%
  - 売上原価      580億円   ▲31%
  ─────────────────────────────
  売上総利益    1,294億円    69%  ← 高い粗利率
  - 販管費      1,119億円   ▲60%
    うち広告宣伝費 442億円
    うち人件費   約300億円
  ─────────────────────────────
  営業利益       175億円    9.3% ← 低い営業利益率
```

「販管費が売上の60%を占めている……」

「広告宣伝費442億円、人件費約300億円。合わせて700億円以上が、**成長のための投資**だ」

ミサキは計算した。

「もし広告費を半分にしたら、200億円以上の利益が増える。営業利益率は20%近くになりますね」

「その通り。でも、そうすると成長が止まる。**成長と利益のトレードオフ**——これが成長企業の経営判断だ」

## 第5章：メルカリの変化——投資から収穫へ

「メルカリの営業利益推移を見てみよう」

```
【メルカリ 営業利益の推移】

  2017年6月期：▲28億円
  2018年6月期：▲44億円
  2019年6月期：▲121億円（メルペイ投資）
  2020年6月期：▲193億円（赤字ピーク）
  ─────────────────────────────
  2021年6月期：+52億円（初の通期黒字）
  2022年6月期：▲37億円（再び赤字）
  2023年6月期：+164億円（本格黒字転換）
  2024年6月期：+175億円（黒字継続）
  2025年6月期予：+280〜320億円（拡大見込み）
```

「2020年が赤字のピークで、その後は黒字化している」

「そう。メルカリは**投資フェーズから収穫フェーズ**に移行した」

安藤先生は変化の理由を説明した。

```
【黒字化した理由】

① 広告費の抑制
   ・2022年：377億円 → 2023年：330億円

② 国内市場の成熟
   ・MAU 2,260万人で飽和に近づく
   ・新規獲得より既存顧客活性化へ

③ Fintech事業の黒字化
   ・メルカードが急成長
   ・金利収入がストック型収益に

④ 米国事業のリストラ
   ・人員削減でコスト構造改善
   ・2025年6月期に初の通期黒字化
```

「成長企業も、いつかは黒字化する必要があるんですね」

「**永遠に赤字でいい企業はない**。成長投資が一巡したら、利益を出す段階に移る。メルカリは今、まさにその転換点にいる」

## 第6章：Amazonの教訓

「成長投資の極端な例を紹介しよう。**Amazon**だ」

```
【Amazonの営業利益推移】

  創業：1994年
  上場：1997年
  初の年間黒字：2003年（創業から9年）

  ・20年以上、ほぼ利益ゼロで成長投資を続けた
  ・投資家は「いつでも黒字化できる」と信じた
  ・2024年：営業利益 約7兆円
```

「20年も利益ゼロ……投資家はよく待ちましたね」

「**AWSという隠し玉**があった。本業のECは薄利でも、クラウド事業が高収益を生む構造だった」

ミサキは考え込んだ。

「メルカリで言うと、Fintech事業がAWSに当たるんですか？」

「いい洞察だ。メルカリのFintech事業——メルカードの定額払いは**年率15%の金利収入**を生む。これがストック型の収益源になりつつある」

「今日の学びを整理しよう」

安藤先生はホワイトボードに書いた。

```
【成長企業を見分ける5つのチェックポイント】

① 単体事業は黒字化できるか？
   → メルカリ国内事業：営業利益率40%
   → 黒字化の「ポテンシャル」があるか

② 赤字の原因は「成長投資」か？
   → 広告費、研究開発費、人件費
   → 止めれば黒字化できるものか

③ ユニットエコノミクスは健全か？
   → LTV > CAC（理想は3倍以上）
   → 1人の顧客から利益を取れるか

④ 市場（TAM）は成長しているか？
   → CtoC市場、Fintech市場は拡大中
   → 天井が見えていないか

⑤ 競争優位（MOAT）はあるか？
   → ネットワーク効果、ブランド、技術
   → 後発が追いつけない壁があるか
```

「この5つをクリアしていれば、赤字でも評価されるんですね」

「逆に、1つでも欠けていれば**危険信号**だ」

## 第9章：投資家の思考法

「最後に、成長企業への投資について触れておこう」

```
【成長企業への投資の考え方】

■ 今の利益ではなく、将来の利益を見る
  → PERは高くなる（将来の利益を先取り）

■ 赤字の「質」を見極める
  → 成長投資か、構造的問題か

■ 黒字化のタイミングを予測する
  → いつ投資フェーズから収穫フェーズに移るか

■ リスクを織り込む
  → 成長が止まれば株価は急落
```

「成長企業への投資は、**将来を読む力**が試されるんですね」

「だから難しい。でも、財務三表を読む力があれば、他の人より深く企業を理解できる」

安藤先生は微笑んだ。

「君は今、その力を身につけつつある」

## コラム：なぜスタートアップは赤字でも資金調達できるのか

未上場のスタートアップは、創業から何年も赤字が続くことが珍しくない。それでも、VCやエンジェル投資家から数十億円、時には数百億円の資金を調達できる。なぜか。

**答え：将来の価値に投資しているから**

スタートアップへの投資は、**今の企業価値**ではなく**将来の企業価値**に対して行われる。

例えば、メルカリは上場前に累計100億円以上を調達した。当時は赤字だったが、投資家は「いずれ数千億円の企業になる」と信じて投資した。結果、上場時の時価総額は6,000億円を超えた。

**投資家が見ているポイント**

1. **市場規模（TAM）**
   - その市場はどれだけ大きくなるか
   - メルカリ：CtoC市場 約2兆円

2. **成長スピード**
   - どれだけ速く成長しているか
   - 年率50%以上の成長は高評価

3. **競争優位性**
   - なぜこの会社が勝つのか
   - ネットワーク効果、技術、ブランド

4. **経営チーム**
   - 創業者は信頼できるか
   - 過去の実績、ビジョンの明確さ

**リスクも大きい**

もちろん、スタートアップ投資はハイリスク・ハイリターンだ。統計的には、スタートアップの9割は失敗する。でも、成功した1社が100倍以上のリターンを生めば、ポートフォリオ全体ではプラスになる。

この「べき乗則」がスタートアップ投資の本質だ。
