## プロローグ：宿題の答え、そして新たな視点

6週目の土曜日。

「先生、宿題をやってきました」

ミサキはノートを開いた。東洋テックのROE（自己資本利益率）を過去4年分まとめてある。

「どうだった？」

「一応、同業他社とも比較してみました」

```
【東洋テックのROE推移】

  3年前：8.5%
  2年前：9.2%
  前年度：10.8%
  今年度：12.1%

【同業他社比較（今年度）】
  東洋テック：12.1%
  大手A社：10.5%
  大手B社：9.8%
```

「東洋テックのROEは順調に改善していて、同業他社より高いです。良い傾向……だと思うんですが」

ミサキは言葉を濁した。

「何か引っかかるところがある？」

「うまく言えないんですけど……前回調べた**売掛金と在庫の異常な増加**と、今回のROE改善が、どうつながるのか気になって」

安藤先生は静かに微笑んだ。

「いい疑問だ。今日の学びで、その答えが見えてくるかもしれない」

安藤先生のオフィスには、いつもの席に分厚い資料が置かれていた。表紙には「**株主総会参考資料**」の文字。

「先生、今日は何か違う雰囲気ですね」

「第II部の始まりだからね」

安藤先生は続けた。

「ここまでの5週間で、君は財務三表を読む力を身につけた。損益計算書、キャッシュフロー計算書、貸借対照表——会社の『成績表』『血流』『体質』を見る目だ」

ミサキは頷いた。確かに、以前は「数字の羅列」にしか見えなかった決算書が、今は「物語」として読めるようになっていた。

「でも、財務三表を読めるだけでは足りない。**投資家がどこを見ているか**——その視点を持てば、経営者の意思決定がより深く見えてくる」

投資家の視点。

それはミサキにとって未知の領域だった。

「今日のテーマは**ROEとROA**。そして、君が調べてきた東洋テックのROE——その『質』を見極める方法を教えよう」

## 第1章：株主は何を見ているのか

「まず質問だ」

安藤先生はホワイトボードに二つの会社の数字を書いた。

```
【A社】当期純利益 100億円、自己資本 500億円
【B社】当期純利益 100億円、自己資本 1,000億円
```

「どちらの会社が『効率的』に稼いでいると思う？」

ミサキは少し考えた。

「利益は同じ100億円ですが……A社の方が少ない元手で稼いでいますよね。効率的なのはA社だと思います」

「正解。**その効率性を測る指標がROE**だ」

安藤先生は式を書いた。

```
【ROE（自己資本利益率）】

         当期純利益
  ROE ＝ ────────── × 100
          自己資本

  A社：100億 ÷ 500億 × 100 ＝ 20%
  B社：100億 ÷ 1,000億 × 100 ＝ 10%
```

「ROEは**株主から預かったお金を、どれだけ効率的に利益に変えたか**を示す指標だ。株主にとって最も重要な数字の一つと言っていい」

「株主から預かったお金……」

「そう。株主は自分のお金を会社に預けている。そのお金を使って、会社がどれだけ稼いでくれるか——それがROEだ」

ミサキは納得した。

**自分がお金を預ける立場になれば、「いくら稼いでくれるか」は最重要ポイントだ。**

## 第2章：日本企業のROEの現実

「では、実際の企業を見てみよう」

安藤先生はトヨタの資料を開いた。

```
【トヨタ自動車のROE推移】

  2021年3月期：10.2%
  2022年3月期：10.9%
  2023年3月期：8.7%
  2024年3月期：14.5%（過去最高益の年）
  2025年3月期：13.3%
```

「トヨタのROEは約10〜14%で推移している。これは高いのか、低いのか」

「比較対象がないと判断できません……」

「いい視点だ。では他の企業と比べてみよう」

```
【主要企業のROE比較（直近期）】

  トヨタ自動車：13.3%
  ファーストリテイリング：約24%
  三井住友FG：8.0%
```

「ユニクロのファーストリテイリングは24%！トヨタの約2倍ですね」

「そう。アパレル業界は製造業より資産効率が良い傾向がある。ただし、注意が必要だ——」

安藤先生は続けた。

「**業種によってROEの『正常値』は違う**。銀行の三井住友FGは8%だが、これでも銀行業界では高い方だ。第5話で学んだね」

「はい。銀行は高レバレッジで稼ぐビジネスだから、ROAは低くてもROEで見ると健全、でした」

「よく覚えているね。では、ここで新しい指標を紹介しよう——**ROA**だ」

---

## 第3章：ROA——経営者の視点

安藤先生は新たな式を書いた。

```
【ROA（総資産利益率）】

         当期純利益
  ROA ＝ ────────── × 100
          総資産

  ※分子に営業利益を使う場合もある
```

「ROEが**株主視点**なら、ROAは**経営者視点**の指標だ」

「どう違うんですか？」

「ROEは『自己資本（株主のお金）』に対する効率性。ROAは『総資産（会社全体の資産）』に対する効率性だ」

```
【トヨタ自動車のROA推移】

  2021年3月期：3.6%
  2022年3月期：4.4%
  2023年3月期：3.5%
  2024年3月期：6.0%
  2025年3月期：5.2%
```

「ROEは13.3%なのに、ROAは5.2%……かなり差がありますね」

「その差を生んでいるのが**財務レバレッジ**だ」

---

## 第4章：デュポン分解——ROEの正体

「ROEをより深く理解するために、**デュポン分解**という手法を使う」

```
【デュポン分解】

  ROE ＝ 売上高純利益率 × 総資産回転率 × 財務レバレッジ

      当期純利益     売上高      総資産
  ＝ ────────── × ────────── × ──────────
       売上高       総資産      自己資本
```

「なぜデュポン分解と呼ばれるんですか？」

「アメリカの化学メーカー、デュポン社が1920年代に開発した手法だからだ。100年以上使われ続けている、経営分析の古典だね」

安藤先生は各要素を説明した。

**① 売上高純利益率**（収益性）
- 売上からどれだけ利益を残せるか
- 「稼ぐ力」を示す

**② 総資産回転率**（効率性）
- 資産をどれだけ効率的に使って売上を生んでいるか
- 「回す力」を示す

**③ 財務レバレッジ**（レバレッジ）
- 借入金を使ってどれだけ自己資本を増幅しているか
- 「てこの原理」を示す

「この三つを掛け合わせたものがROEなんだ」

---

## 第5章：実際の企業で分解してみる

「では、トヨタとファーストリテイリングをデュポン分解してみよう」

```
【トヨタ自動車（2025年3月期）】

  当期純利益：4兆7,650億円
  売上高：48兆367億円
  総資産：93兆6,013億円
  自己資本：約36.8兆円

  売上高純利益率：4.76兆 ÷ 48兆 ＝ 9.9%
  総資産回転率：48兆 ÷ 93.6兆 ＝ 0.51回
  財務レバレッジ：93.6兆 ÷ 36.8兆 ＝ 2.54倍

  ROE ＝ 9.9% × 0.51 × 2.54 ＝ 約12.8%
  ※計算上の誤差あり
```

「ファーストリテイリングはどうですか？」

```
【ファーストリテイリング（2024年8月期）】

  当期純利益：3,719億円
  売上高：3兆1,038億円
  総資産：約2兆7,000億円（推計）
  自己資本：約1兆5,500億円（推計）

  売上高純利益率：3,719億 ÷ 3.1兆 ＝ 12.0%
  総資産回転率：3.1兆 ÷ 2.7兆 ＝ 1.15回
  財務レバレッジ：2.7兆 ÷ 1.55兆 ＝ 1.74倍

  ROE ＝ 12.0% × 1.15 × 1.74 ＝ 約24%
```

「面白いですね！」

ミサキは数字を見比べた。

「トヨタは財務レバレッジが**2.54倍**とファーストリテイリングの**1.74倍**より高い。つまり、借入金を多く使っている」

「その通り。でも、もっと注目すべき点がある」

「総資産回転率ですね。ファーストリテイリングは**1.15回**なのに、トヨタは**0.51回**しかない」

「いい着眼点だ。製造業は工場や設備などの資産が多く必要だから、回転率が低くなりがちなんだ。一方、小売業は比較的少ない資産で売上を作れる」

**同じROEでも、その「稼ぎ方」はまったく違う——。**

ミサキは財務分析の奥深さを感じていた。

---

## 第6章：財務レバレッジの光と影

「ここで注意点だ」

安藤先生の表情が少し真剣になった。

「財務レバレッジは、ROEを高める**最も簡単な方法**だ。借金を増やせば、見た目のROEは上がる」

```
【レバレッジ効果の例】

  当期純利益：100億円（同じ）

  【ケースA】自己資本500億、借入金0
  → ROE：100億 ÷ 500億 ＝ 20%

  【ケースB】自己資本250億、借入金250億
  → ROE：100億 ÷ 250億 ＝ 40%
```

「借金を増やすとROEが上がる……」

「そう。でも、これには**大きなリスク**がある」

「業績が悪化したとき、ですね」

「その通り。第3話で学んだJALのケースを思い出してほしい。高レバレッジ経営は、好調時は高収益を生むが、不況時は致命傷になりかねない」

安藤先生は三井住友FGの数字を示した。

```
【三井住友FG（第5話の復習）】

  ROE：8.0%
  ROA：約0.3%
  財務レバレッジ：約26倍
```

「銀行のレバレッジは**26倍**。トヨタの2.5倍やユニクロの1.7倍とは桁が違う」

「だからBIS規制があるんですね。レバレッジをかけすぎないように」

「そう。**高レバレッジには高い規律が必要**——これが銀行業界の教訓だ」

## 第7章：ROEは高ければ良いのか

「ここで重要な問いを投げかけたい」

安藤先生はホワイトボードに書いた。

```
【問い】
ROEは高ければ高いほど良いのか？
```

ミサキは考え込んだ。

「株主からすれば、高い方が嬉しい気がしますが……」

「では、こう考えてみよう。**ROE30%の企業が、来年は赤字に転落するかもしれない**としたら？」

「それは嫌ですね……」

「ROEが高い理由を見極める必要がある」

安藤先生は三つのパターンを示した。

```
【ROEが高い3つのパターン】

① 収益性が高い（売上高純利益率が高い）
  → 本質的に強い企業。ユニクロ型

② 回転率が高い（総資産回転率が高い）
  → 効率経営に長けた企業。コンビニ型

③ レバレッジが高い（財務レバレッジが高い）
  → 借金で水増し。リスクに注意
```

「デュポン分解すると、ROEの『質』が見えてくるんですね」

「その通り。**ROEの数字だけでなく、その中身を見る**——これが投資家の視点だ」

## 第8章：東洋テックのROEを分解する

「さて、今朝君が持ってきた東洋テックのROE——これをデュポン分解してみよう」

ミサキの心臓が少し速くなった。

「ROEは12.1%で、同業他社より高い。表面上は良い数字だ。でも——」

安藤先生は言葉を区切った。

「デュポン分解すると、違う景色が見えてくる」

ミサキはノートを開いた。

```
【東洋テック デュポン分解（今年度）】

  売上高純利益率：6.2%（3年前：5.8%）→ 微増
  総資産回転率：0.68回（3年前：0.72回）→ 低下
  財務レバレッジ：2.87倍（3年前：2.04倍）→ 大幅上昇
```

「ROEが上がっているのは、**財務レバレッジが上がったから**……」

安藤先生は頷いた。

「収益性は微増、効率性はむしろ悪化。ROE改善の主因は借入金の増加だ」

「でも、第5話で調べた**売掛金と在庫の異常な増加**……これも関係ありますよね」

「いい視点だ。総資産回転率が低下しているのは、**売掛金や在庫が膨らんで資産が肥大化している**からだ」

ミサキは眉をひそめた。

**売上の伸びを上回って売掛金と在庫が増えている。ROEは財務レバレッジで水増しされている。**

何かがおかしい——その直感は、ますます強くなっていた。

## 第9章：株主資本コストという概念

「最後に、もう一つ重要な概念を紹介しよう」

安藤先生は新たな式を書いた。

```
【株主資本コスト】

  株主が投資に対して期待するリターン
  一般的に 8〜9% と言われる
```

「株主資本コスト……」

「株主はリスクを取って投資している。そのリスクに見合うリターンを求める——それが株主資本コストだ」

「つまり、ROEが8%以下だと……」

「**株主の期待に応えていない**ということになる。第5話のコラムで、銀行株のPBRが長年1倍を割っていた話をしたね」

ミサキは思い出した。

「銀行のROEが株主資本コストを下回っていたから、株が割安に放置されていた」

「その通り。ROEと株主資本コストの関係は、次回の第7話で詳しく学ぶ予定だ」

## コラム：なぜ日本企業のROEは低かったのか

「日本企業はROEが低い」——長年、これは日本株が割安に放置される理由として語られてきた。

2014年、経済産業省の「伊藤レポート」が発表され、「**ROE 8%以上を最低ラインとすべき**」という提言が話題となった。当時、東証一部上場企業の平均ROEは約5%程度。グローバルスタンダードからは程遠かった。

**なぜ日本企業のROEは低かったのか。**

その理由は、デュポン分解で説明できる。

**① 売上高純利益率が低かった**

日本企業は「売上至上主義」の傾向があり、利益率よりも売上規模を重視してきた。価格競争に陥りやすく、利益が削られる構造だった。

**② 財務レバレッジを使わなかった**

バブル崩壊とその後の「失われた30年」を経験した日本企業は、借入金を減らし、内部留保を積み上げる傾向が強まった。自己資本比率は高まったが、ROEは押し下げられた。

**③ 株主を意識しない経営**

「三方よし」の精神で、株主だけでなく従業員・取引先・地域社会すべてを重視してきた日本企業。結果として、株主リターンへの意識が希薄だった。

**2024年の変化**

2024年、東証が「資本コストを意識した経営」を上場企業に求める施策を打ち出した。PBR1倍割れ企業に対し、改善計画の開示を要請。

この動きを受け、多くの企業がROE向上に本腰を入れ始めている。自社株買いや増配も増加。日本企業のROEは着実に改善傾向にある。

トヨタは2025年の株主総会で「**ROE20%を目安とする**」と表明。日本を代表する企業が、株主視点を明確にした意義は大きい。
