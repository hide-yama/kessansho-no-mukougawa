
## 第3章：配当という「安定収入」

「株主還元の基本は**配当**だ」

```
【トヨタ 配当金の推移】

  2021年3月期：48円
  2022年3月期：52円
  2023年3月期：60円
  2024年3月期：75円
  2025年3月期：90円

  → 5年で約2倍に増配
```

「配当が増え続けている……」

「トヨタは『**安定的・継続的に増配を行う**』という方針を掲げている。過去30年以上、減配したことがない」

ミサキは計算した。

「配当利回りは……株価3,000円として、90円÷3,000円＝約3%ですね」

「そう。銀行預金の金利がほぼゼロの時代に、毎年3%の配当がもらえる。しかも増配が続けば、将来はもっと利回りが上がる」

「これが成熟企業に投資する魅力なんですね」

## 第4章：自社株買いの効果

「もう一つの株主還元が**自社株買い**だ」

```
【自社株買いの仕組み】

  会社が自分の株を市場から買い戻す
  → 発行済株式数が減少
  → 1株当たりの価値が上昇

【トヨタの例】

  純利益：4.7兆円
  発行済株式数：130億株

  EPS（1株当たり利益）＝4.7兆÷130億＝361円

  もし10%の自社株買いで株数が117億株に減ると…
  EPS＝4.7兆÷117億＝401円（+11%）
```

「株数が減ると、1株当たりの利益が増える……」

「同時に、ROEも上昇する。純資産が減るからね」

「なるほど。自社株買いは**資本効率を高める**手段なんですね」

「2024年5月、トヨタは**上限1兆円**という大規模な自社株買いを発表した。これは株主に対する強いメッセージだ」

## 第5章：PBR1倍問題への対応

「ここで、第7話で学んだ**PBR**を思い出してほしい」

```
【PBRの復習】

  PBR ＝ 株価 ÷ 1株当たり純資産

  PBR < 1倍：「解散した方がマシ」という評価
```

「トヨタのPBRはどうですか？」

「2024年時点で約**1.1〜1.3倍**程度。かつては1倍を下回っていた時期もあった」

ミサキは思い出した。

「2024年に東証がPBR1倍割れ企業に改善を求めた話でしたね」

「そう。トヨタはその前から株主還元を強化していたが、東証の要請を受けてさらに踏み込んだ」

安藤先生は重要な発表を紹介した。

「2025年の株主総会で、トヨタは『**ROE20%を目安とする**』と表明した。現在のROE13%から大きくジャンプアップする目標だ」

「どうやって達成するんですか？」

「①利益成長、②自社株買い、③配当増加——この三つを組み合わせて、資本効率を高めていく」

## コラム：日本企業の「現金溜め込み」問題

日本企業は「現金を溜め込みすぎ」と批判されることがある。実際、上場企業の現金保有額は過去最高を更新し続けている。なぜこれが問題なのか。

**資本効率の観点**

現金は「働いていないお金」だ。銀行に預けても金利はほぼゼロ。その間、株主は配当をもらえず、企業価値は上がらない。

```
【現金保有のトレードオフ】

現金を持つメリット：
  ・危機対応力（リーマンショック対策）
  ・M&A機会への備え
  ・安心感

現金を持つデメリット：
  ・資本効率の低下（ROEを押し下げ）
  ・株主への還元不足
  ・PBR1倍割れの原因に
```

**トヨタの「適正水準」**

トヨタは現金を約9兆円保有しているが、これは「溜め込みすぎ」なのか。

トヨタの言い分は「危機対応」だ。リーマンショックでは創業以来初の赤字を経験。その教訓から「何があっても潰れない財務体質」を最優先にした。

しかし2024年以降、トヨタは株主還元を大幅に強化。配当増加と自社株買いで、利益の半分近くを株主に返すようになった。

**バランスの変化**

日本企業全体で見ると、「現金溜め込み」から「株主還元」へのシフトが始まっている。東証のPBR改善要請が一つの転機となった。

企業は今、「いくら現金を持つべきか」「どれだけ株主に還元すべきか」という問いに、真剣に向き合っている。
