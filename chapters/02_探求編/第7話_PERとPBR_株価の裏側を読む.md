# 第7話　PERとPBR——株価の「裏側」を読む

---

## プロローグ：宿題の答え、そして疑問

7週目の土曜日。

「先生、宿題をやってきました」

ミサキはノートを開いた。東洋テックの株価指標をまとめてある。

```
【東洋テックの株価指標】

  株価：1,850円
  PER：約12倍
  PBR：約0.9倍

【同業他社比較】
  大手A社：PER 15倍、PBR 1.2倍
  大手B社：PER 18倍、PBR 1.5倍
```

「東洋テックは同業他社より**割安**に見えます。PERもPBRも低い。でも……」

ミサキは言葉を詰まらせた。

「何か引っかかる？」

「前回の授業で、ROEが**レバレッジで水増し**されていると教わりました。それなのに株価は割安。これって……おかしくないですか？」

安藤先生は静かに微笑んだ。

「いい質問だ。今日はその疑問に答えよう。**PERとPBR**——株価の裏側を読む方法を教える」

---

## 第1章：株価は何を反映しているのか

「まず基本から確認しよう」

安藤先生はホワイトボードに書いた。

```
【株価の決まり方】

  株価 ＝ 企業の価値 ÷ 発行済株式数

  企業の価値 ＝ 将来の利益の合計（の予測）
```

「株価は**将来の利益への期待**を反映している。過去の実績ではなく、これからどれだけ稼ぐかの予測だ」

「将来……」

「だから、同じ利益100億円でも、**成長が期待される企業**と**成熟した企業**では、株価が全く違う」

ミサキは何となく理解できた。

「伸びしろがあるかどうか、ということですね」

「その通り。では、その**期待の度合い**を測る指標を紹介しよう」

---

## 第2章：PER——利益の何倍か

「最も重要な株価指標が**PER**だ」

```
【PER（株価収益率）】

         株価
  PER ＝ ────────
        1株当たり純利益（EPS）

  ※時価総額 ÷ 当期純利益 でも同じ
```

「PERは**株価が1株当たり利益の何倍か**を示す。言い換えると、**投資資金を回収するのに何年かかるか**だ」

「何年かかるか……」

「例えば、PER10倍なら、今の利益が続けば**10年で投資回収**できる計算になる。PER20倍なら20年だ」

安藤先生は具体例を示した。

```
【PERの具体例】

  トヨタ自動車：PER 約10倍
  → 今の利益水準なら、10年で投資回収

  メルカリ：PER 約30倍
  → 今の利益水準なら、30年で投資回収
```

「メルカリは30年かかるんですか？　それなのに投資する人がいるんですか？」

「いい疑問だ。**今の利益水準が続くなら**30年かかる。でも、成長企業は利益が毎年増える。だから、将来の利益成長を織り込んで、高いPERが許容されるんだ」

---

## 第3章：成長企業と成熟企業

「PERの違いを理解するために、二つの企業を比較してみよう」

```
【トヨタ vs メルカリ（2024年度）】

                  トヨタ          メルカリ
  当期純利益      4兆7,650億円    135億円
  利益成長率      ▲3.6%          +2.7%
  PER             約10倍          約30倍

  将来の期待      成熟・安定      成長・拡大
```

「トヨタは利益が4兆円超えているのに、PERは10倍。メルカリは利益135億円なのに、PERは30倍」

「なぜこんなに違うんですか？」

「**成長期待**の差だ」

安藤先生は説明を続けた。

**トヨタ（PER 10倍）**
- 既に世界トップクラスの規模
- 今後の利益が大きく伸びる余地は限定的
- 「安定配当」に価値がある

**メルカリ（PER 30倍）**
- 国内フリマ事業は成熟しつつある
- Fintech事業が急成長中
- 将来の利益が大きく伸びる**期待**がある

「つまり、PERは**将来への期待値**なんですね」

「その通り。ただし、注意が必要だ」

---

## 第4章：PERの罠

「PERには**落とし穴**がある」

安藤先生はホワイトボードに書いた。

```
【PERの注意点】

① 赤字企業には使えない
   → 利益がマイナスなら、PERは計算できない

② 特殊要因に注意
   → 一時的な利益で低PERに見えることも

③ 業種で「正常値」が違う
   → IT企業と製造業で比較してはダメ
```

「特に注意したいのが②だ。一時的に利益が膨らんで、PERが低く見えることがある」

「どういうことですか？」

「例えば、資産売却で一時的に利益が増えたケース。来年は元に戻るのに、今期だけPERが低くなる」

ミサキは東洋テックのことを思い出した。

（東洋テックのPERが低いのも、何か理由があるのかもしれない……）

---

## 第5章：PBR——純資産の何倍か

「もう一つの重要な指標が**PBR**だ」

```
【PBR（株価純資産倍率）】

         株価
  PBR ＝ ────────
        1株当たり純資産（BPS）

  ※時価総額 ÷ 純資産 でも同じ
```

「PBRは**株価が1株当たり純資産の何倍か**を示す。言い換えると、**会社の解散価値との比較**だ」

「解散価値？」

「会社を今すぐ解散して、資産を全部売り払い、負債を返済したら、株主に残るのが純資産。PBR1倍は**解散価値と株価が同じ**という意味だ」

```
【PBRの解釈】

  PBR > 1倍：市場は「この会社は純資産以上の価値がある」と評価
  PBR = 1倍：株価と解散価値が同じ
  PBR < 1倍：市場は「解散した方がマシ」と評価
```

「PBR1倍割れは、**株主にとって屈辱的な評価**なんですね」

「その通り。だから近年、東証がPBR1倍割れ企業に改善を求めている」

---

## 第6章：ROEとPBRの関係

「ここで、前回学んだ**ROE**とPBRの関係を見てみよう」

安藤先生は式を書いた。

```
【ROEとPBRの関係】

  PBR ≒ ROE ÷ 株主資本コスト
```

「株主資本コストは**投資家が期待するリターン**——だいたい8〜9%と言われている」

ミサキは計算した。

「ROEが8%で株主資本コストが8%なら、PBRは1倍ということですか？」

「その通り。**ROEが株主資本コストを上回れば、PBRは1倍を超える**。下回れば、1倍を割る」

```
【具体例】

  企業A：ROE 15%、株主資本コスト 8%
  → PBR ≒ 15% ÷ 8% ≒ 1.9倍

  企業B：ROE 6%、株主資本コスト 8%
  → PBR ≒ 6% ÷ 8% ≒ 0.75倍
```

「第5話で銀行株のPBRが長年1倍を割っていた話をしたね。あれはROEが株主資本コストを下回っていたからだ」

ミサキは点と点がつながる感覚を覚えた。

「だから今、東証がROE改善を求めているんですね。ROEが上がればPBRも上がる」

---

## 第7章：バリュートラップの恐怖

「ここで重要な警告をしておきたい」

安藤先生の表情が真剣になった。

「**バリュートラップ**——割安の罠という言葉を聞いたことがあるかい？」

「いえ……」

「PERやPBRが低い株を見ると、**割安だ、買いだ**と思いがちだ。でも、**低いには理由がある**」

安藤先生は例を挙げた。

```
【バリュートラップの例】

■ ケース1：構造的な問題
  ・業界が衰退している
  ・競争力を失っている
  ・利益が減り続ける運命
  → PERは低いが、さらに下がる可能性

■ ケース2：隠れたリスク
  ・財務内容に問題がある
  ・訴訟リスクを抱えている
  ・経営者に不信感がある
  → 市場はリスクを織り込んで安くしている

■ ケース3：会計操作
  ・利益が水増しされている
  ・実態より良く見せている
  → PERは低く見えるが、実際は高い
```

ミサキの背筋が凍った。

**ケース3——これは、東洋テックのことではないか？**

「先生、東洋テックのPERが低いのは……」

「君はどう思う？」

「売掛金と在庫が異常に増えている。ROEはレバレッジで水増し。もしかすると、**利益自体も水増しされている**可能性が……」

安藤先生は静かに頷いた。

「今すぐ結論は出せない。でも、**割安に見える株には、割安な理由がある**。それを見極める目が必要だ」

---

## 第8章：成長企業の評価——PSR

「話を戻そう。成長企業の評価について」

安藤先生は新しい指標を紹介した。

```
【PSR（株価売上高倍率）】

         時価総額
  PSR ＝ ────────
          売上高
```

「赤字企業はPERで評価できない。そこで使われるのが**PSR**——株価が売上高の何倍かを見る指標だ」

「メルカリが上場した2018年は赤字でしたよね」

「そう。当時はPSRで評価されていた。売上成長率が高ければ、高いPSRが許容される」

```
【メルカリの評価推移】

  2018年（上場時）：赤字 → PSRで評価
  2019年：赤字 → PSRで評価
  2020年：赤字 → PSRで評価
  2021年：初黒字 → PER約100倍
  2023年：黒字定着 → PER約40倍
  2024年：利益成長 → PER約24倍
  2025年予想：さらに成長 → PER約17倍
```

「PERがどんどん下がっていますね」

「利益が成長すれば、株価が同じでもPERは下がる。成長企業への投資は、**将来のPER低下を先取り**しているんだ」

---

## 第9章：東洋テックの株価を読む

「では、東洋テックの株価を分析してみよう」

```
【東洋テックの株価指標（再掲）】

  PER：約12倍（同業他社：15〜18倍）
  PBR：約0.9倍（同業他社：1.2〜1.5倍）
```

「数字だけ見ると割安だ。でも、君が発見した異常を踏まえると——」

ミサキは整理した。

```
【東洋テックの「異常」リスト】

  ① 売掛金が売上を上回るペースで増加
  ② 在庫も売上を上回るペースで増加
  ③ ROEはレバレッジで水増し
  ④ 利益とキャッシュフローが乖離
```

「これらが事実なら、**今の利益は持続可能ではない**可能性がある」

「つまり、PERは低く見えるけど、実態はもっと高い……」

「その可能性は否定できない。だから、株価指標だけで判断してはいけないんだ」

安藤先生は念を押した。

「**PERが低い＝割安＝買い**——こんな単純な投資は、プロはしない。なぜ低いのか、その理由を突き詰める必要がある」

---

## 第10章：株価指標を使いこなすために

「最後に、今日の学びをまとめよう」

安藤先生はホワイトボードに書いた。

```
【株価指標を使いこなす5つのルール】

① PERは「成長期待」を反映する
   → 成長企業は高く、成熟企業は低い

② PBRは「ROE」と連動する
   → ROE > 株主資本コストならPBR > 1倍

③ 低PER・低PBRには「理由」がある
   → バリュートラップに注意

④ 赤字企業はPSRで評価する
   → 売上成長率がカギ

⑤ 財務三表と併せて見る
   → 株価指標だけでは判断できない
```

「⑤が最も重要だ。株価指標は**財務三表の延長線上**にある。三表を読めなければ、株価指標も正しく使えない」

ミサキは深く頷いた。

「先生、私……東洋テックの決算書をもっと深く読みたいです」

「いい心がけだ。次回は**成長企業の見分け方**を学ぶ。メルカリを題材に、赤字企業がなぜ評価されるのか、その本質を教えよう」

「楽しみです」

「宿題を出そう。メルカリの決算書を読んできてほしい。なぜ赤字だったのに高く評価されたのか、君なりの仮説を立ててきて」

「わかりました」

## コラム：なぜ日本株は長年「割安」だったのか

「日本株は割安」——これは長年、海外投資家から言われ続けてきた評価だ。実際、東証プライム市場の約半数がPBR1倍割れという状況が続いていた。

**なぜ日本株は割安に放置されたのか。**

その答えは、第6話で学んだ**ROE**にある。

日本企業のROEは長年、欧米企業に比べて低かった。株主資本コスト（約8%）を下回るROE 5〜6%の企業が多く、理論上、PBRは1倍を下回って当然だった。

**なぜROEが低かったのか**

1. **内部留保の積み上げ**
   - バブル崩壊の教訓で、現金を溜め込む傾向
   - 自己資本が膨らみ、ROEを押し下げた

2. **株主還元への消極性**
   - 配当性向が欧米より低い
   - 自社株買いも限定的

3. **成長投資の不足**
   - 溜め込んだ現金を有効活用できない
   - 「守り」の経営が常態化

**2024年の転換点**

東証がPBR1倍割れ企業に改善計画の開示を要請。これを受けて、多くの企業がROE向上策を打ち出している。

- **増配・自社株買い**：株主還元を強化
- **事業売却**：非効率な資産を圧縮
- **成長投資**：M&Aや設備投資を拡大

トヨタは「ROE20%を目安」と宣言。日本を代表する企業が株主視点を明確にした意義は大きい。

日本株の「割安」が解消されつつあるのは、企業経営が変わりつつある証拠だ。
